# TW-Overlay v1.17.5 릴리즈 노트 (2026.06.05)

이번 업데이트는 필드보스 알림 설정에 "혼란한 대지" 및 스폰 시간대 추가와, 다중 필드보스 출현 시 알림이 누락되지 않도록 로직을 개선하고 알림음 겹침 현상을 해결하기 위해 순차 재생 큐 시스템을 적용했습니다.

---

## 🚀 Added (기능 추가)

- **필드보스 알림에 "혼란한 대지" 추가**:
  * 스폰 시간대(00시, 07시, 13시, 18시, 21시)를 추가하고 관련 소리 및 토글 활성화 설정을 지원합니다.

## 🐛 Fixed (버그 수정)

- **동시간대 다중 필드보스 알림 누락 방지 및 순차 재생**:
  * 여러 보스가 동일한 시간에 출현할 때 알림이 누락되는 현상을 수정하여 모든 필드보스 알림이 동작하도록 개선했습니다.
  * 여러 보스의 사운드가 겹쳐서 재생이 끊기는 현상을 방지하기 위해 순차 재생 큐를 도입했습니다. 첫 사운드가 끝난 후 0.5초 뒤에 다음 사운드가 연속적으로 재생됩니다.
- **필드보스 설정 UI 버그 수정**:
  * 설정 페이지 오픈 시 "혼란한 대지"의 스폰 시간이 정상적으로 로드되지 않고 비어 보이던 데이터 연동 버그를 수정했습니다.

---

## 📦 설치 및 업데이트 방법

1. **설치 파일 다운로드**:
   * 본 릴리즈 페이지 최하단의 **Assets** 섹션에서 `twOverlay-Setup-1.17.5.exe` 파일을 클릭하여 다운로드합니다.
2. **프로그램 실행 및 설치**:
   * 다운로드한 `.exe` 파일을 실행하면 자동으로 설치 및 업데이트가 진행됩니다.
   * *참고: Windows SmartScreen 경고가 뜰 경우, `추가 정보` 클릭 후 `실행`을 누르면 됩니다.*
