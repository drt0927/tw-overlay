# TW-Overlay v1.12.4 (2026.05.10)

## 🐛 Fixed
* **어벤던로드 입장료 정규식 고도화:** 게임 내 다양한 형태의 한글 숫자 로그(예: "1억 40만 Seed", "1억 Seed", "40만 Seed")를 정상적으로 금액으로 환산하도록 정규표현식을 개선했습니다.
* **어벤던로드 입장 로그 예외 처리:** 입장 로그의 오타나 비일관된 띄어쓰기 패턴(예: "횟수" 및 "횟 수")을 모두 감지하도록 유연한 정규식을 적용하여 누락을 방지했습니다.

## 🔄 Changed
* **어벤던로드 트래커 상태 관리 아키텍처 개편:** 트래커의 사용 여부를 결정하는 전역 설정(`isEnabled`)과 게임 화면의 UI 표시 여부(`isActive`)를 명확하게 분리했습니다. 계산기 창에서 설정한 ON/OFF 상태는 내부 설정 파일에 안전하게 저장되어 재시작 시에도 유지되며, 즉시 숨김(X) 버튼은 화면에서만 임시로 숨기도록 동작합니다.
