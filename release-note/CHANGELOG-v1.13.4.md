# 📦 TW-Overlay Release Note - v1.13.4 (2026.05.23)

이번 v1.13.4 업데이트에서는 **실시간 채팅 로그 기반 숙제 자동 체크 시스템 신설**과 **신규 버프 타이머 5종 추가 및 편의성 개선**을 진행하였습니다.

---

## 🚀 주요 변경 사항

### 📅 지능형 숙제 자동 체크 시스템 신설
매번 수동으로 클릭하여 완료해야 했던 번거로움을 해결하기 위해, 게임 내 실시간 채팅 로그를 추적하여 숙제 클리어를 감지하고 자동으로 완료 횟수를 기록하는 시스템이 새롭게 도입되었습니다.
* **자동 완료 감지 지원 콘텐츠 대폭 추가**:
  * **주간 레이드**: `베스티지`, `아페티리아 (일반/어려움)`
  * **이클립스**: `이클립스 (에토스/마티아/티로로스/라이코스/체리아/셀피나)`, `이클립스 보급품 탈환`, `이클립스 별동대 토벌`
  * **기타 핵심 던전**: `머큐리얼 (실반/샐리온/실라이론/샐레아나/루미너스/루미너스 EX)`, `어비스 코어 마스터 (심층 I~III)`, `머큐리얼 코어 마스터 (실반/샐리온/실라이론/샐레아나/루미너스)`, `신조의 둥지 (일반/어려움)`, `테시스 코어 던전`, `지하요새의 망령`, `프라바 방어전 (1인)`, `고대 렐릭의 성소 (신조/키시니크)`, `시오칸 하임 보스 토벌전`, `오를리 방어전`, `어비스 보스전 (EX)`, `발굴지`, `심연의 보물창고`, `힘의 근원`, `망각의 카타콤 (지옥)`, `어비스 보스 (심층 1~3)` 등 총 20여 종의 핵심 숙제 자동 클리어 연동.
* **실시간 누적 카운트**: 보스 및 콘텐츠를 클리어하여 채팅창에 관련 메시지가 출력되면, UI 상의 완료 횟수가 실시간으로 자동 증가합니다.

### ⚙️ 커스텀 숙제 기능 고도화
* **커스텀 숙제 최대 횟수(maxCount) 설정**: 사용자가 직접 등록하는 커스텀 주간 숙제에서도 최대 수행 횟수를 마음대로 설정할 수 있도록 기능이 완전히 연동되었습니다.
* **최대 횟수 변경 시 안전 보정**: 커스텀 숙제 수정 과정에서 최대 횟수를 기존 완료 횟수보다 낮추더라도, 데이터 오버플로우가 나지 않도록 한도 초과분을 최대치에 맞춰 안전하게 자동 보정합니다.

### ⏱️ 신규 버프 타이머 5종 추가 및 제어 개선
자주 사용하는 핵심 소모품 버프 5종을 추가하고 타이머 관리 편의성을 향상했습니다.
* **버프 5종 추가**: 로토의 부적, 속성 앰플, 이자벨의 비법(데미지), 이자벨의 비법(비율능력치), 일루미네이션 축제 음료 버프의 아이콘과 잔여 시간 타이머가 연동됩니다.
* **버프 수동 비활성화**: 오버레이에서 활성화된 버프 배지를 클릭하여 수동으로 즉시 비활성화할 수 있도록 IPC 채널 및 제어 로직을 추가했습니다.

## 🛠 상세 내역

### Added (추가)
* 신규 버프 5종 아이콘 이미지 및 데이터 연동 (`로토의 부적.png`, `속성앰플.png`, `이자벨의 비법_데미지.png`, `이자벨의 비법_비율능력치.png`, `일루미네이션축제음료.png`)
* 버프 타이머 수동 비활성화 IPC 채널 및 프론트엔드 연동 (`buffTimerDeactivate`)
* 커스텀 숙제 등록/수정 시 최대 수행 횟수(`maxCount`) 선택 및 설정 기능
* 다양한 주간/일일 숙제에 대한 채팅 로그 기반 자동 감지(ECLIPSE, MERCURIAL, CORE_MASTER 등) 이벤트 추가

### Changed (변경)
* `xpTracker.ts` 내 경험의 심장 외 5종의 신규 버프 사용 감지 로직 업데이트
* 신조의 둥지 최대 수행 횟수 수정
* `contents.json` 기본 숙제 설정 보완 및 자동 체크 플래그(`auto`) 지원 여부 반영
* `preload.ts` 및 `ipcHandlers.ts` 내 API 호출 시그니처에 `maxCount` 선택적 매개변수 추가

### Fixed (수정)
* `contentsChecker.ts` 내 일일 숙제로 변경 시 `maxCount` 필드 가비지 삭제 로직 및 한도 초과분 보정 로직 보완
* `contents-checker.html` 내 중복 닫는 괄호(`};`) 문법 오류 해결
