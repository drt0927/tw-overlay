# TW-Overlay v1.0.5 Release Note

## 🚀 대규모 아키텍처 개편 및 기능 고도화

### 1. TypeScript 전환 및 프로젝트 구조 최적화
- **완전한 TS 마이그레이션**: 전 소스 코드를 TypeScript로 전환하여 타입 안전성 확보.
- **4분할 독립 창 시스템**: 사이드바(메인), 오버레이, 설정, 갤러리를 독립된 윈도우로 분리하여 리소스 관리 최적화.
- **빌드 자동화**: 리소스 복사 및 컴파일 프로세스를 통합하여 배포 편의성 향상.

### 2. 혁신적인 환경 설정 UI (와이드 대시보드)
- **1000px 와이드 레이아웃**: 시원한 화면에서 설정을 한눈에 관리.
- **탭 네비게이션**: 일반 설정, 갤러리 알림, 퀵슬롯 관리를 카테고리별로 분리.
- **무한 스크롤 아이콘 피커**: 1,400여 개의 Lucide 아이콘을 전체 검색 및 무한 스크롤로 탐색 가능.
- **퀵슬롯 드래그 앤 드롭**: 마우스 드래그로 링크 순서를 간편하게 변경.

### 3. 지능형 갤러리 모니터링
- **키워드 필터링**: 사용자가 등록한 단어가 포함된 게시글만 알림 (정규표현식 기반).
- **실시간 데이터 동기화**: 게시글 리스트 및 감시 중인 글의 댓글 수를 실시간 업데이트.
- **네트워크 에러 복구**: 서버 연결 끊김 시 시각적 알림 및 자동 재시도 로직 강화.

### 4. 사용자 경험(UX) 및 성능 최적화
- **자석 기능 고도화 (1.0.4 계승)**: 가장 안정적이었던 1.0.4의 좌표 계산식을 TS로 이식하여 부드러운 위치 추적 실현.
- **개별 위치 기억**: 모든 독립 창이 각자의 마지막 위치(상대 좌표)를 계층형 객체 구조(`positions`)로 기억.
- **반응성 향상**: `acceptFirstMouse` 적용으로 게임 중 클릭 한 번에 반응하도록 개선.
- **자동 실행**: 윈도우 시작 시 자동 실행 설정 지원.
- **아이콘 테마**: 눈 아이콘을 브라우저 정체성을 가진 `layout-template`으로 교체 및 보라색 테마 적용.

### 5. 편의 기능
- **ESC 닫기**: 팝업 성격의 창(설정, 갤러리)을 ESC 키로 즉시 종료.
- **URL 자동 보정**: 주소창 입력 시 `https://` 누락 건 자동 처리.
- **트레이 아이콘**: 시스템 트레이를 통한 간편한 종료 기능.
