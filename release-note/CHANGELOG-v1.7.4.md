## [1.7.4] - 2026-03-23

### Changed
- **필드 보스 알림 UI 간소화:**
  - 화면 공간을 절약하고 시각적인 방해를 줄이기 위해 필드 보스 알림 팝업(Toast)을 심플한 한 줄 텍스트 형태로 디자인을 변경했습니다.
  - 알림 팝업이 사이드바의 "필드보스 알림 설정" 버튼 바로 옆에 위치하도록 위치를 동적으로 최적화했습니다.
  
### Enhanced
- **보스 사운드 미리보기 디테일 향상:** 
  - 환경설정 메뉴에서 보스의 알림음을 미리듣기로 재생할 때, 알림창에 "미리보기"라는 공통 텍스트 대신 현재 선택한 보스의 실제 이름이 실시간으로 표시되도록 개선했습니다.
