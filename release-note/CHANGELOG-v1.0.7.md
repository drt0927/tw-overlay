# 📋 Release Note - v1.0.7

## 🚀 개요
이번 버전은 앱의 **전반적인 구동 성능 개선**과 **사용자 편의성 강화**에 집중했습니다. 이제 더 이상 웹 라이브러리 로딩을 기다릴 필요가 없으며, 앱 내에서 간편하게 최신 버전을 유지할 수 있습니다.

## ✨ 주요 변경 사항

### ⚡ 성능 최적화 (Speed & Efficiency)
- **로컬 에셋 전환**: 기존 외부 CDN(Tailwind, Lucide) 의존성을 제거하고 프로젝트 내에 내장하여, 창을 열 때마다 발생하던 수 초간의 지연을 해결했습니다.
- **창 관리 로직 고도화**: 설정 및 갤러리 창의 초기화 프로세스를 최적화하여 즉각적인 응답성을 확보했습니다.

### 🆙 자동 업데이트 시스템 (Auto-Update)
- **통합 UI 구현**: '환경설정 > 앱 정보' 메뉴에서 현재 버전을 확인하고 수동으로 업데이트를 체크할 수 있는 기능을 추가했습니다.
- **백그라운드 다운로드**: 새 버전 발견 시 사용자의 동의 하에 백그라운드에서 안전하게 다운로드를 진행하며, 진행률을 실시간으로 표시합니다.
- **원클릭 설치**: 다운로드 완료 후 버튼 클릭 한 번으로 앱을 재시작하고 즉시 설치를 완료합니다.

### 🎨 UX 및 디자인 개선 (Visual Experience)
- **스플래시 화면(Splash Screen) 도입**: 앱 구동 초기, 시스템 초기화(PowerShell 활성화 등) 과정 동안 로고와 로딩 상태를 보여주는 화면을 추가했습니다.
- **브랜드 아이콘 통합**: 스플래시 화면과 앱 정보 메뉴에 공식 아이콘(`.ico`)을 배치하여 디자인 일관성을 높였습니다.
- **즉시 피드백**: 업데이트 버튼 클릭 시 즉각적인 텍스트 변화를 주어 사용자 불안을 해소했습니다.

### 🛠 시스템 안정성 및 빌드 개선 (Robustness)
- **빌드 시스템 현대화**: 윈도우 전용 PowerShell 복사 로직을 Node.js 기반 `copy-resources.js`로 교체하여 빌드 안정성을 확보했습니다.
- **운영 로그 최적화**: 디버깅용 불필요한 로그를 정리하여 로그 파일 비대화를 방지하고 핵심 오류 추적을 용이하게 했습니다.
