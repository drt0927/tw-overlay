# 📦 TW-Overlay Release Note [v1.11.2]

v1.11.2 패치는 버프 물약 관리를 자동화하는 **버프 타이머 알림 시스템**을 새롭게 도입하고, 채팅 로그 파일의 자동 정리 기능과 트레이 메뉴 동기화 버그를 수정하는 데 집중했습니다.

## 🚀 Added (신규 기능)
- **버프 타이머 알림 시스템**: 채팅 로그를 실시간으로 분석하여 버프 물약 사용(경험의 심장, 레어의 심장, 퇴마사의 은총)을 자동으로 감지합니다. 만료 전 사용자가 설정한 시점마다 청각(알림음) 및 시각(게임 화면 HUD) 경고를 제공합니다.
  - 게임 화면 위에 24×24 컴팩트 아이콘 뱃지 + 원형 프로그레스 링으로 남은 시간 표시
  - Normal(퍼플) / 경고(노란 glow) / 임박(빨간 Pulse) 3단계 페이즈 시각화
  - HUD 위치를 화면 픽셀 단위로 자유롭게 설정 가능
  - 설정창에서 버튼 한 번으로 HUD 미리보기(테스트) 가능
  - ※ 동작 조건: 게임 내 설정 > 게임 > **채팅 로그** 옵션 활성화 필요
- **채팅 로그 자동 삭제**: 환경설정에서 보존 기간(일)을 지정하면, 해당 기간을 초과한 채팅 로그 파일을 앱 시작 시 및 매일 자정에 자동으로 삭제합니다. 기본값은 0(비활성화)입니다.

## 🎨 Changed (변경 및 개선)
- **트레이 메뉴 전체 항목 표시**: 기존에 트레이 우클릭 메뉴에서 누락되던 항목들(외침 내역, 커스텀 알림, 버프 타이머, 색상 필터 등)이 이제 모두 표시됩니다.
- **트레이 메뉴 설정 동기화**: 환경설정에서 숨긴 사이드바 메뉴가 트레이에서도 자동으로 제외되며, 설정 저장 시 트레이 메뉴가 즉시 갱신됩니다.

## 🐞 Fixed (버그 수정)
- **게임 최소화 시 오버레이 미숨김 버그**: 테일즈위버 창을 최소화하거나 종료했을 때 게임 전용 오버레이(HUD)가 화면에 계속 남아있던 문제를 수정했습니다. 이제 게임 상태에 맞춰 오버레이가 자동으로 표시/숨김 처리됩니다.
