# 📦 TW-Overlay Release Note (v1.10.4)

이번 버전에서는 앱 전체의 사용자 경험을 한 단계 높여주는 **커스텀 마우스 커서 시스템**과 상황별로 차별화된 **사운드 피드백**, 그리고 더욱 친절해진 **필드보스 설정 UI**가 적용되었습니다.

## ✨ 신규 기능 (Added)
- **🖱️ 커스텀 마우스 커서 시스템 도입**
    - 앱 전역에 테일즈위버의 감성을 담은 Windows Aero 스타일의 커스텀 커서를 적용했습니다.
    - 클릭, 로딩, 링크 이동 등 상태별 애니메이션 커서가 동작하여 시각적 즐거움을 더합니다.
- **🔊 강화된 사운드 피드백**
    - **일일 숙제 완료**: 모든 일일 숙제를 마치는 순간 경쾌한 **"와우!"** 소리가 재생됩니다.
    - **주간 숙제 완료**: 한 주간의 노력을 축하하는 **"Max Affection"** 사운드로 성취감을 높였습니다.
    - **보스 처치 기록**: 보스 설정 화면에서 시간을 클릭해 일지를 기록할 때 효과음이 출력됩니다.

## 🎨 UI/UX 개선 (Improved)
- **⚔️ 필드보스 설정 화면 고도화**
    - **가이드 배너**: 상단에 "모험일지 자동 기록" 안내 문구를 추가하여 기능 접근성을 높였습니다.
    - **상태별 아이콘**: 시간 배지에 **칼(⚔️)**, **체크(✅)**, **시계(🕒)** 아이콘을 적용하여 현재 상태를 직관적으로 파악할 수 있습니다.
    - **액션 툴팁**: 각 버튼에 마우스를 올리면 수행될 동작(기록하기, 취소하기 등)을 툴팁으로 안내합니다.

## 🛠️ 버그 수정 (Fixed)
- **⚙️ 환경 설정 사운드 관리 안정화**
    - 볼륨 슬라이더 조절 시 숫자가 즉시 반영되지 않던 오류를 수정했습니다.
    - 음소거(Mute) 버튼이 정상적으로 동작하며, 설정값이 안전하게 저장되도록 개선했습니다.
- **✅ 숙제 체크리스트 로직 보완**
    - 100% 완료 시 효과음이 중복으로 재생되던 문제를 해결했습니다.
    - 코드 내 불필요한 레거시 조건(`globalMute`)을 제거하여 성능을 최적화했습니다.

---
**업데이트 방법:** 앱을 재시작하면 자동으로 업데이트 확인 및 설치가 진행됩니다. (또는 환경 설정 > 앱 정보 메뉴 이용)
