# 📦 TW-Overlay Release Note - [Mandatory Update] v2.1.2 (2026-07-12)

이번 v2.1.2 버전에서는 사용자가 고유의 사운드 파일(.mp3, .wav 등)을 알림 음원으로 직접 등록하고 관리할 수 있는 전용 탭 환경을 제공하고, 로컬 설정 유실에 대비한 안전 폴백 오디오 재생 구조를 설계했습니다. 또한 장비 사전 내 무기 및 갑옷 탭의 여러 이미지 오매칭 및 깨짐, 이름 오기 오류를 적극적으로 수정 교정하였습니다.

---

## 🚀 주요 변경 사항

### 🎵 Added (새로운 기능)
- **커스텀 알림음 관리 독립 서브 탭 신설**:
  - 환경설정 - [소리 및 알림 설정] 영역 상단에 독립된 **[커스텀 알림음]** 가로 서브 탭을 신설하여, 소리 설정과 분리된 쾌적한 전용 관리 창을 제공합니다.
  - **로컬 백업 영구화**: 사용자가 추가한 오디오 파일은 원본이 변경/삭제되더라도 정상 작동할 수 있도록 `AppData/Roaming/twOverlay/custom_sounds` 폴더로 자동 복제 관리됩니다.
  - **오디오 유실 대비 폴백(Fallback) 보강**: 설정 가져오기/내보내기 등을 통해 로컬 파일이 물리적으로 유실되었을 경우 발생할 수 있는 오디오 에러를 사전에 방지하고자, 존재 여부 판단 후 자동 기본 알림음(`orb.mp3`)으로 부드럽게 대체 재생되도록 특수 예외 처리를 연동했습니다.

### 🕒 Changed (개선 및 변경)
- **알림 기록 데이터베이스 24시간 청소 규칙 적용**:
  - 오버레이 로그의 효율적인 용량 관리 및 데이터 부하 완화를 위해, 기존 7일간 보존하던 알림 기록 자동 삭제 정리 주기를 **24시간(1일)**로 단축 수정하였습니다.

### 🔍 Fixed (버그 수정 및 정돈)
- **장비 사전 내 무기/손목 이미지 오매칭 및 깨짐 복구**:
  - **해머**: 물리적으로 없는 파일명(`_4.jpg`)을 지목해 깨지던 이클립스 해머 이미지를 실존 파일(`_5.jpg`)로 정상 복원하고, 신규 세크리드 해머/改-세크리드 해머의 이미지 누락을 정정(공백 지정)했습니다.
  - **창(스피어)**: 누락된 `_4.jpg` 대신 실제 존재하는 이클립스 스피어 이미지(`_5.jpg`)를 정합 매칭하고, 세크리드 스피어의 이미지를 공백으로 올바르게 분리했습니다.
  - **단검(대거)**: 깨져 보이던 이클립스 대거에 실제 리소스 이미지인 `_5.jpg`를 매칭하고, 세크리드 대거는 이미지가 없으므로 비웠습니다.
  - **손목(스펠북)**: 깨져 보이던 이클립스 스펠북에 실제 리소스 이미지인 `_5.jpg`를 매칭하고, 세크리드 스펠북은 이미지를 비워 오매칭을 정정했습니다.
  - **손목(펜듈럼)**: 깨지던 이클립스 펜듈럼을 포함해 모든 펜듈럼 장비군의 이미지를 디렉토리에 존재하는 유일한 리소스(`_5.jpg`)로 일괄 통일 지정하여 깨짐을 전부 종식했습니다.
- **장비 사전 내 명칭 및 중복 이미지 복원**:
  - **아밍소드**: 세크리드 아밍소드의 깨지던 이미지를 改-세크리드 아밍소드가 사용하는 실제 리소스(`_6.jpg`)와 공유하도록 맞춰 두 아밍소드의 이미지를 완전 복원했습니다.
  - **리스트(손목)**: 세크리드 리스트의 깨지던 이미지를 改-세크리드 리스트가 사용하는 실제 리소스(`_7.jpg`)와 공유하도록 맞춰 두 리스트의 이미지를 완전 복원했습니다.
  - **改-세크리드 해머 명칭 정정**: 310제 세크리드 해머 改 버전의 이름이 "세크리드 해머"로 이름이 중복 노출되던 문제를 **`"改-세크리드 해머"`**로 올바르게 정정했습니다.
- **설명용 텍스트 레코드 소거**:
  - 갑옷 아노마라드 공화국/왕국 시리즈 및 세트 데모닉 장비 세트 아래에 장비 사전 목록의 가독성을 해치며 기재되어 있던 옵션 설명 텍스트 레코드(`*찌르기/베기/마공/마방 중 하나의 옵션만 선택 가능` 등)를 완전히 삭제하여 사전의 데이터를 깔끔히 정돈했습니다.
