# TW-Overlay v1.19.3 릴리즈 노트 (2026.06.28)

이번 업데이트는 **사냥터 동선 시뮬레이터 커스터마이징 개선 및 SQLite 데이터베이스 완전 통합**, 그리고 **알림 기능들의 기본 알람음 단일화 및 사운드 볼륨 크기 일괄 하향 조치**가 적용되었습니다.

---

## 🚀 주요 변경 사항

### 1. 🗺️ 사냥터 동선 시뮬레이터 기능 고도화 (SQLite 연동 및 편의성 개선)
* **SQLite 로컬 DB 완전 연동:** 
  * 개별 선(세그먼트)마다 지정한 커스텀 색상 데이터를 로컬 DB `hunting_paths` 테이블에 `color` 컬럼을 동적으로 생성(안전 마이그레이션 적용)하여 영속 보관하도록 개선했습니다.
  * 기존 `localStorage` 기반 관리 방식을 제거하여 데이터 저장 아키텍처의 파편화를 방지하고 신뢰성을 올렸습니다.
* **커스텀 옵션 컨트롤바 탑재:**
  * 동선 옵션 바를 신설하여 **진행 방향 화살표 표시 토글, 시작-끝 루프 연결 설정, 글로벌 기본 선 색상** 옵션을 실시간 제어할 수 있게 했습니다. (기본 설정은 `localStorage`에 영구 보존됩니다.)
* **세그먼트별 개별 화살표 색상 일체화:**
  * 개별 선의 색상이 변하면 진행 방향 화살표 마커의 색상도 선 색상에 맞춰 실시간으로 동적으로 생성 및 매핑되도록 처리하여 시각적 직관성을 향상했습니다.
* **노드 조작 편의성 보강:**
  * 노드를 마우스로 드래그하여 좌표를 이동할 때 개별 지정한 선 색상 데이터가 깨지지 않고 그대로 계승됩니다.
  * 선 중간에 신규 노드를 삽입(`Ctrl+클릭`)하면 이전 구간의 색상이 분할된 뒤의 구간에도 고스란히 승계되도록 처리했습니다.

## 🛠️ 주요 버그 수정 및 사용성 개선 (Fixed & Improved)
* **경로 선 투명도 완전 제거 (시인성 극대화):**
  * 기존 CSS 파일에 박혀 있던 `opacity: 0.6` (투명도 60%) 설정을 완전히 제거하여, 맵 위에서 선 색상이 본래의 뚜렷한 원색(100% 불투명)으로 렌더링되게 했습니다. 이에 따라 불투명하던 방향 화살표와의 이질감이 해결되고 어두운 맵 내 시인성이 크게 올라갔습니다.
* **커스텀 색상 실시간 드래그 피드백 지원:**
  * 미니 팝업 내 커스텀 컬러 피커를 드래그할 때 수없이 발생하는 이벤트로 인해 팔레트가 강제로 꺼지던 현상을 방지하도록 제어 분기(`shouldClosePopup` 파라미터)를 적용했습니다. 이제 **컬러 다이얼로그 안에서 드래그 조작을 할 때 선 색상이 실시간으로 변하는 모습을 관찰**할 수 있습니다.
* **팝업창 화면 가장자리 잘림 방지 (경계선 안착 필터):**
  * 팝업을 띄울 때 실제 팝업 크기와 창 크기를 실시간 연산하여, 창의 우측 끝부분이나 하단 구석에서 우클릭을 하더라도 잘리지 않고 창 내부로 안전하게 자동으로 밀려 정렬되도록 보정했습니다.
* **선 색상 변경 팝업 마우스 이벤트 꼬임 수정:**
  * 팝업 외곽 영역 클릭 시 팝업을 닫는 전역 `mousedown` 이벤트에 의해 컬러 변경 버튼의 클릭이 유실되던 문제를 `onmousedown` 연동 및 `setTimeout` 0ms 지연 처리를 통해 원천 방지했습니다. 또한 팝업을 `fixed` 포지션으로 수정하여 좌표 오차를 없앴습니다.
* **팔색조의 언덕 5회 완료 알람 로직 단순화:**
  * 보스 처치 수 누적 카운트나 클리어 패키지 이벤트 분석 누락에 따른 알림 유실 문제를 해결하기 위해, 5회차 진입 시 무조건 출력되는 남은 에너지 `16`(SS 등급) 감지 즉시 오버레이 알림이 출력되도록 단순화했습니다.
* **색상 프리셋 버튼 크기 확대 및 UX 개선:**
  * 프리셋 선택 버튼이 너무 작아 클릭하기 피로하다는 의견에 맞춰 하단 글로벌 프리셋 버튼은 **`24px`**, 우클릭 미니 팝업 버튼은 **`28px`**로 크기를 키우고 여백과 모서리 둥글기를 조화롭게 개편했습니다.
* **단축키 안내 배너 설명 추가:**
  * 하단 가이드 배너에 `선 색상 변경: 맵 위 경로 선을 마우스 우클릭` 문구를 추가하고, 선 색상 동적화에 맞추어 기존 `경로 선(초록색)` 설명을 `경로 선`으로 일반화했습니다.

## 🔊 기본 알람음 통일 및 볼륨 일괄 하향 (사운드 개선)
* **알림음 `orb.mp3`로 대통합:**
  * 기본 설정에서 보스 출현 6종(골론, 파멸의 기원, 스페르첸드, 골모답, 아칸, 혼란한 대지)의 알림음 기본값으로 사용되던 `voice_boss_first.wav`를 **`orb.mp3`**로 변경했습니다.
  * 또한 사기꾼 감지 알람음의 기본값으로 지정되어 있던 공습경보(`gongseubgyeongbo-gongseubgyeongbo.mp3`)도 **`orb.mp3`**로 변경했습니다.
* **기본 볼륨 크기 일괄 하향 패치:**
  * 갑작스러운 높은 음량에 따른 자극을 낮추기 위해 프로그램 전반의 기본 볼륨 팩을 조정했습니다:
    * **단어 알림 / 정수 미감지 / 웨이브 경고 / 버프 타이머:** 기본 볼륨 70% → **40%**
    * **필드 보스 출현 / 계산기 완료 / 숙제 체커:** 기본 볼륨 50% → **30%**
    * **사기꾼 감지 알림:** 기본 볼륨 100% → **50%** (미리보기 볼륨 80% → **50%**)

## 📥 설치 및 적용 방법

1. 본 릴리즈 페이지 최하단의 **Assets** 섹션에서 `twOverlay-Setup-1.19.3.exe` 파일을 클릭하여 다운로드합니다.
2. 다운로드 완료 후 실행하시면 자동으로 기존 오버레이가 최신 v1.19.3 사양으로 덮어쓰기 업데이트됩니다.
3. 최초 실행 시, 사냥터 동선 및 바뀐 기본 알람음 볼륨 설정이 오차 없이 안정적으로 구성됩니다.
