# TW-Overlay v1.19.2 릴리즈 노트 (2026.06.26)

이번 업데이트는 **사냥터 동선 시뮬레이션의 점(노드)에 순서 번호 표시 도입** 및 노드 크기 최적화를 통한 시인성 개선이 완료되었습니다.

## 🛠️ 주요 변경 사항 (CHANGELOG)

### Added (신규 기능)
* **사냥터 동선 시뮬레이션 순서 번호 표시**:
  * 동선의 순서 흐름과 진행 방향을 한눈에 파악할 수 있도록, 맵 상에 그려지는 각 점(노드) 내부 중앙에 **1부터 순차적인 숫자 번호**가 표기되도록 기능을 추가했습니다.
  * 숫자가 선명하게 드러나며 어두운 맵 배경 대비 최적의 가독성을 갖도록, 점의 크기(반지름)를 대폭 확대(기존 `4.5` -> `8`)하고 글자 명암비와 굵기를 세밀하게 조율했습니다.
  * 마우스 호버(Hover) 또는 점을 드래그 중인 활성화(Active) 상태에서도 노란색, 빨간색 등 배경 상태에 구애받지 않고 숫자를 명확하게 읽을 수 있도록 세련된 타이포 스타일을 적용했습니다.

---

## 📥 설치 방법

1. 본 릴리즈 페이지 최하단의 **Assets** 섹션에서 `twOverlay-Setup-1.19.2.exe` 파일을 클릭하여 다운로드합니다.
2. 기존에 실행 중이던 `tw-overlay` 앱이 있다면 반드시 완전히 종료한 뒤 설치 파일을 실행합니다.
3. 화면의 안내에 따라 설치를 완료하면 새로운 기능과 패치가 적용됩니다.
