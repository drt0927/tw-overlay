# 📦 TW-Overlay Release Note [v1.11.2]

v1.11.2 패치는 버프 물약 관리를 자동화하는 **버프 타이머 알림 시스템**과 **경험치 HUD 전용 메뉴**를 새롭게 도입하고, 채팅 로그 파일 자동 정리·트레이 메뉴 동기화 개선 및 여러 안정성 버그를 수정합니다.

## 🚀 Added (신규 기능)

- **버프 타이머 알림 시스템**: 채팅 로그를 실시간으로 분석하여 버프 물약 사용(경험의 심장, 레어의 심장, 퇴마사의 은총)을 자동으로 감지합니다. 만료 전 사용자가 설정한 시점마다 청각(알림음) 및 시각(게임 화면 HUD) 경고를 제공합니다.
  - 게임 화면 위에 24×24 컴팩트 아이콘 뱃지 + 원형 프로그레스 링으로 남은 시간 표시
  - Normal(퍼플) / 경고(노란 glow) / 임박(빨간 Pulse) 3단계 페이즈 시각화
  - HUD 위치를 화면 픽셀 단위로 자유롭게 설정 가능
  - 설정창에서 버튼 한 번으로 HUD 미리보기(테스트) 가능
  - ※ 동작 조건: 게임 내 설정 > 게임 > **채팅 로그** 옵션 활성화 필요

- **경험치 HUD 전용 메뉴** (`activity` 아이콘): 사이드바에 경험치 HUD 독립 메뉴를 추가했습니다.
  - 현재 세션 총 획득 XP, 분당 경험치(EPM), 경과 시간, 세션 시작 시각 실시간 표시
  - **세션 초기화 버튼**: 누적 경험치와 EPM을 즉시 0으로 리셋 (플래시 피드백 포함)
  - HUD 표시 여부 토글 (변경 즉시 자동 저장)
  - HUD 위치(Left/Bottom) 설정 및 즉시 적용
  - 기존 `settings.html`의 XP HUD 설정이 이 메뉴로 이전됨

- **채팅 로그 자동 삭제**: 환경설정에서 보존 기간(일)을 지정하면, 해당 기간을 초과한 채팅 로그 파일을 앱 시작 시 및 매일 자정에 자동으로 삭제합니다. 기본값은 0(비활성화)입니다.

## 🎨 Changed (변경 및 개선)

- **사운드 채널 범용화**: 기존 `play-boss-sound` 채널을 `play-sound`로 통합했습니다. 필드보스 알림, 버프 타이머 경고, 커스텀 알림 등 모든 사운드 기능이 단일 채널을 공유합니다. 페이로드의 `bossName` 필드가 범용 `label`로 변경되었습니다.
- **버프 타이머 성능 최적화**: 1초마다 반복 실행되던 `config.load()` (디스크 I/O)를 `_cachedWarnSeconds` 필드로 캐싱하여 제거했습니다. 설정 변경 시(`applySettings`) 캐시가 자동으로 갱신됩니다.
- **트레이 메뉴 전체 항목 표시**: 기존에 트레이 우클릭 메뉴에서 누락되던 항목들(외침 내역, 커스텀 알림, 버프 타이머, 경험치 HUD, 색상 필터 등)이 이제 모두 표시됩니다.
- **트레이 메뉴 설정 동기화**: 환경설정에서 숨긴 사이드바 메뉴가 트레이에서도 자동으로 제외되며, 설정 저장 시 트레이 메뉴가 즉시 갱신됩니다.

## 🐞 Fixed (버그 수정)

- **게임 최소화 시 오버레이 미숨김 버그**: 테일즈위버 창을 최소화하거나 종료했을 때 게임 전용 오버레이(HUD)가 화면에 계속 남아있던 문제를 수정했습니다.
- **`chatLogProcessor` 이벤트 핸들러 구조 오류**: `TRADE_SHOUT`, `XP_CHANGED`, `BUFF_USED` 이벤트 핸들러가 `start()` 메서드 외부에 잘못 배치되어 등록되지 않던 버그를 수정했습니다.
- **`windowManager` 크래시 방지**: 게임 오버레이 창 복귀 시 `isVisible()` 호출 전 `isDestroyed()` 체크를 추가하여 창이 이미 파괴된 상태에서 발생하던 크래시 가능성을 제거했습니다.
- **경험치 HUD 전송 최적화**: `XP_CHANGED` 이벤트에서 `game-overlay`와 `xp-hud` 두 창 모두 안전하게 수신하도록 개선했으며, 이미 파괴된 창에 IPC 전송 시 예외 처리를 추가했습니다.
