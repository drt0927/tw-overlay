# TW-Overlay v1.18.0 릴리즈 노트 (2026.06.12)

이번 업데이트는 테일즈위버 전체 전투 장비 스펙 검색 및 슬롯별 비교/분석이 가능한 **장비 사전(Equipment Dictionary) 창을 신규 출시**하고, 이를 캐릭터 데미지 **계수 계산기 및 진화 비용 계산기와 긴밀히 연계**하는 연동 기능을 탑재했습니다. 아울러 계수 계산기 화면의 레이아웃을 **우측 세로 2분할 와이드 디자인으로 전면 리뉴얼**하고 수식 공식 오류들을 정밀 검증/보정하였습니다.

---

## 🌟 Added (신규 기능)

- **장비 사전(Equipment Dictionary) 윈도우 신설**:
  - 게임 내 모든 부위별 장비의 기본 능력치 범위, 보정 한계치, 레벨 제한, 장착 제한 및 딜레이 정보를 쉽고 빠르게 검색 및 비교할 수 있는 별도 창을 새롭게 추가했습니다.
  - **초성 검색(자음 검색)** 지원: 한글 입력기 매칭 알고리즘을 도입하여 초성(예: `ㅅㅋㄹㄷ` -> `세크리드`)만 쳐도 원하는 장비를 고속 검색할 수 있습니다.
  - **장비 비교 매트릭스 (최대 4개)**: 동일 부위의 장비 최대 4개를 선택해 가로 행렬 형태로 세부 스펙을 동시 대조할 수 있습니다. 각 항목의 한계치 값 중 가장 높은 값은 **금색 텍스트와 배경 강조 표시**를 제공해 최강의 템셋팅을 가려내는 데 기여합니다.

- **장비 사전 ➡️ 진화 비용 계산기 다이렉트 연동**:
  - 장비 사전에서 업그레이드가 지원되는 대표 등급 장비(아카드, 엔키라, 인퍼널, 아퀼루스, 어비스, 이클립스) 상세 확인 시 「진화 비용 계산」 버튼이 나타납니다.
  - 클릭 시 진화 계산기 창이 포커싱되며, 대상 장비의 부위와 등급 단계가 자동으로 미리 지정되어 즉시 필요 재화와 재료를 계산합니다.

- **장비 사전 ➡️ 계수 계산기 원클릭 장비 주입 연동**:
  - 장비 사전 상세 카드 우측 하단의 「계수 계산기 입력」 버튼을 누르면, 계수 계산기 내 해당하는 장비 슬롯(투구, 갑옷, 무기, 손목 등)에 해당 장비 스펙이 자동으로 등록 및 장착됩니다.
  - 기본 목록에 수록되지 않았던 특수 장비이더라도, 사전 데이터의 한계치(최대 스탯) 및 딜레이 스펙을 추출해 가상의 `<option>`을 즉각 만들어 장착해 줍니다. 장착 시 최종 데미지 계수와 명중 DEX가 백그라운드에서 실시간 재연산되어 유기적으로 맞물립니다.

## 🔧 Improved & Fixed (개선 및 수정)

- **계수 계산기 레이아웃 우측 세로 2분할 와이드 최적화**:
  - 기존 화면 하단에 다소 뭉뚱그려져 있던 콘텐츠 가이드 영역을 우측 고정 **세로 360px** 단일 열 피드로 재배치하는 Flex 가로 2분할 마크업 개편을 진행했습니다.
  - 모니터 해상도 짤림과 답답함을 해결하기 위해 창 규격을 **`1420x760`** (오프셋: `-1430`) 와이드 형태로 전면 최적화하고, 세로 공간 여유 확보에 따라 테이블 td 셀 및 입력창의 세로 패딩(td, ci)을 넉넉하게 확장해 가독성을 대폭 복원했습니다.
  - 좌측 계산 테이블 영역에 세로/가로 스크롤바가 전혀 나타나지 않도록 `overflow-hidden` 밀착 설정을 지정하고, 스크롤바 없는 쾌적한 화면을 보장했습니다. (우측 피드 영역은 `overflow-y-auto` 스크롤바 유지)

- **콘텐츠 가이드 카드 디자인 리뉴얼 (프로그레스 바 탑재)**:
  - 각 콘텐츠 가이드에 목표 수치 대비 현재 계수 도달 비율을 직관적으로 보여주는 얇은 수평 **게이지 바(Progress Bar)** 및 백분율 표시를 새롭게 추가했습니다.
  - 던전별 진입 명중률 도달 여부 배지(`DEX ✔/✘`)와 현재 데미지 스펙 상태(원활/가능/힘듬/불가능)를 표기하며, 상태에 부합하는 LED 스타일의 **상태 도트 인디케이터(Status Dot)**를 부착했습니다.
  - 상태별 색상 테마: 원활(초록), 가능(노랑), 힘듬(주황), 불가능(빨강), 정보부족(회색)

- **계수 계산기 공식 및 가중치 정밀 보정**:
  - 엑셀 계산 수식과의 1:1 대조 검증을 수행하여, **마법베기(maghack)**의 기본/강화 및 캐릭터 스탯 가중치 공식을 물리복합 공식을 참조하도록 수정 완료했습니다.
  - **물리복합 및 마법베기 계열의 코어 가중치를 `28.75`**로 정밀 보정하여 엑셀 최종 합계와 소수점 둘째 자리까지 100% 일치함을 통과(PASS)했습니다.
  - 아바타 기본 세트 스탯(주/부/DEX 각 +15) 및 효과 기본 세트 스탯(주/부 각 +20, DEX +5) 가산 방식을 분리 보정하여 스탯 누락을 완벽히 해결했습니다. (효과 기본 부스탯 기본값을 5로 교정)

- **프로필/데이터 로드 시 장비 드롭다운 동기화 및 복구 핫픽스**:
  - 프로필 전환 및 창 기동 시, 프로그램 방식의 select 값 대입에 맞추어 커스텀 프록시 드롭다운의 대표 이미지와 이름 겉표시가 즉각 갱신되도록 양방향 `change` 이벤트를 강제 발행해 동기화했습니다.
  - 사전에서 가져와 임의 등록 후 프로필에 저장된 커스텀 장비의 경우, 창 재부팅 시 select options 목록에 없어 유실(장착 안함 초기화)되던 예외를 방어하기 위해 옵션 자동 동적 복구 로직을 구현했습니다.

- **안내 가이드 및 물음표(?) 도움말 툴팁 아이콘 일괄 적용**:
  - 마우스 호버 설명이 숨겨져 있는 추가 보너스 라벨(아바타, 커프, 효과, 렐릭) 및 세부 지역 코어 라벨(머큐리얼, 어비스, 이클립스, 루비코나) 우측에 **물음표 도움말 아이콘(`help-circle`)**을 일괄 배치하여 유저 시인성을 대폭 끌어올렸습니다.
  - 테이블 열 너비(colgroup) 설정을 최적화하여 "계수 기여" 열 너비를 100px로 제한하고, 장비명 선택 드롭다운 영역 가로폭을 시원하게 확보했습니다.
  - 코어 세부 입력란 placeholder에 "주 스탯" 입력 가이드를 명시하고, 코어 가이드 텍스트 폰트 크기 및 글자 색상 대비를 극대화했습니다.

- **"어비스" 등급 방어구 데이터 복구**:
  - 스프레드시트 HTML의 여백 열들로 인해 아이템 행 시작점 판정이 무시되던 파싱 스크립트(`parse-equipment.js`)의 정규 탐색 범위를 수정하여, 누락되었던 **어비스 등급 방어구 5종**(어비스 헬름, 아뮬렛, 윙, 건틀렛, 부츠) 데이터를 완벽히 복구 및 계산기 리스트에 재수록 완료했습니다.

- **효과(Effect) 장비 부위 입력란 정상화**:
  - 효과 부위를 세트효과 보너스로 오인하여 하드코딩(Span)했던 기본 주/부스탯(20/5) 자리를 유저가 자신의 효과 템 스펙에 따라 자유롭게 편집할 수 있도록 일반 숫자 입력 필드로 복구하였습니다. 툴팁 가이드 역시 슬롯 정보에 부합하도록 교정하였습니다.

- **코드 리뷰 피드백 기반 최적화 및 방어 코드 적용**:
  - **메모리 누수 차단**: 전역 클릭 이벤트 리스너의 중복 등록을 차단하는 플래그 변수를 도입하여 메모리 누수 요인을 해결했습니다.
  - **파싱 연산 가속화**: 장비 주입 시 루프를 돌며 수행되던 `JSON.parse` 연산을 걷어내고 단순 문자열 매칭(`opt.text === item.name`)으로 리팩토링하여 연산 부하를 최소화했습니다.
  - **음수 스탯 방어**: 스탯 변환 헬퍼 `getV`에 `Math.max(0, ...)` 방어 처리를 추가하여 비정상적인 음수 스탯 입력 시 연산 오작동이 발생하는 현상을 방지했습니다.

## 📦 설치 및 업데이트 방법

1. **설치 파일 다운로드**:
   - 본 릴리즈 페이지 최하단의 **Assets** 섹션에서 `twOverlay-Setup-1.18.0.exe` 파일을 클릭하여 다운로드합니다.
2. **프로그램 실행 및 설치**:
   - 다운로드한 `.exe` 파일을 실행하면 자동으로 설치 및 업데이트가 진행됩니다.
   - *참고: Windows SmartScreen 경고가 뜰 경우, `추가 정보` 클릭 후 `실행`을 누르면 됩니다.*
