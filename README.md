# 🎮 TW-Overlay (TalesWeaver Companion Browser)

테일즈위버(TalesWeaver) 플레이어를 위한 **자석형 오버레이 브라우저**입니다. 게임 화면 위에 공략, 유튜브, 커뮤니티 등을 띄워두고 게임과 한 몸처럼 움직이는 편리한 환경을 제공합니다.

## ✨ 주요 기능

*   **🧲 자석 트래킹 (Magnet Tracking):** 게임 창의 위치와 크기를 실시간으로 감지하여, 게임 창 내부에 오버레이가 항상 따라다닙니다.
*   **💜 사이드바 퀵슬롯 (New):** 게임 창 우측에 밀착되는 전용 퀵슬롯 바를 통해 자주 가는 사이트를 원클릭으로 이동하거나 외부 브라우저로 열 수 있습니다. (보라색 호버 글로우 효과 적용)
*   **📏 자동 경계 제한 (Boundary Clamping):** 오버레이가 게임 화면 밖으로 나가지 않도록 자동으로 위치를 보정합니다.
*   **🌗 다크 모드 & Glassmorphism UI:** 테일즈위버의 분위기와 잘 어울리는 세련된 디자인의 대시보드를 제공합니다.
*   **🔗 통합 대시보드:** 즐겨찾기 관리, 투명도 조절, 홈 URL 설정을 한곳에서 처리할 수 있습니다.
*   **⚡ 성능 최적화:** 게임 성능에 영향을 주지 않도록 CPU 및 메모리 점유율을 최소화했습니다.

## 🚀 설치 및 사용 방법

### 1. 설치하기
[Releases](https://github.com/drt0927/tw-overlay/releases) 페이지에서 최신 버전의 `twOverlay-Setup-1.0.4.exe` 파일을 다운로드하여 설치하세요.

### 2. 실행하기
1. 테일즈위버를 실행합니다.
2. `twOverlay`를 실행합니다.
3. 자동으로 테일즈위버 창을 찾아 내부와 우측 끝에 자석처럼 붙습니다.

### 3. 주요 단축키
- **`Ctrl + Shift + T`**: **클릭 투과 모드** 토글 (브라우저 뒤의 게임을 직접 클릭해야 할 때 사용하세요).

### 4. 사이드바 사용법
- 사이드바 최상단의 **`>`** 버튼을 눌러 슬롯을 접거나 펼칠 수 있습니다.
- 슬롯에 마우스를 올리면 사이트 이름과 URL이 포함된 툴팁이 나타납니다.

## 🛠 기술 스택
- **Engine:** Electron
- **UI:** Tailwind CSS, Lucide Icons
- **Tracking:** PowerShell (Win32 API 호출)

## ⚠️ 주의사항
- **넷플릭스(Netflix) 및 일부 OTT 서비스:** DRM 보안 정책으로 인해 브라우저 뷰 내에서 영상 재생이 불가능할 수 있습니다. (유튜브는 정상 작동)
- 자석 기능은 게임이 '창 모드' 또는 '전체 창 모드'일 때만 정상 작동합니다.
