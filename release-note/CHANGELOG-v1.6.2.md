# 📦 TW-Overlay Release Note (v1.6.2)

## 🗓 배포일: 2026-03-01

## ✨ 신규 및 개선 사항 (Added / Changed)

### 1. 오버레이 툴바(상단바) 자동 숨김 (Event-Driven Auto Hide)
- 오버레이 툴바 표시 방식을 마우스 이벤트 기반으로 개선했습니다. 오버레이 창이나 웹뷰에 마우스를 올릴 때만 상단 주소창 툴바가 나타나며, 마우스가 창을 벗어나면 자동으로 숨겨집니다.
- 게임 중 오버레이 툴바가 항상 표시되어 몰입을 방해하던 문제를 해결하고 시야를 넓혔습니다.

### 2. 창 포커스 및 Z-Order 논리 강화
- 사이드바나 오버레이 창(TW-Overlay 본체)을 클릭하여 포커스를 획득할 때, 연동된 테일즈위버 게임 클라이언트도 Z-Order 최상단 그룹으로 동반 상승하도록 개선했습니다.
- 이를 통해 듀얼모니터 환경이나 다중 작업 시 오버레이 클릭 직후 게임 창이 다른 브라우저 등 뒤로 가려버리는 현상을 방지합니다.

## 🛠 버그 수정 (Fixed)
- 마우스 투과(Click-through) 모드로 전환 시, 화면에 남아있던 오버레이 툴바가 즉시 숨겨지지 않던 버그를 수정했습니다.

---
**Note:** 이 패치는 오버레이 사용성과 다중 창 환경에서의 편의성을 극대화한 개선입니다.
