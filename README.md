# 🎮 TW-Overlay (TalesWeaver Companion Browser)

테일즈위버(TalesWeaver) 플레이어를 위한 **지능형 다중 창 오버레이 브라우저**입니다. 게임 화면과 연동되는 자석형 위젯과 독립된 브라우저 창을 통해 최상의 플레이 환경을 제공합니다.

## ✨ 주요 기능 (v1.1.0 대규모 편의성 업데이트)

*   **🧪 지능형 버프 매니저 (Buff Manager):** 1000px 대화면 기반의 버프 백과사전 및 계산기입니다. 내가 선택한 도핑 아이템의 총 대미지, 능력치, 경험치 효과를 실시간으로 합산해 보여줍니다.
*   **🧩 스마트 도핑 프리셋 (Smart Presets):** 자주 사용하는 도핑 조합을 저장하고 원클릭으로 불러올 수 있습니다. 기본 국룰 도핑 세트도 기본 제공됩니다.
*   **📖 지능형 약어 사전 (Abbreviation Search):** 게임 내에서 사용하는 복잡한 약어들의 풀네임과 상세 설명을 검색 한 번으로 확인하세요.
*   **🚀 성능 강화 (Performance Booster):** 게임 프로세스(`InphaseNXD`) 감지 시 CPU 우선순위를 자동으로 '높음(High)'으로 설정하여 프레임 안정성을 향상시켰습니다.
*   **🔋 지능형 폴링 시스템 (Smart Polling):** 게임 창의 위치가 고정될 경우 폴링 주기를 자동으로 늘려 시스템 리소스 소모를 최소화했습니다.
*   **🧲 정밀 자석 트래킹:** 게임 창의 위치를 실시간 감지하여 모든 독립 창이 자석처럼 따라다니며 지정한 위치를 기억합니다.
*   **📢 열린 소통창구:** 앱 정보 화면에 건의 및 버그 제보 링크를 추가하여 사용자의 의견을 실시간으로 수렴합니다.

## 🚀 설치 및 사용 방법

### 1. 설치하기
[Releases](https://github.com/drt0927/tw-overlay/releases) 페이지에서 최신 버전의 `twOverlay-Setup-1.1.0.exe` 파일을 다운로드하여 설치하세요. 기존 사용자는 앱 내 업데이트 기능을 사용할 수 있습니다.

### 2. 실행하기
1. 테일즈위버를 실행합니다.
2. `twOverlay`를 실행하면 자동으로 게임 창 우측에 사이드바가 나타납니다.
3. 사이드바의 각 아이콘을 클릭하여 버프 매니저, 약어 사전, 환경 설정 등을 조작할 수 있습니다.

## 📝 최신 업데이트 내역 (v1.1.0)
- **신규:** 1000px 대화면 기반 지능형 버프 매니저 및 계산기 창 추가.
- **신규:** 게임 내 약어 사전 검색 기능 도입.
- **프리셋:** 사용자 정의 도핑 프리셋 저장/로드 및 기본 도핑 세트 프리셋 지원.
- **UI 개선:** 고해상도 환경 대응을 위한 전체 폰트 크기 상향 및 헤더 드래그 편의성 개선.
- **버그 수정:** 게임창 이동 시 장판 감지(Monitor Zone) 화면이 따라오지 않던 문제 해결.
- **버그 수정:** 갤러리 알림 발생 시 사이드바 아이콘에 레드닷이 표시되지 않던 오류 수정.
- **소통:** 앱 정보 화면에 건의 및 버그 제보를 위한 구글 폼 링크 추가.

## ⚖️ 장판 감지 기능에 관한 안내 (ScreenWatcher)

본 프로젝트는 테일즈위버 게임사의 **매크로 사용 금지 및 관련 처벌 정책**을 전적으로 지지하며, 게임의 공정성과 고유의 가치를 훼손할 의사가 없음을 분명히 밝힙니다.

`ScreenWatcher`(장판 감지) 기능은 사용자가 매크로 방지를 위해 설계된 특정 패턴(장판 등)을 보다 명확하고 직관적으로 인지할 수 있도록 돕는 **시각적·청각적 보조 알림**을 목적으로 개발되었습니다.

개발자는 다음과 같은 원칙을 준수합니다:
1. **자동 회피 금지:** 감지된 정보를 바탕으로 캐릭터를 자동으로 이동시키거나 조작하는 기능은 제공하지 않습니다.
2. **비인가 행위 방지:** 게임 내 데이터나 메모리를 변조하여 게임 시스템에 직접적인 영향을 주는 기능을 일체 구현하지 않습니다.
3. **사용자 인지 보조:** 본 기능은 오직 사용자에게 현재 상태를 알리는 '알림' 역할에만 집중하며, 최종적인 판단과 조작은 항상 사용자의 몫입니다.

**Note:** 이 프로그램은 게임 메모리를 변조하지 않는 안전한 외부 도구입니다.
