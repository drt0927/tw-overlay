# TW-Overlay v1.18.2 릴리즈 노트 (2026.06.16)

이번 업데이트는 **독바(Dock) 모드에서 채팅 오버레이 위치가 화면 중앙으로 튕기며 자꾸 초기화되던 버그**를 수정하고, 불필요한 백그라운드 동기화 루프를 차단하여 **CPU 리소스 소모를 대폭 최적화**했습니다.

## 🛠️ 주요 변경 사항 (CHANGELOG)

### Fixed (버그 수정)
* **사이드바 Dock 모드 시 채팅 오버레이 위치 초기화 버그 해결**: 사이드바를 `dock` 모드로 두고 사용할 때 100ms마다 강제 동기화 루프가 돌면서 오프셋 저장을 방해하고 스냅백시키던 버그를 수정했습니다.
* **게임 창 비활성화 시 해상도 축소 대응 보정**: 테일즈위버 창모드 전체화면에서 오버레이 조작을 위해 포커스를 옮길 때 게임 해상도가 일시적으로 줄어들며 오프셋 좌표가 깨지던 현상을 보정했습니다.
* **독바(Dock) 창 무한 재생성 오류 수정**: 독바가 예기치 않게 닫힐 때 내부 가시성 플래그가 정상적으로 리셋되지 않아 계속 무한 재부팅(무한 부활)되던 버그를 해결했습니다.
* **창 닫기 수명 주기 좌표 오염 방지**: 윈도우 창이 닫히는 과정에서 발생하는 불필요한 `move` 이벤트로 인해 위치 정보가 0, 0 또는 이상한 위치로 덮어씌워지던 현상을 `isClosing` 안전 가드로 방지했습니다.

### Improved (성능 및 사용성 개선)
* **백그라운드 동기화 최적화**: 100ms마다 무작정 감지하던 가시성 확인 루프를 차단하고 오버레이 창들의 켜짐/꺼짐 이벤트 수명 주기에 맞춰 즉시 연동하도록 변경하여, CPU 및 자원 소모를 크게 줄였습니다.

## 📥 설치 방법

1. 본 릴리즈 페이지 최하단의 **Assets** 섹션에서 `twOverlay-Setup-1.18.2.exe` 파일을 클릭하여 다운로드합니다.
2. 기존에 실행 중이던 `tw-overlay` 앱이 있다면 반드시 완전히 종료한 뒤 설치 파일을 실행합니다.
3. 화면의 안내에 따라 설치를 완료하면 새로운 기능과 패치가 적용됩니다.
