# TW-Overlay v1.14.1 릴리즈 노트 (2026.05.26)

이번 업데이트는 경험치 HUD 오버레이 창에 '잡은 몬스터' 마리수를 실시간으로 표기해 주는 기능을 추가하고, 세션 초기화 연동을 개선한 패치 배포입니다.

---

## 🆕 Added (새로운 기능)

- **📊 경험치 HUD 오버레이에 '잡은 몬스터' 표시 추가**:
  - 기존 '총 경험치', '분당 경험치'에 더해, 이번 세션 동안 잡은 몬스터 수를 실시간으로 오버레이 HUD 창에서 바로 확인할 수 있도록 편의성을 강화했습니다.
  - 레이블 색상은 다른 항목들과 조화를 이루도록 `rose-400` 로즈 톤으로 적용했습니다.

---

## 🛠️ Fixed (버그 수정 및 개선)

- **⏱️ 세션 초기화 시 몬스터 처치 수 동기화 개선**:
  - 경험치 세션을 수동 초기화할 때, 오버레이에 전달되는 페이로드에 `kills: 0`을 명시하여 화면에서도 잡은 몬스터 수가 즉각 `0`으로 초기화되도록 처리했습니다.
