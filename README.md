# 🎮 TW-Overlay (TalesWeaver Companion Browser)

테일즈위버(TalesWeaver) 플레이어를 위한 **지능형 다중 창 오버레이 브라우저**입니다. 게임 화면과 연동되는 자석형 위젯과 독립된 브라우저 창을 통해 최상의 플레이 환경을 제공합니다.

## ✨ 주요 기능

*   **🔔 하이브리드 알림:** 게임 화면을 보고 있지 않을 때(최소화/종료) 필드보스 및 커스텀 알림을 **Windows 네이티브 알림(Toast)**으로 즉시 전달합니다.
*   **📖 지능형 모험 일지:** 일지 창을 열면 오늘 날짜가 자동 선택되며, 동일 보스의 중복 기록을 DB 레벨에서 완벽하게 차단합니다.
*   **🖱️ 커스텀 마우스 커서:** 앱 전체에 테일즈위버 감성을 담은 Windows Aero 스타일의 커스텀 애니메이션 커서를 적용했습니다.
*   **🔊 강화된 사운드 피드백:** 숙제 완료, 보스 처치 기록 시 상황에 맞는 효과음이 출력되어 플레이의 즐거움을 더합니다.
*   **📊 월간 모험 리포트 (Infographic):** 이번 달 출석 히트맵, 최애 보스 Top 3, 수익 합계 등 나의 모험을 시각적인 인포그래픽으로 한눈에 파악할 수 있습니다.
*   **⚙️ 독립 유틸리티 모드:** 게임이 꺼져 있어도 시스템 트레이 메뉴를 통해 계산기나 일지 기능을 자유롭게 사용할 수 있습니다.
*   **💾 데이터 백업 및 복구:** 모든 설정과 일지 데이터를 ZIP 파일로 안전하게 백업하고 원클릭으로 복구할 수 있습니다.
*   **🚀 게임 네트워크 최적화 (Fast Ping):** 네이글 알고리즘 제어를 통한 응답 속도 개선.

## 🚀 설치 및 사용 방법

### 1. 설치하기
[Releases](https://github.com/drt_0/tw-overlay/releases) 페이지에서 최신 버전의 `twOverlay-Setup-1.10.5.exe` 파일을 다운로드하여 설치하세요. 기존 사용자는 앱 내 업데이트 기능을 사용할 수 있습니다.

### 2. 실행하기
1. `twOverlay`를 실행하면 테일즈위버 감지 시 자동으로 사이드바가 나타납니다.
2. 게임이 없어도 **시스템 트레이 아이콘** 우클릭 메뉴로 모든 도구를 실행할 수 있습니다.

## 📝 최신 업데이트 내역 (v1.10.5)
- **신규 기능: Windows 네이티브 알림:** 게임 최소화 또는 종료 상태 시 Windows Noti 제공 (필드보스/커스텀 알림).
- **데이터 무결성: 중복 기록 차단:** 동일 보스/스폰 시간 중복 일지 기록 방지 및 UI 피드백('이미 기록됨!') 추가.
- **UX 개선: 모험 일지 자동화:** 모험 일지 열기 시 현재 날짜 자동 선택 기능 추가.
- **UI 보정:** 필드보스 알림 발생 시 사이드바 너비 자동 확장(짤림 방지), 설정 창 높이 최적화, 커서 상속 버그 수정.

## 📝 이전 업데이트 내역 (v1.10.4)
- **신규 기능: 커스텀 마우스 커서:** 앱 전역 애니메이션 커서 시스템 도입.
- **사운드 강화:** 일일/주간 숙제 완료, 보스 기록 시 전용 효과음 적용.
- **UI/UX 개선:** 필드보스 설정 화면 내 가이드 배너 및 상태 아이콘 추가.
