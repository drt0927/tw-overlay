# 🚀 TW-Overlay Release Notes (v1.9.0)

## 📅 출시일: 2026-03-27

---

## 🌟 주요 변경 사항

### 1. 커스텀 알림 (Custom Alerts) 기능 도입
사용자가 직접 알림 시계와 메시지를 설정하여 반복적인 작업을 관리할 수 있습니다.
- **다양한 유형**: 매일 특정 시각(HH:mm) 또는 매시 특정 분(?분)에 울리는 알림 설정 가능.
- **다중 오프셋**: 정각뿐만 아니라 10분 전, 5분 전 등 여러 시점에 미리 알림을 받을 수 있습니다.
- **개별 사운드**: 알림마다 서로 다른 사운드를 지정하여 소리만으로 어떤 알림인지 구분할 수 있습니다.
- **관리 인터페이스**: 전용 창(`custom-alert.html`)에서 모든 알림을 추가/수정/삭제/활성화할 수 있습니다.

### 2. 구글 애널리틱스(GA4) 분석 시스템 표준화 및 최적화
데이터의 정확성을 높이고 앱 부하를 줄이기 위해 분석 아키텍처를 전면 개편했습니다.
- **이벤트 평탄화 (Flattening)**: 중첩된 파라미터 구조를 제거하고 모든 이벤트를 1-depth(예: `boss_time_골론`, `toggle_custom_alert`)로 변경하여 분석 효율을 극대화했습니다.
- **표준 세션 도입**: GA4 표준인 `ga_session_id`와 `ga_session_number`를 도입하여 정확한 방문 통계를 집계합니다.
- **경량화**: 매시간 전송되던 수동 하트비트(Ping) 로직을 제거하고 GA4의 네이티브 세션 측정 방식을 활용하여 네트워크 트래픽을 줄였습니다.

### 3. 설정 UI 통합 및 구조 개선
- **필드보스 설정 통합**: 메인 설정창에 흩어져 있던 보스 관련 공통 옵션(활성화, 오프셋, 음량)을 `필드보스 설정` 패널(`boss-settings.html`)로 일원화했습니다.
- **메뉴 정비**: 사이드바 메뉴 가시성 설정 기능을 개선하여 앱 전반의 일관성을 높였습니다.

### 4. 개발 및 안정성 강화
- **스플래시 화면 버그 수정**: 개발(DEV) 모드에서 게임이 실행 중이지 않을 때 스플래시 화면이 지속되던 문제를 해결했습니다.
- **성능 개선**: 불필요한 이벤트 리스너를 정리하고 IPC 핸들러의 명명 규칙을 표준화했습니다.
