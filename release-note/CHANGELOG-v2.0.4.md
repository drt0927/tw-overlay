# TW-Overlay v2.0.4 릴리즈 노트 (2026.07.04)

이번 업데이트는 캐릭터 계수 계산기의 사용성 향상 및 서브 창 간 Z-Order 포커스 정렬 버그 등을 해결한 **패치 v2.0.4 업데이트**입니다.

---

## 🚀 주요 변경 사항

### ⚡ Added (새로운 기능)

* **도핑 상세 보너스 툴팁 수동 ON/OFF 기능 추가:**
  * 도핑 선택 셀렉터 옆에 `툴팁 ON` / `툴팁 OFF` 텍스트 토글 버튼이 새롭게 추가되었습니다. 
  * 툴팁이 화면을 과도하게 가리는 경우, 버튼을 클릭하여 툴팁의 활성화 상태를 손쉽게 제어할 수 있습니다.

### ⚙️ Changed (기능 개선 사항)

* **계수 계산기 내 DEX 명칭 '명중'으로 완전 일원화:**
  * 계수 계산기 테이블, 요약 정보, 상세 툴팁, 콘텐츠 가이드 등 모든 화면 영역에서 복잡한 영문 표기인 `DEX`를 완전히 제거하고 직관적인 `명중` 단어로 통일하여 유저 사용성을 강화했습니다.
* **도핑 툴팁의 가독성 및 레이아웃 개선:**
  * 도핑 보너스 툴팁에 불투명한 짙은 슬레이트 백그라운드 디자인과 외곽선 그림자(`shadow-2xl`)를 적용해 뒷배경의 글자들과 겹치지 않고 온전히 보일 수 있도록 시인성을 대폭 강화했습니다.
  * 툴팁 내 `찌르기:` 등 3글자 스탯이 좁은 영역 때문에 2줄로 깨져 줄바꿈되는 현상을 해결하기 위해 텍스트 너비를 `w-10 whitespace-nowrap`으로 개선했습니다.

### 🐛 Fixed (버그 수정 사항)

* **서브 창 간 클릭/포커스 시 Z-Order 우선순위 정렬 버그 수정:**
  * 설정, 계산기 등 여러 개의 서브 창이 동시에 띄워져 있을 때, 클릭해서 포커스된 창이 다른 창보다 위에 뜨지 못하고 Z-Order상 가라앉아 보이지 않던 윈도우 정렬 오류를 수정했습니다.
  * Z-Order를 재배치하는 루프 방식을 정방향으로 교정하고 서브 창 정렬 순서를 역순(`reverse()`)으로 공급하여, 최신 포커스 창이 항상 다른 서브 창들보다 가장 상위에 활성화되도록 정상화했습니다.
* **미리보기(테스트) 알람 시 토스트 미출력 버그 수정:**
  * 버프 타이머 등의 소리 설정 미리보기 시, game-overlay에 토스트 알림 카드가 생략되는 버그를 고쳤습니다. 미리보기 이벤트 라우터와 오버레이 리스너를 단일화하여 테스트 시에도 설정 상태와 오버레이 옵션에 맞추어 토스트가 정상 작동하게 고쳤습니다.
* **상단 독(dock-top) 상태일 때의 감지 분기 버그 수정:**
  * 상단 독 모드 사용 시 화면 감지 가시성 체킹 조건에서 `dock-top` 분기가 빠져 있어 루프 상태가 오작동하던 누락을 수정했습니다.

## 📥 설치 및 적용 방법

**방법 1: 앱 내 설정 메뉴를 통한 업데이트 (권장)**
1. 앱 내부에서 **[설정]** 메뉴를 엽니다.
2. **[앱 정보]** 탭으로 이동한 뒤 **[업데이트 확인]** 버튼을 클릭하여 수동 업데이트를 진행합니다.

**방법 2: 직접 다운로드하여 업데이트**
1. [Releases](https://github.com/drt0927/tw-overlay/releases) 페이지에서 최신 버전의 `twOverlay-Setup-2.0.4.exe` 파일을 다운로드합니다.
2. 기존 앱을 완전히 종료하신 후, 다운로드한 `.exe` 설치 파일을 실행하여 설치를 진행합니다.
