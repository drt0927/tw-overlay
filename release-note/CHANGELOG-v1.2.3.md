# 🚀 TW-Overlay Release Note - v1.2.3

## 📅 배포 일자: 2026-02-21

이번 업데이트에서는 사이드바의 사용 편의성을 극대화하고, 오버레이 리소스 관리 문제를 해결했습니다. 또한 아이콘 검색 버그 등 실사용 시 불편했던 점들을 꼼꼼히 수정했습니다.

---

### 🆕 주요 개선 사항

#### 1. 사이드바 인터랙션 혁신
*   **지능형 클릭 통과**: 이제 사이드바 윈도우가 툴팁 표시를 위해 확장되어도, 실제 버튼이 없는 빈 공간을 클릭하면 뒤에 있는 게임이 정상적으로 조작됩니다.
*   **아이콘 가시성 개선**: 사이드바 아이콘들의 기본 색상을 더 깔끔한 `slate-400`으로 통일하고, 상태 변경 시의 깜빡임 로직을 부드럽게 다듬었습니다.
*   **툴팁 전체 복구**: 사이드바의 모든 버튼에 마우스를 올리면 기능 설명 툴팁이 나타나도록 복구했습니다.

#### 2. 오버레이 리소스 최적화
*   **완벽한 백그라운드 종료**: 오버레이 창을 닫으면 실행 중이던 웹페이지와 소리(유튜브 등)가 즉시 파괴되도록 수정하여 메모리 점유와 소음 문제를 해결했습니다.

#### 3. 설정 및 검색 편의성
*   **아이콘 검색 버그 수정**: 이제 대소문자를 구분하지 않고 아이콘을 검색할 수 있습니다. (예: `pill` 검색 시 `Pill` 아이콘 정상 노출)
*   **실시간 미리보기 완성**: 음량 슬라이더를 조절하는 즉시, 변경된 볼륨으로 사운드 미리보기가 가능합니다.

---

### 🛠 수정 사항
*   내부 UI 로직을 공통화하여 앱 전체의 안정성을 높였습니다.
*   장판 감지 아이콘의 상태별 색상 구분을 더 명확하게 개선했습니다.

---

### 📦 설치 방법

*   **신규 설치 또는 수동 업데이트**: [GitHub Releases](https://github.com/drt0927/tw-overlay/releases) 페이지에서 **`twOverlay-Setup-1.2.3.exe`** 파일을 다운로드하여 실행해 주세요. 기존 버전을 삭제하지 않고 덮어씌워 설치하시면 됩니다.
