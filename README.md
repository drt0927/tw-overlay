# 🎮 TW-Overlay (TalesWeaver Companion Browser)

테일즈위버(TalesWeaver) 플레이어를 위한 **지능형 다중 창 오버레이 브라우저**입니다. 게임 화면과 연동되는 자석형 위젯과 독립된 브라우저 창을 통해 최상의 플레이 환경을 제공합니다.

## ✨ 주요 기능

*   **⚡ 네이티브 엔진 (New):** PowerShell 의존성을 제거하고 **Win32 API(Koffi)**를 직접 호출하여 창 추적 및 장판 감시 속도를 극대화했습니다.
*   **🚀 게임 네트워크 최적화 (Fast Ping):** 네이글 알고리즘(Nagle's Algorithm)을 버튼 하나로 제어하여 게임 응답 속도(핑)를 개선할 수 있습니다.
*   **⏰ 필드 보스 알림:** 주요 필드 보스의 출현 시간을 감지하여 소리 알림을 제공합니다. 정각, 1분 전, 5분 전 등 원하는 시점 선택과 보스별 커스텀 사운드 설정이 가능합니다.
*   **🧪 지능형 버프 매니저 (Buff Manager):** 1000px 대화면 기반의 버프 백과사전 및 계산기입니다. 내가 선택한 도핑 아이템의 총 효과를 실시간으로 합산해 보여주며, 사용자 정의 프리셋을 자유롭게 수정하고 저장할 수 있습니다.
*   **📖 지능형 약어 사전 (Abbreviation Search):** 게임 내에서 사용하는 복잡한 약어들의 풀네임과 상세 설명을 검색 한 번으로 확인하세요.
*   **🧲 정밀 자석 트래킹:** 게임 창의 위치를 실시간 감지하여 모든 독립 창이 자석처럼 따라다니며 지정한 위치를 기억합니다.

## 🚀 설치 및 사용 방법

### 1. 설치하기
[Releases](https://github.com/drt0927/tw-overlay/releases) 페이지에서 최신 버전의 `twOverlay-Setup-1.3.0.exe` 파일을 다운로드하여 설치하세요. 기존 사용자는 앱 내 업데이트 기능을 사용할 수 있습니다.

### 2. 실행하기
1. 테일즈위버를 실행합니다.
2. `twOverlay`를 실행하면 자동으로 게임 창 우측에 사이드바가 나타납니다.
3. 사이드바의 각 아이콘을 클릭하여 기능을 조작할 수 있습니다.

## 📝 최신 업데이트 내역 (v1.3.0)
- **혁신: Native Win32 엔진 도입** (PowerShell 의존성 제거, 반응 속도 대폭 향상).
- **신규: 게임 네트워크 최적화 (Fast Ping)** (네이글 알고리즘 제어 기능 추가).
- **신규: 사이드바 메뉴 관리** (필요한 메뉴만 골라서 노출하는 커스터마이징 기능).
- **개선: GDI 기반 장판 감시** (Access Denied 에러 해결 및 CPU 점유율 획기적 감소).
- **개선: 지능형 Z-Order 스택** (오버레이 창이 게임 위에 고정되면서도 다른 앱 사용 시 유연하게 숨김).
- **디자인: 전방위 UI 리팩토링** (디자인 토큰 기반의 고퀄리티 다크 테마 및 애니메이션 적용).

## ⚖️ 장판 감지 기능에 관한 안내 (ScreenWatcher)

본 프로젝트는 테일즈위버 게임사의 **매크로 사용 금지 및 관련 처벌 정책**을 전적으로 지지하며, 게임의 공정성과 고유의 가치를 훼손할 의사가 없음을 분명히 밝힙니다.

개발자는 캐릭터의 자동 조작이나 게임 데이터 변조를 일체 금지하며, 오직 사용자의 **인지 보조**를 위한 알림 기능만을 제공합니다. 최종적인 판단과 조작은 항상 사용자의 몫입니다.
