# 📦 TW-Overlay Release Note [v1.11.2]

v1.11.2 패치는 버프 물약 관리를 자동화하는 **버프 타이머 알림 시스템**과 사냥 효율 분석을 위한 **지능형 경험치 HUD**를 새롭게 도입하고, 대규모 수치 처리 안정성 및 오프라인 작동 환경을 대폭 강화했습니다.

## 🚀 Added (신규 기능)

- **경험치 HUD 고도화 및 시각화** (`activity` 아이콘):
  - **실시간 사냥 리듬 차트**: 최근 30분간의 분당 경험치 획득량을 선 그래프로 시각화하여 사냥의 고점과 저점을 한눈에 파악할 수 있습니다.
  - **실시간 페이스 (5m)**: 최근 5분간의 이동 평균(Moving Average) 지표를 도입하여, 도핑 효과나 사냥 속도 변화를 즉각적으로 확인할 수 있습니다.
  - **경험의 정수 기대값 리포트**: 현재 페이스 기준 **시간당 생산 가능한 경험의 정수(100억 XP) 개수**와 **다음 정수 획득까지 남은 시간**을 실시간 계산합니다.
  - **데이터 영속성 확보**: 이제 HUD 창을 닫았다가 다시 열어도 사냥 데이터와 차트 히스토리가 증발하지 않고 즉시 복구됩니다.
  - **백그라운드 트래킹**: 사냥을 쉬는 동안에도 차트가 멈추지 않고 실시간 흐름을 유지하도록 설계되었습니다.

- **버프 타이머 알림 시스템**: 채팅 로그를 실시간으로 분석하여 핵심 버프(경험의 심장, 레어의 심장, 퇴마사의 은총) 사용을 자동으로 감지하고 HUD 타이머를 활성화합니다.
  - 36×36 컴팩트 뱃지 + 원형 프로그레스 링 + 3단계 페이즈(Normal/Warn/Danger) 알림
  - 불필요한 버프를 제외하고 사냥에 필수적인 3종 버프만 HUD에 표시되도록 필터링을 강화했습니다.

- **채팅 로그 자동 삭제**: 환경설정에서 지정한 보존 기간을 초과한 로그 파일을 자동으로 정리하여 디스크 공간을 절약합니다.

## 🎨 Changed (변경 및 개선)

- **지능형 숫자 포맷팅 도입**:
  - 앱 전체의 금액 및 경험치 표기를 **한국어 단위(조, 억, 만) 누적 표시** 방식으로 변경했습니다. (예: 1억 5000만)
  - 가독성을 위해 만 단위 미만의 자잘한 숫자는 생략하되, 전체 수치가 1만 미만인 경우에만 상세 수치를 표시합니다.
  - **UI 최적화**: 긴 수치가 레이아웃을 깨뜨리지 않도록 단위(조, 억, 만)를 작게 스타일링하는 '단위 스케일링' 기법을 적용했습니다.
- **오프라인 환경 최적화**: 외부 CDN을 사용하던 Chart.js 라이브러리를 로컬 에셋(`chart.umd.min.js`)으로 전환하여 인터넷 연결 없이도 모든 기능을 정상적으로 사용할 수 있습니다.
- **사운드 채널 통합**: 모든 알림 사운드를 `play-sound` 단일 채널로 통합하여 시스템 구조를 단순화했습니다.

## 🐞 Fixed (버그 수정)

- **21억(32비트) 오버플로 버그 수정**: 경험치가 약 21억 4천만을 초과할 때 수치가 마이너스(-)로 표시되던 치명적인 오버플로 문제를 `parseInt`에서 `Number` 변환 방식으로 전면 교체하여 해결했습니다. (최대 9,000조까지 안전하게 처리)
- **창 재오픈 시 데이터 유실 수정**: HUD 창을 닫으면 데이터가 초기화되던 버그를 IPC `invoke` 핸들러 기반의 상태 복구 로직을 도입하여 수정했습니다.
- **차트 렌더링 타이밍 수정**: 특정 환경에서 HUD 창을 열었을 때 차트가 즉시 그려지지 않던 렌더링 지연 문제를 해결했습니다.
- **기타 안정성 수정**: 제복 색상 시뮬레이터 로직 복구, 게임 최소화 시 오버레이 미숨김 현상, 윈도우 파괴 상태 체크 누락으로 인한 크래시 등을 수정했습니다.
