# TW-Overlay Release Notes - v1.0.8 (2026-02-20)

## 🚀 주요 변경 사항

### Added (신규 기능)
- **장판 감지 효과음 알람**: 보라색 장판 감지 시 사용자가 인지하기 쉽도록 효과음(.webm) 재생 기능을 추가했습니다.
- **사운드 에셋 내장**: `src/assets/sound/purple_alarm.webm` 파일을 포함하여 오프라인 상태에서도 알람이 정상 작동하도록 구성했습니다.

### Changed (변경 및 개선 사항)
- **빌드 시스템 유지**: `copy-resources.js`가 새로운 `.webm` 확장자도 누락 없이 `dist` 폴더로 복사하도록 검증되었습니다.
- **리소스 최적화**: 사운드 파일을 ASAR 패키지 내부에 포함하여 보안 및 로딩 속도를 확보했습니다.

### Fixed (버그 수정)
- 없음.

---
**설치 방법:** `twOverlay-Setup-1.0.8.exe` 파일을 실행하거나, 앱 내 환경설정에서 업데이트를 진행하세요.
