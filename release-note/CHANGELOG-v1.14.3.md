# TW-Overlay v1.14.3 릴리즈 노트 (2026.06.01)

이번 업데이트는 테일즈위버 게임을 종료한 백그라운드 대기 상태에서 주기적인 알림 사운드가 계속 흘러나오는 현상을 해결하는 핫픽스 패치 배포입니다.

---

## 🐞 Fixed (버그 수정)

- **🔊 게임 종료 시 백그라운드 알림 소리 차단**:
  - 테일즈위버 게임이 꺼진 백그라운드 상주 상태에서도 보스 타임 알람, 버프 타이머 만료음, 채팅 로그 키워드 알림 등의 사운드가 중복 재생되는 버그를 수정했습니다.
  - 사운드 재생의 최종 진입 장벽인 메인 사이드바(`index.html`) 단에서 게임 실행 상태(`running`, `minimized`)를 실시간으로 가로채 최종 점검하도록 설계했습니다.
  - 사용자가 환경 설정에서 소리를 테스트해보는 수동 기능("미리보기")은 정상 작동하도록 유지했습니다.
