# 📦 TW-Overlay Release Note - v2.1.0 (2026-07-07)

이번 릴리즈는 시간 측정 기록(스톱워치 일지) 내의 스탯 모니터링 체계를 고도화하고, 캐릭터 스탯 가시성을 계수 계산기 설정 스냅샷과 완벽하게 동기화시킨 마이너 업데이트입니다.

---

## 🚀 주요 업데이트 내역

### 📊 Added (추가 기능)
- **스탯 삼원화 분류 저장 및 데이터베이스 구조 패치**
  - 계수 계산기 공식과 정합성을 맞추기 위해 기존의 스탯 집계 방식을 캐릭터 순수 스탯, 장비 기본 스탯, 강화 스탯으로 나누어 각각 저장하도록 아키텍처를 확장했습니다.
  - 이를 위한 `timer_records` 테이블 내 `char_main`, `char_sub` 컬럼 추가를 자동으로 보장하는 **하위 호환 마이그레이션 DDL**을 구비했습니다.
- **상세 스탯 아코디언(Accordion) 펼치기 뷰 도입**
  - 기존 팝오버 호버 툴팁이 테이블 상하단 스크롤 경계선(`overflow`)에 걸려 위아래로 잘려 보이지 않던 레이아웃 버그를 근본적으로 종식하기 위해 클릭식 아코디언 행을 도입했습니다.
  - 눈 모양 버튼 클릭 시 각 기록의 세부 스탯(주스텟, 부스텟, 기타 전투 정보)이 은은한 카드 판넬 레이아웃으로 열리며, 최종 스탯과 함께 `{캐릭터 스탯}/{장비+강화 스탯}`의 구성비가 상세하게 표기됩니다.
  - 테이블에서 계열이나 코어를 변경하여 실시간 재계산이 일어나더라도 사용자가 열어 두었던 상세 카드들의 개폐 여부를 그대로 추적 보존하는 세션 제어 로직을 도입했습니다.

### 🇰🇷 Changed (변경 및 정비)
- **계열 및 코어 상태 정보 한국어화 번역 적용**
  - 영문(STAB, ECLIPSE 등)으로 노출되던 계열 및 코어 상태 표시를 전부 우리말(`찌르기`, `이클립스`, `없음` 등)로 정비했습니다.
  - 실제 게임에서 일반적으로 사용되는 명칭 통일 규격에 맞추어 마법쌍검(`maghack`)의 표시명을 **'마법베기'**로 일괄 교체했습니다.

### 🐛 Fixed (버그 수정)
- **계열 선택창 물리복합 오판정(찌르기 중복) 수정**
  - 계열 드롭다운 렌더링 시 주/부스탯 구성의 이름을 삼항 연산으로 비교하던 중, 찌르기/베기 구조가 겹치는 물리복합(`phycomp`)이 찌르기로 연쇄 오판정되어 드롭다운에 '찌르기'가 두 개 표기되던 버그를 고쳤습니다. 계열 키값을 통한 확실한 1대1 매핑으로 리팩토링했습니다.
- **코어 마스터 수치 강화 스탯 실시간 연동 미비 해결**
  - 드롭다운에서 코어 마스터 옵션을 변경할 때, 최종 계수 값만 갱신되고 아코디언 상세 보기 카드의 '강화 주스탯'과 '최종 주스탯' 수치는 갱신되지 않고 멈춰있던 동적 연동 누락을 해결했습니다.
  - 재계산 시 선택된 코어에 상응하는 실제 스탯 수치(머큐리얼 +8, 어비스 +9, 이클립스 +12, 루비코나 +15)를 계산하여 강화 주스텟(`enchantMain`)에 즉시 덧셈 연산하도록 보완했습니다.
