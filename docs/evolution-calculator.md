# 진화 재료 계산기 (Evolution Cost Calculator)

## 1. 기능 개요 및 목적
테일즈위버의 장비 진화 시스템(엔키라 → 이클립스 등)에 필요한 재료의 총 수량을 계산하고, 현재 시세에 따른 예상 소요 비용을 산출하는 도구입니다. 무기와 방어구의 각 단계별 레시피를 포함하고 있어 효율적인 장비 업그레이드 계획 수립을 돕습니다.

## 2. 주요 UI 구성 요소 설명
- **카테고리 탭:** '무기'와 '장비' 중 진화 대상을 선택합니다.
- **부위 선택기 (장비 전용):** 투구, 갑옷, 머리, 몸 등 세부 장비 부위를 선택합니다.
- **진화 범위 선택기:** 현재 장비 단계(From)와 목표 장비 단계(To)를 선택하여 범위를 지정합니다.
- **재료 리스트:** 선택된 범위 내의 모든 진화 단계에 필요한 재료를 합산하여 표시하며, 사용자가 단가를 직접 입력할 수 있습니다.
- **예상 비용 푸터:** 입력된 단가를 기준으로 총 예상 시드(Seed)와 엘소(Elso) 수량을 실시간으로 합산하여 보여줍니다.

## 3. 세부 기능 및 작동 방식
- **단계별 레시피 합산:** 여러 단계(예: 인퍼널 → 어비스)를 한 번에 선택하면 각 단계별 재료 수량을 자동으로 누적 합산합니다.
- **단가 저장 기능:** 입력한 재료별 단가는 브라우저의 로컬 스토리지에 저장되어 다음에 다시 열었을 때도 유지됩니다.
- **엘소 구입 옵션:** '태청금액신단'과 같은 특수 재료의 경우, 시드 대신 엘소로 구입하는 옵션을 선택하여 별도로 비용을 합산할 수 있습니다.
- **시각적 단계 표시:** 각 재료가 어떤 진화 단계에서 사용되는지 배지를 통해 직관적으로 표시합니다.
