# TW-Overlay v1.15.1 릴리즈 노트 (2026.06.02)

이번 업데이트는 버프 타이머의 청각 알림 사운드가 설정 화면 UI와 백엔드 간에 일치하지 않아 발생하던 사운드 오작동 문제를 수정한 패치 버전입니다.

---

## 🐛 Fixed (버그 수정)

- **🔊 버프 타이머 사운드 기본값 설정 오류 수정**:
  - 버프 타이머 알림 사운드를 사용자가 명시적으로 저장하지 않은 기본 상태에서, 설정 화면(`buff-timer.html`) 상으로는 **"기본 알림"(orb.mp3)**이 선택된 것처럼 표시되었으나, 실제 백엔드(`buffTimerManager.ts`) 타이머 작동 시에는 **"일단은 보스부터 쓰러뜨리자"(voice_boss_first.wav)**가 기본값으로 재생되던 불일치 문제를 해결했습니다.
  - 이제 사용자가 사운드를 별도로 수정하지 않아도, UI 표시 및 실제 알림 작동 모두 **"기본 알림"(orb.mp3)**으로 정상 통일되어 보스 출현 알림으로 오인하는 문제가 발생하지 않습니다.
