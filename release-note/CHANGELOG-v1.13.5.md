# 📦 TW-Overlay Release Note - v1.13.5 (2026.05.24)

이번 v1.13.5 업데이트에서는 **퇴마사의 은총 버프 사용 시 이자벨 비율 버프 오감지 버그 핫픽스** 및 **중복 버프 그룹 자동 대체 기능**을 적용했습니다.

---

## 🚀 주요 변경 사항

### ⏱️ 퇴마사의 은총 버프 오감지 핫픽스 및 그룹 대체 개선
* **오감지 필터링**: 퇴마사의 은총을 사용했을 때 뒤따르는 `"일시적으로 능력치가 상승했습니다 : 상태이상 [비아누의 숨결]"` 메시지가 이자벨의 비법 (비율) 버프로 잘못 감지되던 현상을 해결했습니다. (퇴마사의 은총 사용 후 3초 내에 감지되는 비아누의 숨결 메시지를 필터링)
* **중복 버프 그룹 대체**: 게임 내 규정 상 동시 사용이 불가능한 버프들(예: 이자벨 비율 버프와 퇴마사의 은총 등 동일 버프 그룹)이 활성화되었을 때, 타이머 매니저에서 기존 버프 타이머를 안전하게 자동 종료(덮어쓰기)하도록 로직을 강화했습니다.

---

## 🛠 상세 내역

### Fixed (수정)
* `chatParser.ts` 내 퇴마사의 은총(`stat_exorcist`) 사용 감지 시점 기준으로 3초 내에 감지되는 이자벨 비율(`stat_izabel_ratio`) 버프 발동을 무시하는 필터링 로직 구현.

### Changed (변경)
* `buffTimerManager.ts` 내 `activateBuff` 함수에 동일 그룹(`group`) 버프 중복 활성화 시 기존 버프 자동 제거 로직 추가.
