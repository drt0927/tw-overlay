# TW-Overlay v1.17.12 릴리즈 노트 (2026.06.16)

이번 업데이트는 채팅 오버레이 채널별 글씨색 사용자 설정 기능 제공, 일일 숙제 '클럽던전' 추가, 오를리 방어전 자동 감지 및 숙제 카운팅 연동, 그리고 통찰의 비약 버프 타이머 등록을 포함합니다.

---

## 🌟 Added (신규 기능)

- **🎨 채팅 채널별 글씨색 설정 지원 (Pickr 라이브러리 도입)**:
  * 인게임 채팅 오버레이의 채널별(일반, 귓속말, 팀, 클럽, 외치기, 시스템) 글자 색상을 사용자가 원하는 색상으로 커스터마이징할 수 있습니다.
  * 설정 창(`settings.html`)에 다크 모드 테마에 조화롭게 디자인된 **Pickr** 컬러피커를 연동하여 시각적인 편의성을 높였습니다.
  * 색상 변경 즉시 채팅 오버레이 화면에 실시간으로 새로운 색상이 렌더링되도록 기능을 연동했습니다.
- **📋 숙제 체크리스트 일일 콘텐츠 추가**:
  * 숙제 체크리스트에 '클럽' 카테고리의 일일 숙제인 **'클럽던전'** 항목을 새로 등록할 수 있도록 데이터를 보완했습니다.
- **🛡️ 오를리 방어전 자동 감지 및 숙제 카운팅 연동**:
  * 채팅 로그에 기록되는 `"미션을 완료했습니다. : 오를리 방어전 지옥 난이도 클리어"` 문구를 감지하여 주간 숙제(`weekly-orly-defense`) 카운트가 자동으로 1씩 올라가도록 연동을 개발했습니다.
  * 숙제 화면에서 해당 항목에 자동 감지 지원을 상징하는 `AUTO` 배지가 표출되도록 처리했습니다.
- **🧪 통찰의 비약 버프 타이머 등록 및 이미지 적용**:
  * 채팅 로그에서 통찰의 비약 버프 사용 감지 시 **통찰의 비약(대)** (5분) 및 **통찰의 비약(특대)** (1분) 타이머가 정상 동작하도록 버프 데이터 및 알림 설정을 추가했습니다.
  * 버프 타이머 HUD에 두 비약의 고유 이미지(`통찰대.png`, `통찰특대.png`)를 정상 매핑하여 시각적인 편의를 도모했습니다.

---

## 📦 설치 및 업데이트 방법

1. **설치 파일 다운로드**:
   * 본 릴리즈 페이지 최하단의 **Assets** 섹션에서 `twOverlay-Setup-1.17.12.exe` 파일을 클릭하여 다운로드합니다.
2. **프로그램 실행 및 설치**:
   * 다운로드한 `.exe` 파일을 실행하면 자동으로 설치 및 업데이트가 진행됩니다.
   * *참고: Windows SmartScreen 경고가 뜰 경우, `추가 정보` 클릭 후 `실행`을 누르면 됩니다.*
