일 지난 로그 삭제

* 0으로 설정하면 자동 삭제 기능이 비활성화됩니다. (기본값: 0)

자동 삭제가 실행되는 시점

  • 앱 실행 시 — TW-Overlay를 시작할 때 설정된 기간이 지난 로그를 즉시 정리합니다.
  • 매일 자정 — 날짜가 바뀔 때마다 백그라운드에서 자동으로 정리합니다.

* 저장 후 즉시 적용하려면 앱을 재시작하세요.

채팅 로그가 작동하지 않나요?

테일즈위버 게임 내 [옵션(O) > 게임 > 인터페이스] 탭에서 채팅 로그 항목이 ON으로 설정되어 있어야 실시간 분석이 가능합니다.

* 필터 대상: 데스포이나, 신조, 키시니크, 에레오스, 로카고스, 마티아, 티로로스, 라이코스, 체리아, 실반, 샐리온, 실라이론, 샐레아나, 루미너스, 크라모르 및 공백이 포함된 보스/기믹 몬스터 대사 (예: 심연의 제2사도, 수색대장 등)

사칭 의심 닉네임 강조 설정

* 강조 대상: 닉네임에 특수 문자 콜론('：')이 포함되어 일반 유저 사칭 위험이 있는 경우, 닉네임 색상 변경, 빨간 점선 테두리 및 ⚠️ 경고 배지를 표시합니다.

* 부분 일치(포함 검색): 외치기 내용에 키워드가 포함되면 알림을 발송합니다.

관리자 권한 필요

레지스트리를 수정하기 위해 관리자 권한이 필요합니다. 앱을 종료하고 관리자 권한으로 다시 실행해주세요.

Nagle's Algorithm 최적화: 윈도우의 패킷 전송 지연 기능을 꺼서 게임 응답 속도를 높입니다. (일명 패스트핑/핑업 설정)

레지스트리가 올바르게 설정되었는지 직접 확인하고 싶으시다면, 아래 출처 사이트의 안내를 참고하여 레지스트리 편집기에서 직접 확인하실 수 있습니다.

* 본 설정은 시스템 레지스트리를 직접 수정하며, 활성화된 모든 네트워크 어댑터에 적용됩니다.

단축키 설정

동적 단축키 안내

단축키는 테일즈위버 또는 본 프로그램이 활성화(포커스)된 상태에서만 작동합니다. 바탕화면이나 다른 프로그램을 사용 중일 때는 작동하지 않아 단축키 충돌을 방지합니다.

데이터 백업 및 복구

전체 데이터 백업

모든 설정과 일지 데이터를 하나의 압축 파일(.zip)로 저장합니다.

데이터 불러오기 (복구)

이전에 백업한 파일을 선택하여 모든 데이터를 복구합니다.

주의사항

데이터를 불러오면 현재의 모든 설정과 기록이 백업 파일의 내용으로 덮어씌워집니다. 복구가 완료되면 프로그램이 자동으로 재시작됩니다.
