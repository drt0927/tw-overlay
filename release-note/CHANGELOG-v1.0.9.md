# TW-Overlay Release Notes - v1.0.9

## 🚀 주요 변경 사항

### Added (신규 기능)
- **성능 강화 (Performance Booster)**: 게임 프로세스(`InphaseNXD`) 감지 시 CPU 우선순위를 자동으로 '높음(High)'으로 설정하여 프레임 안정성을 향상시켰습니다.
- **지능형 폴링 시스템 (Smart Polling)**: 게임 창의 위치가 1초 이상 고정될 경우 폴링 주기를 자동으로 늘려(100ms → 1000ms) 시스템 리소스 소모를 최소화했습니다. 창 이동 감지 시 즉각적으로 복구됩니다.

### Changed (변경 및 개선 사항)
- **장판 감지 시스템 성능 최적화**: `getSources` 대신 `getUserMedia` 기반의 분석 효율을 높여 시스템 부하를 최소화했습니다.
- **프로세스 감지 로직 개선**: 실행 파일 이름뿐만 아니라 창 제목을 통한 2단계 검색 방식을 도입하여 환경 변화에도 유연하게 대응하도록 개선했습니다.
- **앱 리소스 점유 최적화**: 오버레이 앱 자체의 프로세스 우선순위를 낮춰 게임 구동에 더 많은 자원이 배분되도록 최적화했습니다.
- **정책 및 안내 추가**: `README.md`에 장판 감지 기능의 개발 취지와 매크로 방지 정책에 대한 공식 안내 문구를 추가했습니다.

### Fixed (버그 수정)
- **갤러리 모니터 알림 개선**: 갤러리 창이 활성화된 상태에서도 사이드바에 레드닷 알림이 남거나 새로 발생하는 현상을 수정했습니다.
