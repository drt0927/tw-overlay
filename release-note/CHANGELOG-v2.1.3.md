# 📦 TW-Overlay Release Note - [Mandatory Update] v2.1.3 (2026-07-12)

이번 v2.1.3 버전에서는 이클립스 에토스 기믹, 심연의 제2사도 반사 기믹, 이클립스 로카고스 기믹의 알림음과 볼륨을 환경설정에서 개별적으로 지정할 수 있는 기능을 추가하였습니다. 또한 사용자가 화면 알림만 보고 소리는 끄고 싶을 때를 위해 `사용 안 함 (소리 없음)` 옵션을 제공하며, 개발자 도구(DevTools) 콘솔의 시뮬레이터 테스트 시에도 오버레이 환경설정 알림음이 똑같이 작동하도록 통합 연동을 진행한 **강제 업데이트(Mandatory Update) 버전**입니다.

---

## 🚀 주요 변경 사항

### 🎵 Added (새로운 기능)
- **기믹 개별 알림음 및 볼륨 설정 기능 신설**:
  - 환경설정 - [소리 및 알림음 설정] - [인게임 기믹 및 시스템 알림] 영역 내의 각 기믹(에토스, 제2사도 반사, 로카고스)에 개별 사운드 선택 드롭다운과 볼륨 조절 슬라이더, 미리듣기 버튼을 추가하였습니다.
  - **알림음 개별 소리 끄기**: 사운드 드롭다운의 맨 위에 `사용 안 함 (소리 없음)` 옵션을 추가하여, 화면 오버레이 경고 표출은 유지하되 알림음만 개별적으로 꺼둘 수 있도록 개선하였습니다.
- **제2사도 반사 시작/종료 개별 알림음 지원**:
  - 심연의 제2사도 반사 기믹에 대해 반사 패턴 시작(경고)과 6.5초 후 반사 패턴 종료(공격 가능) 시점 각각에 대한 독립적인 알림음을 선택하고 지정할 수 있도록 지원합니다.

### 🛠️ Changed (개선 및 변경)
- **DevTools 테스트 시뮬레이터 연동**:
  - 인게임 오버레이 창의 DevTools 콘솔 디버깅용 함수(`testAbyssApostle()`, `testEthos()`, `testLokagos()`)가 단순히 UI만 교체하는 것에서 나아가, 환경설정에 사용자가 지정한 실제 알림음 및 볼륨으로 사운드가 동시에 재생되도록 통합 연동을 적용하였습니다.

## 📥 설치 및 적용 방법

**방법 1: 앱 내 설정 메뉴를 통한 업데이트 (권장)**
1. 앱 내부에서 **[설정]** 메뉴를 엽니다.
2. **[앱 정보]** 탭으로 이동한 뒤 **[업데이트 확인]** 버튼을 클릭하여 수동 업데이트를 진행합니다.

**방법 2: 직접 다운로드하여 업데이트**
1. [Releases](https://github.com/drt0927/tw-overlay/releases) 페이지에서 최신 버전의 `twOverlay-Setup-2.1.3.exe` 파일을 다운로드합니다.
2. 기존 앱을 완전히 종료하신 후, 다운로드한 `.exe` 설치 파일을 실행하여 설치를 진행합니다.
