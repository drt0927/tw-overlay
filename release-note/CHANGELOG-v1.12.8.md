# 📦 TW-Overlay Release Note (v1.12.8)

**배포일:** 2026.05.15

이번 버전에서는 커스텀 숙제의 상세 정보를 언제든 수정할 수 있는 편집 기능을 추가하고, 다중 캐릭터 매트릭스 뷰의 안정성을 강화했습니다.

## ✨ 새 기능 (Added)
- **커스텀 숙제 편집 기능:**
  - 이제 직접 추가한 커스텀 숙제의 이름, 카테고리, 초기화 규칙(매일/매주, 시각, 요일)을 자유롭게 수정할 수 있습니다.
  - 관리 모드(톱니바퀴 아이콘) 활성화 시 각 커스텀 숙제 옆에 나타나는 '수정(아이콘)' 버튼을 통해 상세 정보를 변경할 수 있습니다.

## 🛠 개선 사항 (Changed)
- **Matrix View UI 고도화:**
  - 관리 모드에서 가시성 토글(눈 아이콘), 수정, 삭제 버튼을 직관적으로 배치하여 숙제 목록 관리가 더 쉬워졌습니다.
  - 편집 폼을 재사용하여 새로운 숙제 추가와 기존 숙제 수정을 일관된 UI로 제공합니다.

## 🐛 버그 수정 (Fixed)
- 다중 캐릭터 시스템 도입 과정에서 발생했던 내부 타입 의존성 및 빌드 오류를 해결했습니다.
- 캐릭터 이름 변경 시 UI에 즉시 반영되지 않던 문제를 최적화했습니다.
