# 📦 TW-Overlay Release Note (v1.12.6)

이번 업데이트는 **모험일지 주간 리포트 기능 추가 및 사이드바 노출 안정성 강화**를 주요 목표로 합니다.

## 📅 2026.05.15 배포 내역

### ✨ Added (신규 기능)
- **주간 상세 리포트 대시보드**: 모험일지 달력의 'W' 영역 클릭 시 해당 주간의 성과(숙제 완료 건수, 보스 처치 횟수, 총 획득 수익)를 요약하여 보여주는 직관적인 대시보드를 추가했습니다.
- **주간 통합 타임라인**: 선택한 주의 모든 활동을 날짜별로 통합하여 한눈에 확인할 수 있습니다.
- **주차별 수익 통계 리포트**: 통계 탭에 이번 달 각 주차별로 발생한 수익을 리스트 형태로 보여주는 컴포넌트를 추가하여 수익 흐름 파악을 용이하게 했습니다.

### ⚙️ Changed (개선 및 변경)
- **모험일지 시간 표시 개선**: 타임라인에 표시되는 시간 포맷을 항상 'HH:mm'(예: 00:23, 14:05) 형식으로 통일하여 가독성을 개선했습니다.
- **DB 데이터 보존**: 화면 표시 로직만 개선되었으며, 데이터베이스에는 여전히 초(seconds) 단위까지 포함된 원본 데이터가 안전하게 저장됩니다.

### 🐞 Fixed (버그 수정)
- **사이드바 비정상 노출 수정**: 게임이 종료되었거나 최소화된 상태에서 알람(필드보스, 커스텀 알림 등)이 울릴 때 사이드바가 화면에 나타나던 버그를 해결했습니다.
- **가시성 체크 강화**: 이제 사이드바가 실제로 보이고 있을 때만 창 크기 조절이 발생하며, 게임 상태 추적 실패 시 이전 좌표 정보를 즉시 초기화하여 오동작을 차단합니다.
- **입장료 지출 내역 오인식 해결**: 채팅 파서 정규식 고도화를 통해 어벤던로드 입장료 지출 메시지가 '수익 획득'으로 오인되어 모험일지에 기록되던 현상을 수정했습니다.
