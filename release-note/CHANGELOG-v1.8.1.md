# Release Notes - v1.8.1 (2026-03-25)

테일즈위버 컴패니언 브라우저 **TW-Overlay v1.8.1** 업데이트 내역입니다.

## 🆕 Added (새로운 기능)
- **장비 진화 계산기 데이터 확충**: 
  - 누락되었던 **손목** 장비의 진화 재료 데이터가 추가되었습니다.
- **사용자 리텐션 분석 시스템 (Heartbeat)**: 
  - 앱 사용 지속 시간을 보다 정밀하게 측정하기 위해 1시간 주기의 Heartbeat(생존 신고) 이벤트를 도입했습니다.
  - 이를 통해 실제 사용자들이 앱을 얼마나 유용하게 활용하고 있는지 분석하여 향후 기능 개선에 활용할 예정입니다.

## 🚀 Changed (변경/개선 사항)
- **네트워크 안정성 강화**: 구글 애널리틱스 이벤트 전송 시, 인터넷 연결이 오프라인 상태이거나 불안정한 경우에도 앱이 영향받지 않도록 예외 처리 로직을 강화했습니다.
- **UI 개선**: 진화 재료 계산기에서 '손목' 부위를 명확히 선택할 수 있도록 버튼 UI를 최적화했습니다.

## 🛠 Fixed (버그 수정)
- **데이터 오타 수정**: 장비 진화 데이터 내 원격 수정 과정에서 발생한 '군단의의 인장' 오타를 '군단의 인장'으로 일괄 수정했습니다.
