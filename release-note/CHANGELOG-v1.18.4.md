# TW-Overlay v1.18.4 릴리즈 노트 (2026.06.18)

이번 업데이트는 **경험치 HUD 초기화 단축키 추가 및 버프 타이머 비우기 개선**과 **신규 경험치 버프 3종 감지 연동**이 적용되었습니다.

## 🛠️ 주요 변경 사항 (CHANGELOG)

### Added (신규 기능 추가)
* **경험치 세션 초기화 단축키 추가**:
  - 전역 단축키 `Ctrl + Shift + X` (맥: `Cmd + Shift + X`) 입력을 통해 게임 도중 경험치 HUD 데이터를 한 번에 리셋할 수 있는 편의 기능이 제공됩니다.
  - 설정 화면의 단축키 설정 탭에서 단축키 조작 지정 및 초기화가 가능합니다.
* **버프 타이머 전체 삭제 버튼 및 단축키 추가**:
  - `Ctrl + Shift + E` (맥: `Cmd + Shift + E`) 단축키를 눌러 활성화 상태의 모든 버프 타이머를 비울 수 있습니다.
  - 버프 타이머 설정 창 내 활성 버프 헤더 영역에 **[전체 삭제]** 버튼이 추가되어 마우스 클릭으로도 손쉽게 일괄 제거할 수 있습니다.
* **신규 경험치 버프 3종 감지 추가**:
  - 아래 3가지 경험치 관련 버프가 감지 및 타이머 연동 대상으로 추가되었으며, 고유의 아이콘 이미지가 연동됩니다.
    - `최상급 에오스의 파편` (1시간 유지, `club_eos` 그룹 중복 처리)
    - `전설의 군고구마` (30분 유지, `potato` 그룹 중복 처리)
    - `얼리버드 경험치 부스터` (1시간 유지, `none` 그룹)

### Fixed (오류 수정 및 보완)
* **테스트 미리보기 시 일부 버프 누락 수정**:
  - 기존 미리보기 테스트 실행 시, 동일한 그룹에 속해있는 버프들이 켜지는 과정에서 그룹 중복 배타 제거 로직에 걸려 화면에 노출되지 않던 오류를 수정했습니다. 테스트 모드에서는 그룹 제한 우회가 적용되어 통찰의 비약(대/특대) 및 군고구마류, 에오스 파편류 등 모든 감지 버프들이 정상 노출됩니다.

## 📥 설치 방법

1. 본 릴리즈 페이지 최하단의 **Assets** 섹션에서 `twOverlay-Setup-1.18.4.exe` 파일을 클릭하여 다운로드합니다.
2. 기존에 실행 중이던 `tw-overlay` 앱이 있다면 반드시 완전히 종료한 뒤 설치 파일을 실행합니다.
3. 화면의 안내에 따라 설치를 완료하면 새로운 기능과 패치가 적용됩니다.
