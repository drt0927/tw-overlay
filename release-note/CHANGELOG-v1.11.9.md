# 📋 Release Note - v1.11.9

## 🚀 개요
이번 버전은 XP HUD 사용 중 경험치를 판매할 때 발생하는 통계 하락 문제를 해결하고, 시각적 데이터 표현의 정확도를 높였습니다.

## ✨ 주요 변경 사항

### 📊 XP HUD 개선 및 필터링 (XP Tracking)
- **경험치 감소량 무시 옵션 추가**: 테일즈위버에서 경험치를 판매하여 100억 XP가 감소할 때, HUD의 총 획득량과 EPM이 함께 깎여서 사냥 효율 분석이 어려웠던 점을 개선했습니다. 
    - '환경 설정 > 표시 설정'에서 **경험치 감소 무시** 토글로 제어 가능합니다.
    - 기본값은 **활성(무시)**이며, 이제 경험치를 팔아도 세션 통계는 유지됩니다.

### 🎨 시각화 버그 수정 (UI/UX)
- **사냥 리듬 차트 y축 수정**: 최근 30분 사냥 리듬 그래프의 y축 라벨이 단위 변환 로직 중복 적용으로 인해 항상 "0만"으로 표시되던 버그를 수정했습니다. 이제 사냥량에 맞는 정확한 수치가 표시됩니다.
