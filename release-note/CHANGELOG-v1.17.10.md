# TW-Overlay v1.17.10 릴리즈 노트 (2026.06.11)

이번 업데이트는 날짜 변경 및 게임 재접속 시 채팅 오버레이 데이터 흐름의 정합성을 개선하였으며, 사기꾼 탐지 기능 작동 시 새로운 1:1 대화방을 감지하면 탐지 모니터 창이 자동으로 팝업 및 활성화되도록 연동하고 스위치 UI 디자인 오류를 수정했습니다.

---

## 🌟 Added (신규 기능)

- **사기꾼 탐지기 1:1 대화 자동 활성화**:
  * 실시간 1:1 대화 감시 기능이 켜져 있을 때, 새로운 대화방 로그 파일이 생성되면 사기꾼 탐지 창(`scam-detector.html`)이 자동으로 열리고 포커스되도록 연결하였습니다.
  * 사기꾼 탐지 창 내부 설정 토글들의 CSS 스타일이 누락되어 생기던 디자인 렌더링 오류를 수정하여, 경험치 HUD 등 타 위젯과 동일하게 미려한 스타일 스위치로 출력되도록 복구했습니다.

---

## 🐛 Fixed (버그 수정)

- **날짜 변경 및 게임 재기동 시 채팅 로그 리플레이 꼬임 해결**:
  * 컴퓨터를 켜둔 채 게임을 껐다 켰을 때(또는 자정이 지나 날짜가 변경되었을 때) 새로운 로그 파일 추적 시작 시 기존 채팅 데이터가 중복되거나 표시되지 않던 누락 현상을 해결했습니다.
  * 추적 대상 로그 파일 변경으로 오늘 일자 채팅 로그 리플레이(재처리)가 발생하기 전 버퍼스토어 공간을 온전히 초기화하고, 렌더러에 갱신 알림(`chat-history-cleared`)을 보내 데이터 충돌 없이 당일 대화 기록만 깨끗하게 새로고침하도록 개선했습니다.

---

## 📦 설치 및 업데이트 방법

1. **설치 파일 다운로드**:
   * 본 릴리즈 페이지 최하단의 **Assets** 섹션에서 `twOverlay-Setup-1.17.10.exe` 파일을 클릭하여 다운로드합니다.
2. **프로그램 실행 및 설치**:
   * 다운로드한 `.exe` 파일을 실행하면 자동으로 설치 및 업데이트가 진행됩니다.
   * *참고: Windows SmartScreen 경고가 뜰 경우, `추가 정보` 클릭 후 `실행`을 누르면 됩니다.*
