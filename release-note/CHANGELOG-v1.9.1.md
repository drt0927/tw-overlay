# 🚀 TW-Overlay Release Notes (v1.9.1)

## 📅 출시일: 2026-03-27

---

## 🌟 주요 변경 사항

### 1. 데이터 유지보수 및 오타 수정 (Maintenance)
사용자 편의성을 높이기 위해 게임 데이터 내의 명칭 오타와 조사 중복을 제거했습니다.
- **진화 재료 명칭 수정**: `evolution_data.json` 파일 내에서 '건틀렛'으로 표기된 부분을 '건틀릿'으로 통일하여 검색 및 가독성을 개선했습니다.
- **조사 중복 제거**: 진화 재료 설명 중 '군단의의'와 같이 조사가 중복된 부분을 '군단의'로 수정했습니다.
- **데이터 일관성 확보**: 앱 전반에서 사용되는 아이템 명칭의 표기법을 표준화했습니다.
