# 📦 TW-Overlay Release Note [v1.11.1]

v1.11.1 패치는 약어 사전의 대규모 데이터 업데이트와 UI 리디자인, 그리고 실시간 채팅 로그 엔진의 안정성을 강화하는 데 집중했습니다.

## 🚀 Added (신규 기능)
- **약어 사전 데이터 전수 확충**: 공식 홈페이지(알롱지님) 및 매직위버(햄찌님)의 방대한 게임 용어 데이터를 통합하여 160개 이상의 항목을 수록했습니다.
- **약어 사전 Blue 시그니처 테마**: 사이드바 아이콘 색상과 톤을 일치시킨 세련된 블루 테마를 적용하고 가독성을 위해 정보 계층을 재설계했습니다.
- **모험 일지 수익 기록 상세화**: 자동 기록되는 SEED 수익 내역에 게임 로그 원문 메시지를 포함하여, 어떤 콘텐츠로 얼마를 벌었는지 더 명확하게 확인할 수 있습니다.

## 🎨 Changed (변경 및 개선)
- **약어 사전 필터 UI 개선**: 가로 스크롤 방식의 필터를 자동 줄바꿈(Flex Wrap) 방식으로 변경하여 모든 카테고리를 한눈에 보고 즉시 선택할 수 있습니다.
- **가독성 정밀 교정**: 전체 데이터의 띄어쓰기, 문장 부호, 괄호 및 기호 간격을 전수 조사하여 깔끔하게 정제했습니다.
- **창 디자인 통일**: 약어 사전 창의 헤더와 푸터 디자인을 프로젝트 표준 스타일로 조정하여 시각적 일관성을 확보했습니다.

## 🐞 Fixed (버그 수정)
- **채팅 로그 날짜 갱신 버그**: 자정 이후 게임 로그 파일이 교체될 때 기록 날짜가 이전 날짜로 고정되던 이슈를 해결했습니다. 이제 파일 헤더의 날짜 정보를 실시간으로 추적합니다.
- **수익 내역 렌더링 오류**: 특정 기록 형식에서 설명 문구가 보이지 않던 파싱 로직 버그를 수정했습니다.
- **SEED 파싱 엔진 고도화**: 대괄호`[]` 패턴 및 "만/억" 단위가 섞인 복잡한 수익 메시지도 정확하게 숫자로 변환합니다.
