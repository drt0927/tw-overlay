# TW-Overlay v1.4.1 릴리즈 노트

## 🛠 버그 수정 및 개선 (Hotfix)
- **창 로딩 레이스 컨디션 해결**: 일부 환경에서 설정 창이나 오버레이 창이 열릴 때 데이터가 즉시 반영되지 않던 문제를 `did-finish-load` 이벤트 연동을 통해 해결했습니다.
- **설정 창 무한 로딩 방지**: 특정 상황에서 설정 화면의 로딩 스피너가 사라지지 않던 문제를 방지하기 위해 3초 폴백(Fallback) 타이머를 추가했습니다.
- **창 위치 미세 조정**: ETA 랭킹 창의 기본 위치(`offsetX`)를 최적화했습니다.
