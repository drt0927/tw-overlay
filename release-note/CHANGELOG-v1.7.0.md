# TW-Overlay Release Notes - v1.7.0

## 🚀 새로운 기능 (Added)
- **일일/주간 숙제 체크 리스트**:
  - 테일즈위버의 30여 종 기본 컨텐츠를 포함한 지능형 체크 리스트 기능이 추가되었습니다.
  - 게임 내 실제 초기화 시간(00:00 자정, 월요일 06:00 등)을 준수하는 자동 초기화 시스템을 탑재했습니다.
  - 실시간 검색 기능으로 원하는 숙제를 빠르게 찾을 수 있습니다.
  - 사용자가 직접 숙제를 추가/삭제하거나 카테고리명을 수정할 수 있는 커스터마이징 기능을 제공합니다.
  - '가시성 제어'를 통해 본인의 스펙에 맞는 숙제만 골라볼 수 있습니다.

## 🔄 변경 및 개선 사항 (Changed)
- **게임 종료 리마인더 고도화**:
  - 게임 종료 시 나타나는 리마인더 팝업창에 **"아직 완료하지 않은 숙제"** 목록이 자동으로 표시됩니다.
  - 일일 숙제(Blue)와 주간 숙제(Purple)를 색상으로 구분하여 가독성을 높였습니다.
- **디자인 시스템 통합 (UI Unification)**:
  - 모든 창에 'Purple & Glass' 테마를 적용하여 시각적 정체성을 통일했습니다.
  - 카드 기반 레이아웃과 정교한 타이포그래피를 적용하여 가독성을 대폭 개선했습니다.
  - 숙제 체크 리스트 창의 크기를 400x1200(최대)으로 확장하여 쾌적한 뷰를 제공합니다.
- **백그라운드 동기화 엔진**:
  - 창이 닫혀 있어도 매 분마다 초기화 시각을 체크하여, 정각이 되는 순간 실시간으로 데이터를 갱신하도록 로직을 강화했습니다.

## 🐞 버그 수정 (Fixed)
- 특정 상황에서 설정 데이터가 지연 저장되던 문제를 즉시 저장(saveImmediate) 방식으로 수정하여 안정성을 확보했습니다.
- 창 위치 기억 오프셋이 비정상적으로 계산되던 문제를 수정했습니다.
