# 📦 TW-Overlay Release Note (v1.10.6)

## 📅 배포일: 2026-04-04

이번 버전에서는 비설화(twsnowflower)님의 제복 색상 시뮬레이터를 앱 내 유틸리티 창으로 통합하고, 외부 서비스 연동을 위한 아키텍처를 개선했습니다.

---

### 🚀 Added (신규 기능)
- **제복 색상 시뮬레이터 연동:** 비설화님의 제복 시뮬레이터를 사이드바 메뉴(`palette` 아이콘)를 통해 즉시 실행할 수 있습니다. (사전 동의 하에 연동되었습니다.)
- **WebContentsView 아키텍처 도입:** 유틸리티 창에서도 오버레이 브라우저와 동일한 고성능 뷰 방식을 사용하여 외부 웹사이트 연동의 안정성을 높였습니다.

### ⚡ Changed (변경 및 개선 사항)
- **UI 일관성 강화:** 모든 유틸리티 창에 표준 헤더(`win-header`)를 적용하여 드래그 영역과 디자인을 통일했습니다.
- **CSS 동적 주입 최적화:** 외부 사이트 연동 시 불필요한 스크롤바를 숨기고 컨텐츠를 앱 레이아웃에 맞게 자동 조정합니다.
- **개발 환경 개선:** 개발 모드(`IS_DEV`) 시 메인 창과 웹 뷰의 개발자 도구가 자동으로 열리도록 설정했습니다.

### 🐞 Fixed (버그 수정)
- **창 드래그 이슈:** 음수 마진으로 인해 창 드래그가 차단되던 레이아웃 버그를 해결했습니다.
- **깜박임 현상 완화:** 배경색 명시 및 렌더링 타이밍 최적화를 통해 웹 뷰 로드 시의 깜박임을 개선했습니다.
