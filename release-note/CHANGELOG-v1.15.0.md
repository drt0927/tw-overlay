# TW-Overlay v1.15.0 릴리즈 노트 (2026.06.02)

이번 업데이트는 디스코드 키워드 알림 기능의 감지 대상을 대폭 단순화하고 외치기 대화까지 확장하였으며, 명시적 저장 단계를 제거하여 실시간 즉시 저장이 동작하는 반응형 UI를 구축한 기능 고도화 배포입니다.

---

## 🚀 Added (새로운 기능)

- **📢 외치기 대화(Shout) 디스코드 알림 추가**:
  - 디스코드 키워드 알림 감지 대상에 '외치기 대화' 옵션을 새롭게 탑재했습니다.
  - 이를 통해 본인이나 타인이 확성기를 통해 외치는 인게임 채팅 내용도 디스코드로 실시간 감지하여 웹훅 알림을 받아볼 수 있습니다.

## ⚙️ Changed (기능 개선 및 변경)

- **💬 디스코드 알림 감지 카테고리 단순화**:
  - 기존의 복잡했던 3가지 대상 분류(일반대화, 내가 쓴 글, 클럽대화) 중 '내가 쓴 일반글'을 일반 대화 카테고리에 통합하여 **'일반 대화 (나 포함)'** 및 **'클럽 대화'** 2종으로 단순화하고 설정의 직관성을 개선했습니다.
  - 기존 구버전 규칙 데이터를 불러올 때 자동으로 일반 대화로 병합되도록 런타임 마이그레이션을 지원합니다.
- **⚡ 실시간 자동 저장(Auto-save) 및 미니멀 UI 구축**:
  - 단어 추가/삭제, 토글 변경, Webhook URL 수정 등의 조작을 실행하는 즉시 설정 파일에 자동 반영되도록 개선했습니다.
  - 매번 수동으로 누르던 '저장' 버튼 및 '닫기' 버튼을 제거하여 설정창의 하단 레이아웃을 정돈했습니다. (우측 상단 X 버튼 또는 ESC 키로 닫기 작동)
