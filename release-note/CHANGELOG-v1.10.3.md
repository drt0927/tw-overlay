# 📦 TW-Overlay Release Note (v1.10.3)

이번 버전에서는 게임을 켜지 않아도 필요한 도구들을 사용할 수 있는 **독립 유틸리티 모드**와, 한 달간의 활동을 시각적으로 즐길 수 있는 **모험 리포트** 기능이 추가되었습니다.

## ✨ 신규 기능 (Added)
- **📊 모험 일지: 월간 통계 리포트**
    - 다이어리 내 [통계] 탭이 추가되었습니다.
    - 월간 출석 현황을 한눈에 보는 **'활동 히트맵'** 기능.
    - 가장 많이 처치한 보스를 보여주는 **'최애 보스 Top 3'**.
    - 요일별 활동 패턴을 분석하는 **'주간 활동 리듬'** 차트.
    - 월간 총 수익(Seed) 자동 합산 및 표시.
- **⚙️ 시스템 트레이 메뉴 대폭 확장**
    - 트레이 아이콘 우클릭 시 계산기, 다이어리, 약어 사전 등 사이드바의 모든 도구를 즉시 실행할 수 있습니다.
- **🛡️ 독립 유틸리티 모드**
    - 테일즈위버가 꺼져 있거나 최소화된 상태에서도 트레이 메뉴를 통해 유틸리티 도구들을 독립적으로 사용 가능합니다. (게임 미실행 시 창이 중앙에 배치됨)

## 🚀 개선 사항 (Changed)
- **운명 공동체 로직 고도화**: 게임이 실행 중일 때는 게임 창의 상태(보임/최소화)에 따라 모든 오버레이 창이 함께 반응합니다.
- **다이어리 UI 개선**:
    - 통계 탭 추가에 맞춰 창의 기본 가로 넓이를 **1000px**로 확장했습니다.
    - 우측 상세 화면의 공간을 넓혀 가독성을 높였습니다.
- **커스텀 툴팁 도입**: 모험 등급 등 정보 아이콘에 마우스를 올리면 즉시 나타나는 세련된 전용 툴팁을 적용했습니다.
- **트레이 UX 일관성**: 게임이 활성화된 상태에서만 트레이 더블 클릭으로 사이드바를 호출할 수 있도록 개선했습니다.

## 🛠️ 기술적 수정 (Fixed)
- 게임 미실행/최소화 시 특정 상황에서 오버레이 창이 잔상처럼 남아있던 문제를 해결했습니다.
- `pollingLoop`의 상태 전이 기반 최적화를 통해 불필요한 리소스 사용을 줄였습니다.
