# v1.0.4 - 테일즈위버 스타일 사이드바 퀵슬롯 추가 (2026-02-14)

## 주요 변경 사항

### 1. 테일즈위버 스타일 사이드바 (Sidebar Window) 도입
- 메인 오버레이와 독립된 별도의 얇은 창(Sidebar Window)이 테일즈위버 게임창 우측 끝에 자석처럼 밀착되어 이동합니다.
- 게임 UI와의 일체감을 위해 **30x30 픽셀**의 정규 슬롯 사이즈를 적용했습니다.
- 은색 금속 질감 테두리 디자인을 구현했습니다.

### 2. 고정 및 접기/펴기 기능
- 사이드바는 사용자가 직접 드래그할 필요 없이 항상 게임 창 우측에 고정됩니다.
- 최상단의 토글 버튼(`>`)을 통해 사이드바를 접거나 펼칠 수 있습니다.
- 접힘 상태에서는 너비가 12px로 축소되어 게임 화면 가림을 최소화합니다.

### 3. 비주얼 및 사용자 경험 개선
- **보라색 글로우(Purple Glow):** 슬롯 호버 시 테일즈위버 특유의 보라색 광채가 안팎으로 퍼지는 효과를 추가했습니다.
- **순백색 아이콘:** Lucide 아이콘을 순백색(#ffffff)으로 강제 고정하여 가독성을 극대화했습니다.
- **테일즈위버 스타일 툴팁:** 슬롯 호버 시 우측에 보라색 테두리의 고전적인 툴팁이 나타납니다.

### 4. 기술적 개선
- **멀티 윈도우 동기화:** 메인 창과 사이드바 창이 동시에 게임 좌표를 추적하며, 투과 모드(`Ctrl+Shift+T`)도 실시간으로 동기화됩니다.
- **외부 브라우저 호출:** 사이드바 버튼을 통해 앱 내 네비게이션뿐만 아니라 시스템 기본 브라우저 호출이 가능합니다.
