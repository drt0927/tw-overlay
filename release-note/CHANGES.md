# TW-Overlay 변경 이력

- **운영 안정화**: 디버깅 로그 정리 및 시스템 초기화 로직 최적화.

## 아키텍처 개요 (v1.0.7)

```
src/
├── main.ts                  # 앱 진입점 (라이프사이클, 업데이트, 폴링)
├── preload.ts               # 렌더러 ↔ 메인 프로세스 브릿지
├── index.html               # 메인 사이드바 UI
├── splash.html              # 초기 구동 스플래시 화면
├── overlay.html             # 브라우저 오버레이 창
├── settings.html            # 와이드 설정 대시보드
├── gallery.html             # 갤러리 모니터 전용 창
├── style.css                # 공통 스타일
├── assets/                  # 로컬 JS 라이브러리 (Tailwind, Lucide)
└── modules/
    ├── constants.ts          # 전역 상수 및 설정 타입
    ├── updater.ts            # 자동 업데이트 관리 로직
    ├── tracker.ts            # PowerShell 기반 게임 창 추적
    ├── windowManager.ts      # 다중 창 생성 및 동기화 관리
    ├── config.ts             # JSON 설정 저장 및 로드
    ├── ipcHandlers.ts        # 모든 IPC 이벤트 등록
    ├── galleryMonitor.ts     # 갤러리/댓글 감시 서비스
    ├── logger.ts             # 파일 로깅 시스템
    └── tray.ts               # 시스템 트레이 관리
```

## 주요 변경사항 (v1.0.7 핵심)

### 1. 초고속 로딩 (Zero-Network Dependency)
기존에는 모든 창이 뜰 때마다 CDN에서 대용량 라이브러리를 받아와 수 초간의 흰 화면이 발생했습니다. 1.0.7 버전부터는 모든 필수 에셋을 내장하여 **즉각적인 창 전환**이 가능합니다.

### 2. 통합 업데이트 관리자
별도의 웹사이트 방문 없이 앱 내에서 직접 최신 버전을 유지할 수 있습니다.
- 환경설정 '앱 정보' 메뉴에서 버전 확인 및 업데이트 체크.
- 다운로드 진행률 실시간 표시 및 자동 설치 지원.

### 3. 사용자 경험(UX) 고도화
- **스플래시 화면:** 무거운 외부 프로세스(PowerShell 등)가 준비되는 동안 정지된 화면 대신 부드러운 로딩 애니메이션을 보여줍니다.
- **브랜드 아이콘:** 앱 곳곳에 공식 아이콘을 배치하여 완성도를 높였습니다.

### 4. 현대적인 빌드 파이프라인
윈도우 환경에 의존하던 PowerShell 복사 명령을 Node.js 스크립트로 대체하여 빌드 환경의 안정성을 높이고 오류 가능성을 최소화했습니다.
