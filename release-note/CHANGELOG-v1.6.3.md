# CHANGELOG v1.6.3

이 릴리즈는 새로운 '계수 및 명중 계산기' 기능의 추가와 함께 앱 전반의 내부 시스템 안정성을 극대화한 업데이트입니다. 장시간 켜두거나 다중 모니터를 사용할 때 발생하던 불편함들이 대거 해소되었습니다.

## ✨ Added (신규 기능)
- **계수 및 명중 계산기 추가:** 테일즈위버 캐릭터의 스탯, 장비, 버프를 종합하여 총 계수와 명중률을 실시간으로 시뮬레이션할 수 있는 전용 계산기 기능이 새롭게 추가되었습니다. 사이드바 메뉴를 통해 접근할 수 있습니다.

## 🚀 Changed (개선 및 최적화)
- **메모리 최적화:** 모든 오버레이/독립 창이 닫힐 때 내부 이벤트 리스너(IPC)를 깔끔하게 해제하는 `cleanupAllListeners` 로직을 추가하여 앱을 장시간 켜두었을 때 발생할 수 있는 메모리 누수를 원천 차단했습니다.
- **종료 프로세스 안정화:** 앱 종료 시, 백그라운드에서 돌고 있던 타이머(폴링 루프, 필드보스 알림 등)를 안전하게 중단시키도록 보완했습니다.
- **네이티브 API 안정성:** 윈도우 환경에 따라 Win32 DLL(user32 등) 로드에 실패할 경우 앱 전체가 크래시 나지 않고 예비 동작(Mock)으로 전환되도록 방어 코드를 적용했습니다.
- **내부 코드 최적화:** 앱 전반에 걸쳐 사용되지 않는 변수 및 쓰레기 코드(Dead Code)를 정리하여 컴파일러 경고(0-warning)를 완전히 제거하고 빌드 속도를 개선했습니다.

## 🐛 Fixed (버그 수정)
- **다중 모니터 좌표 보정:** 듀얼 모니터 환경에서 주 모니터와 게임이 켜진 모니터의 윈도우 배율(DPI 125% 등)이 다를 경우, 오버레이 창이 엉뚱한 곳에 뜨던 버그를 수정했습니다. 이제 게임이 어떤 배율의 모니터에 있든 정확히 옆에 자석처럼 붙습니다.
- **마우스 포커스 깜빡임:** 우리 앱과 게임 창 사이에서 0.1초 단위로 포커스를 뺏고 뺏기며 마우스 입력이 씹히던 현상(Z-Order 루프)을 막기 위해 디바운싱(Debounce) 안전장치를 도입했습니다.
- **외부 링크 보안:** 악의적인 URL 주입을 막기 위해, 브라우저가 새 창을 열 때 한 번 더 프로토콜(http/https)을 깐깐하게 검증하도록 패치했습니다.
