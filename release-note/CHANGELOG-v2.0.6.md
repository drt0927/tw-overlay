# TW-Overlay v2.0.6 릴리즈 노트 (2026.07.04)

이번 업데이트는 커스텀 알림 화면 가로 확장, 알림별 개별 볼륨 슬라이더 추가, 설정창 바로가기 하이라이트 효과 및 1/2depth 탭 간헐적 먹통 현상 해결을 담은 **패치 v2.0.6 업데이트**입니다.

---

## 🚀 주요 변경 사항

### ⚙️ Changed (기능 개선 사항)

* **📢 커스텀 알림 화면 가로 확장**:
  * 커스텀 알림 설정 창의 너비를 420px에서 580px로 키워 가독성과 여백을 대폭 개선했습니다.
  * 생성 및 수정 폼의 레이아웃을 2열 그리드로 정렬하여 메시지와 시간을 한눈에 입력할 수 있도록 UX를 개편했습니다.
* **📢 알림음별 개별 볼륨 설정 기능 도입**:
  * 폼 내 및 각 알림 카드 하단에 개별 볼륨 조절 슬라이더(0% ~ 100%)를 추가하여, 각 알림음 크기를 독립적으로 조절하여 저장할 수 있습니다.
  * 미리듣기 버튼을 연속해서 클릭할 시, 이전 재생 중이던 소리를 즉각 정지하고 새 소리를 처음부터 즉시 다시 재생(인터럽트 재생)하게 처리했습니다.
* **✨ 설정창 대메뉴 기능별 알림 바로가기 하이라이트**:
  * 설정창 "소리 및 알림" 서브 탭 하단의 기능별 알림 설정 바로가기 링크를 클릭했을 때, 해당 설정 창이 켜지면서(이미 켜져 있다면 최상단으로 포커싱) 타겟 설정 영역이 해당 창의 시그니처 테마 색상(Red, Indigo, Orange, Blue)으로 세련되게 깜빡이는 시각적 피드백 효과를 추가했습니다.

### 🐛 Fixed (버그 수정 및 최적화)

* **🐛 설정창 1/2depth 탭 전환 간헐적 가림 현상 해결**:
  * 서브 탭 전환 시 active 클래스 정리 범위를 현재 섹션 내부로 제한하여 다른 탭의 가시성이 파괴되던 문제를 해결했습니다.
  * 대메뉴 1depth 이동 시 해당 섹션 내 활성화된 서브 탭이 없다면 자동으로 첫 번째 서브 탭을 활성화하는 안전장치를 추가했습니다.
* **🐛 버프 타이머 설정 창 내 사운드 드롭다운 찌그러짐 현상 해결**:
  * 드롭다운이 찌그러지거나 미리보기 버튼이 카드 바깥으로 삐져나가는 현상을 `min-w-0` 및 `w-full` 속성 조치로 해결했습니다.
* **🎨 경험치 HUD 미리듣기 버튼 디자인 및 테마 컬러 통일**:
  * 미리듣기 버튼을 시그니처 테마 색상(Blue/Sky Blue)과 정사각형 play 아이콘 형태로 통일하여 시각적 완성도를 높였습니다.

## 📥 설치 및 적용 방법

**방법 1: 앱 내 설정 메뉴를 통한 업데이트 (권장)**
1. 앱 내부에서 **[설정]** 메뉴를 멉니다.
2. **[앱 정보]** 탭으로 이동한 뒤 **[업데이트 확인]** 버튼을 클릭하여 수동 업데이트를 진행합니다.

**방법 2: 직접 다운로드하여 업데이트**
1. [Releases](https://github.com/drt0927/tw-overlay/releases) 페이지에서 최신 버전의 `twOverlay-Setup-2.0.6.exe` 파일을 다운로드합니다.
2. 기존 앱을 완전히 종료하신 후, 다운로드한 `.exe` 설치 파일을 실행하여 설치를 진행합니다.
