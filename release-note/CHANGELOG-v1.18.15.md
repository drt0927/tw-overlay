# TW-Overlay v1.18.15 릴리즈 노트 (2026.06.26)

이번 업데이트는 **전체화면 상태에서 타이머 알람(사이드바 크기 변화)이 올 때 채팅 오버레이 창이 좌측으로 튕기는 버그 수정**이 적용되었습니다.

## 🛠️ 주요 변경 사항 (CHANGELOG)

### Fixed (버그 수정)
* **전체화면 상태에서 사이드바 리사이즈 시 오버레이 위치 튕김 버그 수정**:
  * 타이머 알람 발생 또는 환경설정 변경 등으로 메인 사이드바가 확장/축소될 때, 예외 조건 누락으로 자식 오버레이 창들이 강제로 좌표 갱신되며 기본 위치(좌측 부근)로 snap 되던 현상을 해결했습니다.
  * 사이드바 변경(`isSidebarResize`)으로 인한 자식 창 재배치 수행 시 게임이 전체화면 모드(`isGameFullscreen`)인 경우에는 개별 오버레이 창들의 위치 조정을 건너뛰도록 처리했습니다.

---

## 📥 설치 방법

1. 본 릴리즈 페이지 최하단의 **Assets** 섹션에서 `twOverlay-Setup-1.18.15.exe` 파일을 클릭하여 다운로드합니다.
2. 기존에 실행 중이던 `tw-overlay` 앱이 있다면 반드시 완전히 종료한 뒤 설치 파일을 실행합니다.
3. 화면의 안내에 따라 설치를 완료하면 새로운 기능과 패치가 적용됩니다.
