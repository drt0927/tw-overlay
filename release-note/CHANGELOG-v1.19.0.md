# TW-Overlay v1.19.0 릴리즈 노트 (2026.06.26)

이번 업데이트는 외부에서 단독으로 제공되던 경로 편집기를 메인 프로그램으로 이식하고 고도화한 **'사냥터 동선 시뮬레이션' 기능 신규 통합** 및 **마우스 투과 모드 버그 수정**이 적용되었습니다.

## 🛠️ 주요 변경 사항 (CHANGELOG)

### Added (신규 기능)
* **사냥터 동선 시뮬레이션 기능 신규 통합**:
  * 기존의 외부 단독 `path_editor.html`을 삭제하고, 메인 사이드바 및 하단 독(Dock) 메뉴에 Emerald 테마 디자인을 통일 반영하여 내장형 창으로 전면 이식했습니다.
  * 창을 닫아도 경로가 초기화되지 않고 SQLite 데이터베이스(`diary.db`)의 `hunting_grounds` 및 `hunting_paths` 테이블에 실시간 자동 영구 저장되어, 언제든 다시 열었을 때 동선이 즉시 복원되도록 보존 기능을 완비했습니다.
  * 맵 위의 불필요한 기본 선 및 파란색 기준점 요소를 완전히 소거하여 맵을 깔끔하게 유지할 수 있도록 개편했습니다.
  * 마우스 우클릭 및 Ctrl+클릭을 활용하여 노드 수의 제한 없이(0개 노드까지) 자유롭게 추가, 이동, 삭제 편집이 가능하도록 제약을 완화했습니다.
  * 시스템 트레이 우클릭 메뉴에서도 사냥터 동선 시뮬레이션을 원클릭으로 직접 켜고 끌 수 있도록 메뉴 링크를 추가 연동했습니다.

### Fixed (버그 수정)
* **마우스 투과(Click-Through) 상태 유지 및 초기 구동 오작동 해결**:
  * 마우스 투과 상태를 활성화한 채로 프로그램을 껐다 켤 때, 초기 구동 시 투과 플래그가 비동기 설정에 정확히 일치하지 않아 비투과로 되돌리려면 토글을 두 번 눌러야 했던 오작동 버그를 수정했습니다.

---

## 📥 설치 방법

1. 본 릴리즈 페이지 최하단의 **Assets** 섹션에서 `twOverlay-Setup-1.19.0.exe` 파일을 클릭하여 다운로드합니다.
2. 기존에 실행 중이던 `tw-overlay` 앱이 있다면 반드시 완전히 종료한 뒤 설치 파일을 실행합니다.
3. 화면의 안내에 따라 설치를 완료하면 새로운 기능과 패치가 적용됩니다.
