# TW-Overlay v1.17.1 릴리즈 노트 (2026.06.04)

이번 패치 업데이트는 버프 만료 임박 시 사용자 인지를 돕는 **화면 중앙 대형 만료 경고 오버레이**를 추가하고, 여러 버프가 동시에 꺼질 때 나타나는 화면 중첩 이슈를 우아하게 해결한 **알림 스택(Stack) 아키텍처**를 적용했습니다. 

또한 설정의 간소화를 위해 전역 단일 토글을 도입하고, 테스트 시 실시간 설정을 선제 연동하며 이자벨 비율 버프의 미세한 잔재 코드를 깔끔하게 정리하여 사용성과 안전성을 강화했습니다.

---

## 🌟 Added (새로운 기능 추가)

- **📢 화면 중앙 대형 만료 경고 오버레이(Alert Overlay) 추가**:
  * 버프 만료 **5초 전(임박 경고 시점)**에 도달하면 화면 중앙에 약 2.3초간 강력한 만료 경고 카드가 활성화되어 캐릭터 버프 리필 주기를 명확히 알려줍니다.
  * 시야 및 중요 화면을 과도하게 방해하지 않도록 가로 240px × 세로 38px 규격의 **매우 슬림하고 콤팩트한 가로 바(Bar) 형태**로 카드를 최적화 설계했습니다.
  * 다크 글래스모피즘(`backdrop-filter: blur(10px)`) 백그라운드에 은은한 보라색 광채 테두리를 주어 프리미엄한 감각의 디자인을 완성했습니다.

- **🥞 동시 만료 대처를 위한 알림 중첩 스택(Stack) 탑재**:
  * 3~4개의 버프가 거의 동시에 만료되거나 5초 전이 될 때 화면 한가운데에 경고창이 겹치거나 뒤의 카드가 가리는 현상을 완벽히 리팩토링했습니다.
  * 경고 수신 시 카드를 동적으로 렌더링하여 컨테이너 내부에 수직 정렬(`flex-direction: column`)로 차곡차곡 쌓이게 만들었습니다.
  * 각 카드는 **정확히 2.3초 후(2000ms 유지 + 300ms 페이드아웃)에 DOM 트리에서 스스로 완벽하게 소멸(`card.remove()`)**됩니다.
  * 카드가 소멸하면 아래에 있던 카드들이 Flexbox 레이아웃의 유동성에 의해 자연스럽게 위로 슥 슬라이드 업(Slide Up) 정렬되어 뛰어난 시각적 마감을 제공합니다.

- **⚙️ 대형 알림 전역 설정 토글 구현**:
  * 각 버프마다 눈 모양(시각 알림) 스위치를 일일이 제어하던 임시 구조를 전역으로 일원화했습니다.
  * 환경 설정 창 우측 시각 알림 영역 하단에 **"화면 중앙 대형 알림" 전역 제어 스위치** 하나만 배치하여 한 번의 조작으로 전체 대형 팝업의 활성/비활성을 간소화할 수 있게 개선했습니다. (해제 시 원래 제공되던 우하단 HUD 게이지 및 배지의 깜빡임은 기존대로 100% 정상 가동됩니다.)

---

## ⚡ Changed (변경 및 개선 사항)

- **⚡ 테스트 미리보기 시 설정 실시간 연동**:
  * 설정 화면에서 옵션(예: 화면 중앙 대형 알림 끄기 등)을 수정한 뒤, "저장" 버튼을 미처 누르지 않은 임시 상태에서 하단의 "테스트 (미리보기)"를 실행하더라도 **현재 UI의 변경 사항이 즉시 메인 프로세스에 연동 반영(`applySettings` 선제 실행)**되어 테스트가 구동되도록 개선했습니다.
  * 이로 인해 저장 과정을 깜빡해 테스트 결과를 잘못 오인하게 되는 문제를 원천 해결했습니다.

---

## 🐛 Fixed (버그 수정)

- **🧹 이자벨의 비법 (비율) 잔재 코드 완전 제거**:
  * 버프 타이머 감지 사전에서 이전에 배제되었음에도 불구하고, 테스트 시뮬레이션 목록(`TEST_BUFFS`), 기본 설정값(`DEFAULT_CONFIG`), 그리고 설정 로드 시의 폴백 객체에 남아 테스트 미리보기 시 여전히 이자벨 비율 버프가 튀어나오던 잔재를 깨끗이 찾아내어 제거했습니다.
  * 이제 테스트 버튼을 클릭하셔도 실제 등록된 7종의 정식 추적 버프에 대해서만 깨끗하게 미리보기가 동작합니다.
