# TalesWeaver Companion Browser (TW-Overlay) 프로젝트 가이드 (v1.11.1)

이 문서는 v1.11.1 버전을 기준으로 작성되었습니다.

## 1. 개요 및 최종 상태
- **목적:** 테일즈위버 전용 지능형 오버레이 브라우저 및 실시간 게임 데이터 분석 위젯.
- **최종 빌드:** **TypeScript**, Electron, **Native Win32 API(Koffi)**, **tail (Log Streaming)**, **SQLite(better-sqlite3)** 기반.
- **주요 개선:** 약어 사전 대규모 업데이트 및 채팅 로그 분석 엔진 고도화.

## 2. 주요 기능 명세
- **Real-time Log Engine:** 테일즈위버 채팅 로그를 실시간 분석하여 경험치, 아이템, 외치기 이벤트를 추출.
- **XP/EPM Dashboard:** 게임 화면 위 오버레이 HUD를 통해 실시간 사냥 효율(총 경험치, 분당 경험치)을 표시.
- **Shout History Viewer:** 수집된 외치기 목록을 검색하고 닉네임을 원클릭 복사하는 전용 뷰어 제공.
- **Adventure Log Sync:** 로그에서 감지된 SEED 수익 및 지정 키워드 아이템 획득 시 모험 일지에 자동 기록.
- **Tales Dictionary:** 160개 이상의 방대한 게임 약어 및 용어 정보를 제공하는 지능형 사전.
- **Hybrid Notifier:** 게임 최소화/종료 시 Windows 네이티브 알림(Toast) 발송 (필드보스, 커스텀 알람, 득템 알림).

- **업데이트 히스토리 (v1.11.1 패치)**
- **신규 기능:** 약어 사전 데이터 전수 확충 (알롱지, 햄찌님 데이터 통합).
- **UI/UX:** 약어 사전 전면 리디자인 (Blue 시그니처 테마 및 가독성 개선).
- **버그 수정:** 자정 이후 채팅 로그 날짜 미갱신 이슈 해결.
- **데이터 보강:** 모험 일지 수익 기록 시 로그 원문 메시지 포함 저장.

- **업데이트 히스토리 (v1.11.0 핵심)**
- **신규 기능:** 실시간 채팅 로그 분석 엔진 및 로그 기반 자동화 시스템 구축.
- **UI/UX:** 경험치 HUD 오버레이 추가 및 위치 조정 기능 제공.
- **유틸리티:** 외치기 히스토리 창 추가 (검색 및 닉네임 복사).
- **안정성:** `tail` 기반 로그 추적으로 대용량 파일 핸들링 안정성 확보.

- **업데이트 히스토리 (v1.10.6)**
- **신규 기능:** 제복 색상 시뮬레이터 (비설화님 연동) 추가.
- **아키텍처 개선:** 외부 사이트 연동을 위한 전용 `WebContentsView` 로직 구현.
