# 마정석 계산기 (Magic Stone Calculator)

## 1. 기능 개요 및 목적
게임 내에서 획득한 각종 마정석을 '금화 주머니'로 교환했을 때의 총 시드(Seed) 가치를 계산해주는 도구입니다. 복잡한 교환 비율을 외울 필요 없이 보유 수량만 입력하면 최종 수익을 바로 확인하고 '모험 일지'에 기록할 수 있습니다.

## 2. 주요 UI 구성 요소 설명
- **아이템 입력 필드:** 보유 중인 금화 주머니 및 하급/중급/상급/최상급 마정석의 수량을 입력합니다.
- **어벤던로드 트래커 (Abandon Road Tracker):**
  - 실시간으로 어벤던로드 입장 비용(누적 시드), 남은 판수, 획득한 마정석 가치를 백경에서 실시간으로 자동 추적하여 표기해 줍니다.
  - 사냥 세션 동안의 누적 득템 결과를 마정석 계산기 입력 필드에 자동으로 연계 대입해 줍니다.
- **교환 정보 가이드:** 각 등급별 마정석이 몇 개의 금화 주머니와 매칭되는지 안내합니다.
- **최종 Result 영역:** 총 금화 주머니 개수와 이를 시드로 환산한 금액을 표시합니다.
- **일지 기록 버튼:** 계산된 총 수익을 현재 날짜의 '모험 일지' 수익 내역으로 즉시 저장합니다.

## 3. 세부 기능 및 작동 방식
- **어벤던로드 10판 완료 알림:** 어벤던로드 사냥 진행도를 실시간 감지하여 10판을 모두 완료하는 순간, 화면 전체 플래시 효과와 함께 대형 완료 알림 팝업을 출력해 사냥 종료 시점을 명확히 인지하게 돕습니다.
- **팔색조 언덕 알람:** 보스 에너지를 실시간 추적하여 보스 처치수 5회 도달 시 클리어 알림을 누락 없이 출력합니다.
- **실시간 가치 환산:** 1금화 주머니당 50만 시드(SEED_PER_POUCH) 기준을 적용하여 수량 입력 즉시 결과를 업데이트합니다.
- **금액 가독성 변환:** 큰 숫자의 시드 수치를 '억', '만' 단위의 한글 표기로 병기하여 직관적인 금액 파악을 돕습니다.
- **모험 일지 자동 연동:** 버튼 클릭 시 현재 시각과 계산된 수익 정보를 `diary.db`에 기록하고 시각적인 피드백(체크 아이콘 변경)을 제공합니다.
- **효과음 재생:** 일지 기록 성공 시 '아이템 획득' 효과음을 재생하여 사용자 경험을 강화합니다.
