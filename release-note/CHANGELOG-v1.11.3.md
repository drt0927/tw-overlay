# TW-Overlay Release Note [v1.11.3]

v1.11.3 패치는 **사이드바 좌/우 배치 설정** 기능을 신규 도입하고, 버프 타이머의 **인게임 HUD 개별 제어** 및 **테스트 편의 기능**을 강화했습니다.

## Added (신규 기능)

- **사이드바 좌/우 배치 설정**: 환경설정에서 사이드바를 게임 창 왼쪽 또는 오른쪽에 배치할 수 있습니다.
  - 왼쪽 배치 시 CSS 레이아웃 자동 반전 (보더 방향, 요소 위치)
  - AppConfig에 `sidebarPosition` 설정 필드 추가

- **버프 타이머 HUD 표시 토글**: 알림 기능은 유지하면서 인게임 HUD 표시만 개별적으로 켜고 끌 수 있습니다.

- **버프 타이머 테스트 모드**: 초 단위 지속시간을 지정하여 소리 및 애니메이션을 즉시 확인할 수 있습니다.

## Changed (변경 및 개선)

- **제복 색상 시뮬레이터**: `ui-utils.js` 로드 추가 및 ESC 키로 창 닫기 지원.
- **버프 타이머 설정 변경 시 오버레이에 실시간 반영**되도록 동기화 보강.

## Fixed (버그 수정)

- **왼쪽 사이드바 깜박임 수정**: 마우스 호버 시 사이드바 창의 width(50↔400) 변경으로 인해 X 좌표가 350px씩 점프하던 현상을 해결했습니다. 왼쪽 배치 시 width를 고정하여 윈도우 위치 변경을 방지합니다.
- **마우스 판정 안정화**: `window.innerWidth` 기반의 불안정한 좌표 계산 대신 DOM 요소의 실제 경계(`getBoundingClientRect`)를 사용하도록 전환하여 정확도를 높였습니다.
- **사이드바 축소 타이머 누수 수정**: `shrinkTimer`가 취소되지 않아 발생하던 불필요한 리사이즈 사이클을 차단합니다.
