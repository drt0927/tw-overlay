# TW-Overlay v1.18.5 릴리즈 노트 (2026.06.18)

이번 업데이트는 **이클립스 로카고스 기믹 시각적 알림 추가 및 기믹 기본 설정값 조정**이 적용되었습니다.

## 🛠️ 주요 변경 사항 (CHANGELOG)

### Added (신규 기능 추가)
* **이클립스 로카고스 보스전 기믹 알림 추가**:
  - 선봉대장 로카고스의 마법 공격 대사를 추적하여 기믹 대피 정보를 실시간 제공합니다.
  - **대피 기믹** (지정 구역 제외 공격) 시: 화면 중앙에 큼직한 안전 구역 네온 뱃지와 직관적인 컬러 가이드 텍스트(예: `자주색 (A)으로 이동!`), 그리고 화면 테두리에 은은하게 깜빡이는 전용 컬러 플래시를 노출합니다.
  - **회피 기믹** (지정 구역 공격) 시: 폭격 구역의 원형 뱃지에 🚫 마크를 씌우고 텍스트에 취소선을 적용하여 회피 대상을 알립니다. 또한, 하단에 안전한 나머지 3개 구역의 미니 원형 뱃지를 나란히 배치하여 혼선 없는 빠른 판단을 돕습니다.
* **설정 UI 내 켜기/끄기 옵션 제공**:
  - 설정 화면의 '인게임 기믹 및 시스템 알림' 영역에 `이클립스 로카고스 기믹 알림` 체크박스 옵션을 추가하여 알림 동작 여부를 사용자가 개별 제어할 수 있도록 하였습니다.

### Changed (변경 및 개선)
* **기믹 알림 초기값 '사용 안 함'으로 통일**:
  - 유저의 플레이 편의를 위해 이클립스 에토스 기믹, 심연의 제2사도 기믹 및 신규 로카고스 기믹의 최초 기본 활성화 상태를 `false`(사용 안 함)로 일관되게 정정했습니다.

---

## 📥 설치 방법

1. 본 릴리즈 페이지 최하단의 **Assets** 섹션에서 `twOverlay-Setup-1.18.5.exe` 파일을 클릭하여 다운로드합니다.
2. 기존에 실행 중이던 `tw-overlay` 앱이 있다면 반드시 완전히 종료한 뒤 설치 파일을 실행합니다.
3. 화면의 안내에 따라 설치를 완료하면 새로운 기능과 패치가 적용됩니다.
