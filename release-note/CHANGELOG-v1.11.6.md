# 📦 TW-Overlay Release Note [v1.11.6]

v1.11.6 패치는 사이드바 메뉴 조작 시 발생하던 윈도우 위치 어긋남과 크기 불일치 문제를 해결하여 사용 편의성을 개선했습니다.

## 🐞 Fixed (버그 수정)

- **사이드바 위치 이탈 문제 해결**:
  - 메뉴 클릭 등으로 사이드바 너비가 변할 때, 간헐적으로 게임창 안으로 침범하던 현상을 수정했습니다.
  - 이제 너비가 변할 때마다 최신 게임창 좌표(`gameRect`)를 다시 확인하여 정확한 위치로 자석(Anchoring) 기능이 즉시 재실행됩니다.

- **사이드바 높이 오류 수정**:
  - 사이드바 확장 시 높이가 게임창과 맞지 않거나 기본값(800px)에 고정되던 문제를 해결했습니다.
  - 리사이징 시에도 항상 게임창의 현재 높이에 맞춰 사이드바 크기가 동기화됩니다.

- **화면 경계 충돌 방어**:
  - 모니터 우측 끝에서 사이드바가 확장될 때, 윈도우 OS가 창을 왼쪽(게임창 쪽)으로 밀어넣는 현상을 코드 레벨에서 감지하고 재교정합니다.
