# 📦 TW-Overlay Release Note (v1.6.1)

## 🗓 배포일: 2026-02-28

## ✨ 신규 및 개선 사항 (Added / Changed)

### 1. 오버레이 창 이동 제한 해제 (Overlay Freedom)
- 오버레이 브라우저 창이 테일즈위버 게임 창 영역 내부로만 이동할 수 있었던 기존의 제한(Clamping)을 제거했습니다.
- 이제 오버레이 창을 모니터의 어느 위치로든 자유롭게 드래그하여 배치할 수 있습니다.
- 게임 창을 이동시켜도 오버레이 창이 설정된 상대적인 위치(오프셋)를 유지하며 자석처럼 함께 이동합니다.

### 2. 위치 동기화 로직 고도화
- 게임 창 밖에서도 오버레이의 위치와 크기가 안정적으로 유지되도록 내부 트래킹 로직을 개선했습니다.

## 🛠 버그 수정 (Fixed)
- 오버레이 창의 크기가 게임 창보다 클 경우 강제로 축소되던 현상을 해결했습니다.

---
**Note:** 이 버전은 사용자의 "오버레이를 밖으로도 빼고 싶다"는 요청을 즉각 반영한 패치 버전입니다.
