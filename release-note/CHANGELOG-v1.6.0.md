# 🛒 TW-Overlay v1.6.0 릴리즈 노트

이번 업데이트에서는 **매직위버 거래 게시판 모니터링 기능**을 새롭게 추가 했습니다. 또한, 보안 정책 등의 이유로 더 이상 사용되지 않는 **장판 및 버프 감지 기능을 공식적으로 중단**하고 관련 내용을 정리했습니다.

---

## 🚀 주요 변경 사항

### 1. 매직위버 거래 게시판 모니터 (New)
- **실시간 모니터링:** 하이아칸 및 네냐플 서버의 거래 게시판 글을 실시간(5분 간격)으로 감시합니다.
- **키워드 알림:** 사용자가 등록한 키워드가 포함된 글이 올라오면 알림을 제공합니다.
- **자체 검색 연동:** 카페 자체 검색 기능을 활용하여 정확한 키워드 필터링이 가능합니다.

### 2. UI/UX 통합 및 개선 (Changed)
- **모니터 창 통합:** 갤러리 모니터와 거래 모니터의 창 사이즈를 380x600으로 통일하고, 헤더 및 배너 디자인을 일관성 있게 개편했습니다.
- **다이나믹 안내 배너:** 키워드 등록 상태에 따라 상단 배너에 안내 문구와 설정 바로가기 버튼이 항상 노출되어 접근성을 높였습니다.
- **설정 바로가기:** 각 모니터 창 상단의 버튼을 통해 설정 페이지의 해당 탭으로 즉시 이동할 수 있습니다.

### 3. 기능 중단 및 코드 정리 (Removed)
- **장판/버프 감지 제거:** 화면 분석 기반의 장판 및 버프 감지 기능을 폐쇄했습니다.
- **문서 최적화:** 프로젝트 내 모든 가이드 및 README에서 해당 기능 설명을 삭제하고 최신 버전(v1.6.0) 가이드를 반영했습니다.

---

## 🛠 수정 및 개선 사항 (Fixed/Improved)
- 설정을 변경했을 때 모니터링 모듈에 즉시 반영되지 않던 버그를 수정했습니다.
- 갤러리 모니터 창의 크기를 확대하여 가독성을 개선했습니다.
