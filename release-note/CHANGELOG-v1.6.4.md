# CHANGELOG v1.6.4

이 릴리즈는 직전 버전에 배포된 "계수 및 명중 계산기"의 사용성을 보완하고, 기타 안정성 패치를 완전히 통합한 정식 패치입니다.

## ✨ Added (신규 기능 업데이트)
- **계수 계산기 출처 링크 추가:** 스탯/장비 계수 및 명중률 계산에 활용된 공식 출처(구글 스프레드시트, 디시인사이드 정보글 등 3곳)를 계산기 뷰 화면 우측 하단에 명시적으로 추가했습니다.

## 🚀 Changed (개선 및 최적화)
- **메모리 최적화:** 모든 독립 창(오버레이, 갤러리, 계산기 등)이 닫힐 때 관련된 IPC 이벤트 리스너를 완전히 해제(`cleanupAllListeners`)하여 메모리 누수를 방지하도록 개선했습니다.
- **네이티브 방어 로직:** 윈도우 환경에 따라 Win32 DLL(user32 등) 로드에 실패하더라도 앱이 강제 종료(Crash)되지 않도록 안전장치를 마련했습니다.

## 🐛 Fixed (버그 수정)
- **다중 모니터 좌표 보정:** 듀얼 모니터 환경에서 주 모니터와 게임 모니터의 DPI(화면 배율)가 다를 때 오버레이 창이 엉뚱한 위치로 가는 버그를 수정했습니다.
- **마우스 포커스 깜빡임 해결:** 우리 앱과 게임 창 사이에서 Z-Order를 조정할 때 포커스 무한 루프가 발생하여 클릭이 무시되던 현상(Debounce 처리)을 해결했습니다.
- **보안 강화:** 외부 링크 접근 시 비정상적인 프로토콜이 주입되지 않도록 URL 파싱 및 검증(http/https)을 강화했습니다.
