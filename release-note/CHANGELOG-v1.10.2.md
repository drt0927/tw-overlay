# 📦 TW-Overlay 릴리즈 노트 (v1.10.2)

## 📅 배포 일자
2026년 3월 29일

## 🚀 주요 변경 사항

### ✅ 숙제 체크리스트 (Contents Checker) 개선
- **이름 수정 기능 추가**: 사용자가 직접 추가한 숙제에 한해 이름을 수정할 수 있는 연필 아이콘 버튼을 추가했습니다.
- **커스텀 프롬프트 모달 구현**: Electron 환경의 제약을 극복하기 위해 스타일리시한 입력창 모달을 직접 제작하여 적용했습니다.
- **드래그 앤 드롭(Reorder) 안정성 대폭 강화**:
    - **이벤트 위임(Event Delegation)** 방식을 적용하여 동적 렌더링 시에도 기능이 중단되지 않도록 개선했습니다.
    - **윈도우 레벨 마우스 추적**을 통해 마우스가 창 밖으로 나가도 드래그 상태가 유지됩니다.
    - **플레이스홀더 최적화**로 이동 위치 미리보기 시 발생하는 화면 깜빡임을 제거했습니다.
    - **렌더링 보호 로직**을 추가하여 드래그 도중 데이터 동기화로 인해 아이템이 사라지는 현상을 방지했습니다.

## 🛠 버그 수정 및 개선
- **드래그 간섭 해결**: Electron의 창 드래그 설정과 HTML5 드래그 이벤트 간의 충돌 문제를 완벽히 해결했습니다.
- **포커스 자동 리셋**: 창 포커스 전환 시 드래그 상태가 꼬일 경우를 대비해 상태 자동 초기화 로직을 추가했습니다.
