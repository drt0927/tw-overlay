# TW-Overlay v1.18.6 릴리즈 노트 (2026.06.19)

이번 업데이트는 **지능형 버프 타이머 설정 시간 변경 및 단축키 실행 시 프레임 드랍 개선**이 적용되었습니다.

## 🛠️ 주요 변경 사항 (CHANGELOG)

### Changed (변경 및 개선)
* **지능형 버프 타이머 일부 버프 설정 시간 변경**:
  - 실제 인게임 스펙에 맞춰 `최상급 에오스의 파편` 및 `얼리버드 경험치 부스터` 버프의 기본 설정 시간을 기존 1시간에서 **30분**으로 조정했습니다.
  - 버프 타이머 HUD 화면에 노출되는 설명 텍스트 역시 '30분 유지'로 변경되었습니다.
* **단축키 실행 및 설정 변경 시 프레임 드랍 개선**:
  - 기존 단축키(예: `Ctrl + Shift + B` 버프 HUD 토글)를 입력하여 설정을 제어할 때, 동기식 디스크 저장 방식(`fs.writeFileSync`)이 메인 스레드를 블로킹하여 발생하던 순간적인 프레임 드랍(렉) 현상을 수정했습니다.
  - 디스크 저장 로직을 **비동기 디바운스 방식**(`config.save`)으로 개선하여, 게임 플레이 도중 렉 없이 부드럽게 오버레이를 조작할 수 있도록 했습니다.

---

## 📥 설치 방법

1. 본 릴리즈 페이지 최하단의 **Assets** 섹션에서 `twOverlay-Setup-1.18.6.exe` 파일을 클릭하여 다운로드합니다.
2. 기존에 실행 중이던 `tw-overlay` 앱이 있다면 반드시 완전히 종료한 뒤 설치 파일을 실행합니다.
3. 화면의 안내에 따라 설치를 완료하면 새로운 기능과 패치가 적용됩니다.
