# 📦 Changelog - v1.6.5

## Added
- **게임 종료 리마인더:** 테일즈위버 게임 종료 시 사용자가 설정해둔 리마인더 메시지를 커스텀 팝업 창으로 표시하는 기능 추가.
  - 환경설정 → 일반 설정에서 리마인더 활성화/비활성화 및 메시지 입력 가능.
  - 프로젝트 디자인 톤에 맞춘 글래스모피즘 스타일의 독립 팝업 창 (`game-exit-reminder.html`).

## Changed
- **게임 종료 시 프로그램 유지:** 게임이 종료되어도 TW-Overlay 프로그램이 함께 종료되지 않도록 변경. 프로그램은 백그라운드에서 유지되며 게임 재시작 시 자동으로 다시 추적.
