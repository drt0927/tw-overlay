# 📋 TW-Overlay 릴리즈 노트 (v1.10.0)

본 버전에서는 사용자의 플레이 기록을 영구적으로 저장하고 관리할 수 있는 **'모험 일지'** 시스템과 데이터 안전을 위한 **'백업/복구'** 기능이 추가되었습니다. 또한, 전체적인 UI 테마를 개편하여 더욱 화사하고 일관된 경험을 제공합니다.

## 🚀 Added (신규 기능)

### 1. 모험 일지 (Adventure Log) 시스템 도입
- **SQLite DB 기반:** 이제 앱을 재시작해도 보스 처치 기록, 숙제 완료 기록이 사라지지 않고 데이터베이스에 영구 보관됩니다.
- **달력형 UI:** 일지 전용 창을 통해 일별 활동 내역을 한눈에 확인할 수 있습니다.
- **성장 아이콘 시스템:** 하루 동안의 활동(숙제, 보스, 수익 등) 점수를 합산하여 아이콘이 알에서 최종 단계까지 변화하는 시각적 피드백을 제공합니다.
- **득템 및 수익 기록:** 득템한 아이템 명칭과 수량, 마정석 환전 수익 등을 날짜별로 꼼꼼하게 기록할 수 있습니다.

### 2. 마정석 계산기 신규 추가
- **간편 계산:** 하급부터 최상급까지 마정석 4종과 금화 주머니 개수를 입력하면 현재 시세에 맞춰 총 수익을 자동 계산합니다.
- **일지 연동:** 계산된 결과를 버튼 하나로 당일 모험 일지 타임라인에 즉시 등록하여 수익을 누적 관리할 수 있습니다.

### 3. 통합 데이터 백업 및 복구
- **ZIP 압축 방식:** 현재의 모든 설정(`config.json`)과 일지 DB(`diary.db`)를 하나의 압축 파일로 안전하게 백업할 수 있습니다.
- **간편 복구:** 환경 설정의 [데이터 관리] 탭에서 백업 파일을 불러오기만 하면 즉시 모든 데이터가 복구되며 앱이 자동 재시작됩니다.

### 3. 동적 메뉴 및 통합 테마 시스템
- **중앙 관리:** 사이드바 메뉴를 `sidebar_menus.json`으로 통합 관리하여 안정성을 높였습니다.
- **컬러 매칭 테마:** 사이드바 아이콘의 알록달록한 색상에 맞춰, 해당 메뉴를 눌렀을 때 열리는 모든 서브 창의 헤더와 버튼 색상이 자동으로 통일됩니다.

## 🔄 Changed (변경 및 개선 사항)
- **사이드바 메뉴 관리 개선:** 환경 설정에서 각 메뉴 옆에 실제 아이콘과 색상을 표시하여 더 직관적으로 메뉴를 켜고 끌 수 있게 개선했습니다.
- **가독성 향상:** 일지 상세 내역 및 월간 요약 리스트의 폰트 크기를 키우고 레이아웃을 정돈하여 시인성을 확보했습니다.

## 🐞 Fixed (버그 수정)
- **안정성 강화:** 특정 창을 닫는 순간 폴링 루프에서 발생하던 `Object has been destroyed` 오류를 수정하고, 이미 파괴된 객체에 대한 접근 보호 로직을 전면 적용했습니다.
- **HTML 구조 수정:** 환경 설정 탭 전환 시 화면이 깨지던 HTML 태그 중복 오류를 해결했습니다.
