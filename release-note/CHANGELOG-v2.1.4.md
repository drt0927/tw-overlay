# 📦 TW-Overlay Release Note - [Mandatory Update] v2.1.4 (2026-07-13)

이번 v2.1.4 버전에서는 설정 화면 내 기믹 알림음들의 연속 재생 꼬임 오류 및 먹통 현상과, 환경설정 창 로드 시의 비동기 레이스 컨디션으로 인해 알림음이 초기화되던 문제를 해결하여 오디오 서비스의 전반적인 안정성을 크게 개선하였습니다. 또한 소리 설정에서 기믹 설정으로 쉽게 넘어가실 수 있도록 바로가기 경로 단축 카드를 추가하였습니다.

---

## 🚀 주요 변경 사항

### 🛠️ Fixed (버그 수정)
- **기믹 알림음 연속 클릭 및 재생 먹통 오류 완벽 해결**:
  - 알림음 재생 시 이전 재생이 끝나기 전에 연속하여 다음 알림음 미리보기 등을 클릭할 경우, 크로미움 오디오 엔진의 재생 중단(`AbortError`) 예외와 프로토콜 캐시 문제로 인해 재생 대기열(`audioQueue`)이 영구히 고착되어 사운드가 재생되지 않던 버그를 완전히 해결하였습니다.
  - 기존 단일 오디오 엘리먼트를 재사용하던 방식에서 재생마다 독립된 오디오 인스턴스를 동적으로 생성하도록 개편하고, `tw-sound://` 프로토콜에 난수 캐시 버스팅 파라미터(`?t=...`)를 결합하여 브라우저의 미디어 캐시 버그를 원천적으로 차단하였습니다.
- **설정 창 로드 시 알림음 설정 '사용 안 함' 강제 초기화 수정**:
  - 설정 창 로드 시 설정값 동기화 이벤트와 사운드 파일 목록 로딩 간의 비동기 처리 타이밍 문제(레이스 컨디션)로 인해, 이미 알림음 설정값이 잘 저장되어 있음에도 화면 상에는 `사용 안 함 (소리 없음)`으로 강제로 초기화 및 재설정되던 동기화 로직 오류를 수정하였습니다.

### 🎵 Added (새로운 기능)
- **기믹 & 도전과제 알림 바로가기 카드 추가**:
  - 환경설정 - [소리 및 알림] - [기능별 알림음 설정 바로가기] 구역 내에 `기믹 & 도전과제 알림` 단축 카드를 추가하여, 클릭 시 즉시 `채팅 로그 설정 -> 기믹 & 도전과제` 탭으로 탭 스위칭이 자연스럽게 일어나도록 사용 편의성을 개선하였습니다.

## 📥 설치 및 적용 방법

**방법 1: 앱 내 설정 메뉴를 통한 업데이트 (권장)**
1. 앱 내부에서 **[설정]** 메뉴를 엽니다.
2. **[앱 정보]** 탭으로 이동한 뒤 **[업데이트 확인]** 버튼을 클릭하여 수동 업데이트를 진행합니다.

**방법 2: 직접 다운로드하여 업데이트**
1. [Releases](https://github.com/drt0927/tw-overlay/releases) 페이지에서 최신 버전의 `twOverlay-Setup-2.1.4.exe` 파일을 다운로드합니다.
2. 기존 앱을 완전히 종료하신 후, 다운로드한 `.exe` 설치 파일을 실행하여 설치를 진행합니다.
