# TW-Overlay v1.17.2 릴리즈 노트 (2026.06.05)

이번 업데이트는 게임창 최소화 시 리소스 최적화를 위해 모든 유틸리티 창을 닫는 과정에서 채팅 히스토리 오버레이 창이 영구적으로 비활성화되는 버그를 수정하여 게임창 최소화/복원 시 이전 활성화 상태 그대로 오버레이가 정상 복원되도록 개선했습니다.

---

## 🐛 Fixed (버그 수정)

- **게임창 최소화 시 채팅 히스토리 오버레이 강제 비활성화 문제 해결**:
  * 게임창 최소화 시 오버레이 창이 닫혀도 사용자의 수동 비활성화 동작과 구분하여 설정 값 및 상태 변수를 보존하도록 통제 흐름을 일원화했습니다.
  * 복원 시 이전 켜짐 상태를 감지하여 자동으로 다시 띄웁니다.

---

## 📦 설치 및 업데이트 방법

1. **설치 파일 다운로드**:
   * 본 릴리즈 페이지 최하단의 **Assets** 섹션에서 `twOverlay-Setup-1.17.2.exe` 파일을 클릭하여 다운로드합니다.
2. **프로그램 실행 및 설치**:
   * 다운로드한 `.exe` 파일을 실행하면 자동으로 설치 및 업데이트가 진행됩니다.
   * *참고: Windows SmartScreen 경고가 뜰 경우, `추가 정보` 클릭 후 `실행`을 누르면 됩니다.*
