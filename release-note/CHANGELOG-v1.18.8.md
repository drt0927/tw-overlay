# TW-Overlay v1.18.8 릴리즈 노트 (2026.06.20)

이번 업데이트는 **경험치 및 엘소(Elso) 습득 표시 선택 옵션 제공, 설정 화면 체크박스 UI 디자인 시스템 통일화**가 적용되었습니다.

## 🛠️ 주요 변경 사항 (CHANGELOG)

### Added (신규 기능 추가)
* **경험치 및 엘소 습득 표시 온/오프 기능 탑재**:
  - 환경 설정의 채팅 탭에 경험치 획득 정보와 엘소 획득 정보를 채팅 오버레이(전체/시스템 탭)에 노출할지 선택할 수 있는 체크박스 설정을 추가했습니다.
  - **경험치**: 옵션 체크 해제 시 경험치 HUD의 작동은 유지되나, 채팅 오버레이에서의 지속적인 경험치 획득 시스템 메시지는 차단됩니다.
  - **엘소**: 옵션 체크 해제 시 사냥/보상 드랍인 `[엘소 N포인트]을(를) [K]개 획득하였습니다.` 형식의 대량 득템 도배 로그만 정밀하게 필터링하여 채팅 오버레이에서 차단합니다. (단, 사용자가 인벤토리에서 더블 클릭하여 직접 포인트를 충전했을 때의 성공 알림 로그는 피드백 인지를 돕기 위해 차단 대상에서 제외되어 정상 노출됩니다.)

### Changed / Fixed (변경 및 성능 개선)
* **설정 화면 체크박스 UI 디자인 시스템 통일**:
  - 채팅 설정 탭 내의 일반적인 인라인 체크박스들을 다른 주요 알림 카드들과 동일하게 **카드형 레이아웃 + `w-5 h-5 accent-indigo-600` 스타일 + 보조 설명**으로 일괄 리팩토링하여 UI 일관성을 구현하고 시각적 품질을 높였습니다.

---

## 📥 설치 방법

1. 본 릴리즈 페이지 최하단의 **Assets** 섹션에서 `twOverlay-Setup-1.18.8.exe` 파일을 클릭하여 다운로드합니다.
2. 기존에 실행 중이던 `tw-overlay` 앱이 있다면 반드시 완전히 종료한 뒤 설치 파일을 실행합니다.
3. 화면의 안내에 따라 설치를 완료하면 새로운 기능과 패치가 적용됩니다.
