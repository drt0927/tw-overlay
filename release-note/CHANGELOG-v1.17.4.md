# TW-Overlay v1.17.4 릴리즈 노트 (2026.06.05)

이번 업데이트는 시스템 채팅 로그에 포함된 단어에 의해 발생하는 비정상적인 알림 버그와, 특정 상황에서 데이터베이스 오류로 인해 채팅 실시간 수집 프로세스가 멈추던 문제를 해결하여 안정성을 크게 강화했습니다.

---

## 🐛 Fixed (버그 수정)

- **지정 단어 알림 및 디스코드 전송 시 시스템 메시지 제외**:
  * 버프 사용 메시지 등 시스템이 남기는 채팅은 일반 및 클럽 지정 단어 알림 대상에서 제외되도록 수정하여 오알림을 차단했습니다.
- **알림 삭제 시 발생하는 데이터베이스 오류 크래시 해결**:
  * 지정 단어 알림이 발생한 직후 사용자가 알림을 즉시 삭제하는 경우, 후속 대화 수집 과정에서 외래 키(FOREIGN KEY) 제약 조건으로 인해 발생하던 데이터베이스 에러와 실시간 감시 엔진 멈춤 현상을 방지하는 예외 방어 처리를 적용했습니다.

---

## 📦 설치 및 업데이트 방법

1. **설치 파일 다운로드**:
   * 본 릴리즈 페이지 최하단의 **Assets** 섹션에서 `twOverlay-Setup-1.17.4.exe` 파일을 클릭하여 다운로드합니다.
2. **프로그램 실행 및 설치**:
   * 다운로드한 `.exe` 파일을 실행하면 자동으로 설치 및 업데이트가 진행됩니다.
   * *참고: Windows SmartScreen 경고가 뜰 경우, `추가 정보` 클릭 후 `실행`을 누르면 됩니다.*
