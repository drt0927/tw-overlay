# TW-Overlay v1.19.4 릴리즈 노트 (2026.06.28)

이번 업데이트는 **경험치 HUD 실시간 측정 수동 제어(시작/일시정지) 기능**, **경험치 세션 자동 시작 전역 설정**, 그리고 사냥터에서 간편하게 조작할 수 있는 **세션 제어 전역 단축키 및 단축키 설정 UI 연동**이 적용되었습니다.

---

## 🚀 주요 변경 사항

### 1. 📊 경험치 HUD 실시간 세션 측정 수동 제어 (시작/일시정지)
* **세션 제어 인터페이스 추가:**
  * 경험치 HUD 설정 및 초기화 창(`xp-hud.html`)의 '세션 경과 시간' 카드 내부에 재생/일시정지(`play` / `pause`) 원형 제어 버튼을 신설했습니다.
  * 작동 상태를 표시하는 LED 모양 배지(작동 중: 초록색 펄스 깜빡임, 정지: 빨간색)를 도입하여 현재 측정이 진행 중인지 한눈에 알 수 있게 설계했습니다.
* **일시정지 시 타이머 및 수집 중지 (데이터 오염 방지):**
  * 세션을 정지(일시정지)해 둔 동안에는 경과 시간 타이머가 멈추고, 게임 내 경험치 획득 정보가 들어와도 누적되지 않도록 백엔드 가드를 적용했습니다.
  * 일시정지 중 흐른 시간은 분당 평균 경험치(EPM) 계산 분모에서 완벽히 배제되며, 순수 활성 사냥 시간(`_accumulatedTime`)을 기준으로 EPM이 정확히 연산됩니다.
* **인게임 오버레이 동기화 시각 효과:**
  * 게임 중에도 측정 중지 상태를 직관적으로 확인할 수 있도록, 측정을 멈추면 오버레이 화면의 경과 시간 옆에 ` ⏸` 기호가 붙고 글자 색상이 회색(`text-slate-400`)으로 흐리게 토글되도록 개편했습니다.

### 2. ⚙️ 경험치 세션 자동 시작 여부 전역 설정 추가
* **`xpAutoStart` 옵션 신설 (기본값: ON):**
  * 기존처럼 앱 실행과 동시에 세션이 자동 시작되는 것을 선호하는 사용자를 위해 기본값은 `true`(자동 시작)로 유지했습니다.
  * 수동 제어를 원할 경우, 경험치 HUD 창의 `표시 설정` 영역에서 **경험치 세션 자동 시작** 체크박스를 해제하면 대기 상태로 앱이 켜집니다.

### 3. ⌨️ 경험치 세션 시작/중지 전역 단축키 지원 및 UI 연동
* **기본 전역 단축키 배정:**
  * 사냥터에서 직접 마우스로 HUD 창을 누르지 않아도 키보드로 빠르게 제어할 수 있도록 **`Ctrl + Shift + Z`** 전역 단축키를 새롭게 할당했습니다. (기존 경험치 초기화 단축키 `Ctrl+Shift+X` 옆에 인접하게 구성)
* **단축키 커스터마이징 UI 연동:**
  * 설정 화면(`settings.html`)의 단축키 탭에 **경험치 세션 시작/중지** 항목을 추가하여, 사용자가 원하는 키 조합으로 직접 입력해 바꾸거나 되돌릴 수 있게 구현했습니다.

---

## 📥 설치 및 적용 방법

1. 본 릴리즈 페이지 최하단의 **Assets** 섹션에서 `twOverlay-Setup-1.19.4.exe` 파일을 클릭하여 다운로드합니다.
2. 다운로드 완료 후 실행하시면 자동으로 기존 오버레이가 최신 v1.19.4 사양으로 덮어쓰기 업데이트됩니다.
3. 최초 실행 후 단축키 작동 및 HUD 창 내의 수동 제어 버튼이 오차 없이 안정적으로 적용됩니다.
