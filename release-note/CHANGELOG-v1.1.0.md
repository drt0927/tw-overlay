# 📋 TW-Overlay Release Note - v1.1.0

## 🚀 주요 변경 사항

### Added (신규 기능)
*   **🧪 지능형 버프 매니저 & 계산기**: 1000px 대화면 기반의 버프 백과사전입니다. 선택한 도핑 아이템의 총 대미지, 능력치, 경험치, 레어 확률 등을 실시간으로 합산해 보여줍니다.
*   **🧩 도핑 프리셋 시스템**: 9종의 '필수 기본 세트'를 원클릭으로 적용하거나, 본인만의 커스텀 도핑 조합을 저장하고 관리할 수 있습니다.
*   **📖 약어 검색**: 게임 내에서 사용하는 약어 사전이 추가되어 검색 기능을 통해 풀네임과 설명을 확인할 수 있습니다.
*   **📢 건의 및 버그 제보**: '환경설정 > 앱 정보' 화면에 개발자에게 직접 피드백을 보낼 수 있는 구글 폼 링크를 추가했습니다.

### Changed (변경 및 개선 사항)
*   **🎨 UI 가독성 및 공간 최적화**: 고해상도 환경에 맞춰 전체 폰트 크기를 상향 조정하고, 버프 창을 1000px 너비의 2컬럼 레이아웃으로 개편했습니다.
*   **🖱 드래그 편의성 개선**: 창의 헤더 영역 전체를 드래그 핸들로 지정하고 시각적 가이드를 추가하여 창 이동이 훨씬 수월해졌습니다.
*   **⚡ 코드 최적화**: 윈도우 동기화 로직을 개선하고 불필요한 설정값 저장 로직(MonitorZone 위치 등)을 제거하여 효율을 높였습니다.

### Fixed (버그 수정)
*   **⚖️ 장판 감지 화면 이동 오류**: 게임창 이동 시 장판 감지(Monitor Zone) 화면이 간헐적으로 따라오지 않던 문제를 수정했습니다.
*   **🔔 갤러리 모니터 알림 오류**: 새 글 알림이 발생했음에도 사이드바 아이콘에 레드닷(알림 표시)이 나타나지 않던 문제를 해결했습니다.
*   **💾 데이터 유지 보수**: 창을 닫았다가 다시 열어도 마지막에 선택했던 버프 조합과 활성화된 프리셋이 그대로 복구되도록 수정했습니다.
