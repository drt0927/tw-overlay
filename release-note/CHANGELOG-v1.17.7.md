# TW-Overlay v1.17.7 릴리즈 노트 (2026.06.06)

이번 업데이트는 채팅 오버레이 창 초기 로드(Replay) 과정에서 개별 카테고리 탭(외치기, 클럽 등)의 데이터를 채우기 위해 과거 시간대로 거슬러 올라가 수집한 대화들이 통합(Basic) 탭의 정렬 순서를 깨뜨리는 버그를 수정하였습니다.

---

## 🐛 Fixed (버그 및 예외 처리)

- **채팅 오버레이 초기 로드 시 탭 간 데이터 격리**:
  * 기존에는 모든 탭의 대화 데이터를 단일 임시 배열에 합쳐 수집한 뒤 각 탭에 주입하는 방식을 취하고 있어, 드물게 발생하는 외치기나 클럽 채팅 기록을 채우기 위해 과거 시간대까지 거슬러 올라가 수집한 로그가 통합(Basic) 탭으로 유입되어 정렬 순서를 오염시키는 문제가 있었습니다.
  * 이를 개선하여 수집 배열을 탭 카테고리별로 완전히 격리하고, 주입 시에도 지정된 대상 탭에만 입력되도록 수정하여 탭 간 데이터 간섭 현상을 완벽히 수정했습니다.

---

## 📦 설치 및 업데이트 방법

1. **설치 파일 다운로드**:
   * 본 릴리즈 페이지 최하단의 **Assets** 섹션에서 `twOverlay-Setup-1.17.7.exe` 파일을 클릭하여 다운로드합니다.
2. **프로그램 실행 및 설치**:
   * 다운로드한 `.exe` 파일을 실행하면 자동으로 설치 및 업데이트가 진행됩니다.
   * *참고: Windows SmartScreen 경고가 뜰 경우, `추가 정보` 클릭 후 `실행`을 누르면 됩니다.*
