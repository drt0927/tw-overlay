# 📦 TW-Overlay Release Note (v2.0.2)

## 📅 릴리즈 날짜: 2026-07-01

---

이번 업데이트는 채팅 오버레이 창을 위로 스크롤하여 과거 대화 목록을 불러올 때 에타 레벨 정보가 누락되어 뱃지가 노출되지 않던 버그를 바로잡은 **v2.0.2 마이너 업데이트**입니다.

### 🔧 Fixed (버그 수정 및 개선)
* **과거 채팅 이력 로드 시 에타 레벨 유실 버그 수정:**
  * 채팅 오버레이 창 스크롤을 위로 올려 과거 이력(인피니티 스크롤)을 불러올 때, 캐릭터의 에타 레벨(level) 정보가 매핑되지 않아 UI에 뱃지가 출력되지 않던 현상을 해결했습니다.
  * 파일로부터 과거 데이터를 로드하는 메커니즘 내부에서 발신인 닉네임과 유저 서버 설정을 기반으로 에타 랭킹 캐시 정보(`etaCacheManager.getRankInfo`)를 매핑하여 `level`과 `characterCode` 필드를 온전하게 포함해 반환하도록 전면 수정하였습니다.
  * 이제 실시간 채팅뿐만 아니라 스크롤로 로드된 과거 채팅 기록에서도 다른 유저들의 에타 레벨 뱃지를 누락 없이 정상적으로 확인하실 수 있습니다.

---

## 📥 설치 및 적용 방법

**방법 1: 앱 내 설정 메뉴를 통한 업데이트 (권장)**
1. 앱 내부에서 **[설정]** 메뉴를 엽니다.
2. **[앱 정보]** 탭으로 이동한 뒤 **[업데이트 확인]** 버튼을 클릭하여 수동 업데이트를 진행합니다.

**방법 2: 직접 다운로드하여 업데이트**
1. [Releases](https://github.com/drt0927/tw-overlay/releases) 페이지에서 최신 버전의 `twOverlay-Setup-2.0.2.exe` 파일을 다운로드합니다.
2. 기존 앱을 완전히 종료하신 후, 다운로드한 `.exe` 설치 파일을 실행하여 설치를 진행합니다.
