# [Mandatory Update] TW-Overlay v2.0.3 릴리즈 노트 (2026.07.02)

이번 업데이트는 경험치 HUD(game-overlay) 및 채팅 오버레이의 Z-Order 안정성을 개선한 **패치 v2.0.3 업데이트**입니다.

> [!IMPORTANT]
> 본 업데이트는 오버레이 표시 안정성과 관련된 **필수(강제) 업데이트**입니다. 앱 실행 시 자동으로 최신 v2.0.3으로 업데이트 프로세스가 진행됩니다.

---

## 🚀 주요 변경 사항

### 🐛 Fixed (버그 수정 사항)

* **다른 TW-Overlay 창 클릭 시 game-overlay·채팅 오버레이가 게임 창 뒤로 사라지는 버그 수정:**
  * 경험치 HUD(game-overlay)나 채팅 오버레이가 게임 창 위에 표시된 상태에서, 숙제 체크리스트 등 다른 TW-Overlay 창을 클릭하면 오버레이들이 게임 창 뒤로 사라지던 현상을 해결했습니다.
  * 다른 창으로 포커스가 이동할 때 게임 창의 Z-Order가 재배치되면서 game-overlay가 함께 이동하지 않아 발생한 문제로, 포커스 이동 시 항상 Z-Order를 강제 재정렬하도록 수정했습니다.
  * game-overlay가 항상 게임 창 바로 위, 다른 TW-Overlay 창들의 아래에 안정적으로 배치됩니다.
  * 다시 게임 창을 클릭하지 않아도 오버레이가 정상적으로 유지됩니다.

---

## 📥 설치 및 적용 방법

**방법 1: 앱 내 설정 메뉴를 통한 업데이트 (권장)**
1. 앱 내부에서 **[설정]** 메뉴를 엽니다.
2. **[앱 정보]** 탭으로 이동한 뒤 **[업데이트 확인]** 버튼을 클릭하여 수동 업데이트를 진행합니다.

**방법 2: 직접 다운로드하여 업데이트**
1. [Releases](https://github.com/drt0927/tw-overlay/releases) 페이지에서 최신 버전의 `twOverlay-Setup-2.0.3.exe` 파일을 다운로드합니다.
2. 기존 앱을 완전히 종료하신 후, 다운로드한 `.exe` 설치 파일을 실행하여 설치를 진행합니다.
