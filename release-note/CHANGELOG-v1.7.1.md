# TW-Overlay 릴리즈 노트 (v1.7.1)

본 버전은 컨텐츠 데이터 오류 수정 및 안정성 향상을 위한 패치 업데이트입니다.

## 🛠 수정 사항 (Fixed)
- **Contents Checker:** 숙제 체크 리스트의 일부 항목 데이터 오류를 수정했습니다.
- **Internal Data:** 내부 컨텐츠 참조 데이터의 오타 및 잘못된 연결을 수정했습니다.

## 📦 설치 방법
1. [Releases](https://github.com/drt0927/tw-overlay/releases)에서 `twOverlay-Setup-1.7.1.exe`를 다운로드합니다.
2. 파일을 실행하여 설치를 완료합니다. (기존 버전이 설치되어 있어도 덮어쓰기됩니다.)
3. 앱 내 '환경설정 > 앱 정보'에서 '업데이트 확인'을 통해 자동 업데이트도 가능합니다.
