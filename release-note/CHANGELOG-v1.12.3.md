# TW-Overlay v1.12.3 릴리즈 노트

## 🚀 새로운 기능 (Added)
- **어벤던로드(Abandoned Road) 트래커 추가:**
  - 실시간으로 어벤던로드 입장 비용, 남은 횟수, 마정석 획득 현황을 추적합니다.
  - 마정석 가치 계산기와 자동 연동되며 10회 완료 시 대형 알림과 화면 플래시를 지원합니다.
- **팔색조 언덕(Pitta Hill) 알람 추가:**
  - 팔색조 언덕 보스 에너지를 기반으로 진척도를 추적하여 누락 없이 5회 클리어 알림을 제공합니다.

## 🛠 개선 및 변경 사항 (Changed)
- **Scam Monitor 리팩토링:**
  - 사기꾼 탐지 모듈을 Facade 패턴으로 재설계(모델 관리, 파서, 세션, 서버 분리)하여 성능과 유지보수성을 극대화했습니다.
- **XP Tracker 리팩토링:**
  - 채팅 로그 프로세서에서 경험치 및 리듬 차트 관련 코드를 분리하여 로직 안정성을 확보했습니다.
- **Google Analytics 4 (GA4) 최적화:**
  - 10분 주기 **하트비트(Heartbeat)** 로깅을 추가하고 체류 시간(Engagement Time)을 정밀하게 추적하도록 개선했습니다.
  - 오프라인 전송 무시 처리 및 글로벌 에러(Crash) 자동 수집을 도입했습니다.

## 🐛 버그 수정 (Fixed)
- **프로덕션 환경 로그 최적화:** 빌드된 운영(Prod) 환경에서 불필요하게 반복되는 로그 출력을 필터링하여 리소스 낭비를 차단했습니다.
- **정규식 매칭 유연화:** 띄어쓰기 변경에 취약했던 팔색조 언덕 시스템 메시지(아이템 획득) 파싱 정규식을 개선했습니다.
