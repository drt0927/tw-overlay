# TW-Overlay v1.13.2 Release Notes

## 🛠️ 주요 변경 사항 (Changes)

### 어벤던로드 HUD 개선
- **독립적인 위치 설정 기능 추가**: 어벤던로드 위젯이 기존 XP HUD 위치에 종속되던 로직을 제거하고, 독립적으로 위치를 조정할 수 있도록 변경되었습니다.
- **설정 UI 재구성**:
  - 위치 조정(Left/Bottom) 필드와 적용 버튼을 한 줄로 배치하여 편의성을 높였습니다.
  - '자동 숨김' 옵션, '즉시 표시', '즉시 숨김' 버튼 등을 직관적으로 재배치하였습니다.
- **가독성 향상**:
  - 세션 영역 및 어벤던로드 UI 텍스트의 폰트 크기를 상향 조정하여 시인성을 개선했습니다.
  - 세션 영역 내의 중복 버튼(즉시 숨김)을 제거하여 깔끔하게 정리했습니다.

## 🐛 버그 수정 및 안정화 (Bug Fixes & Stabilization)
- 어벤던로드 세션 활성화 여부에 관계없이 설정 창 내의 기능들이 정상 동작하도록 분리 및 최적화되었습니다.
