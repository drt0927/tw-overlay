# 📋 TW-Overlay 릴리즈 노트 (v1.10.1)

본 버전은 v1.10.0에서 발견된 환경 설정 오류를 수정하고, 필드보스 알림 화면의 사용성을 개선한 마이너 업데이트입니다.

## 🚀 Added (신규 기능)
- (v1.10.0의 주요 기능인 모험 일지, 마정석 계산기, 백업 시스템이 안정화되었습니다.)

## 🔄 Changed (변경 및 개선 사항)
### 1. 필드보스 알림 설정 UI 개선
- **가독성 향상:** 보스 리스트 상단에 여백을 추가하고, 시간 배지의 글씨 크기를 키워 시인성을 높였습니다.
- **테마 일치:** 볼륨 조절 슬라이더 핸들의 색상을 보라색에서 테마 컬러인 빨간색으로 변경했습니다.
- **기능 제한:** 아직 오지 않은 미래의 보스 시간은 클릭하여 처치 기록을 남길 수 없도록 변경하여 데이터 정확도를 높였습니다.

## 🐞 Fixed (버그 수정)
### 1. 환경 설정 데이터 바인딩 오류 수정
- **비동기 UI 대응:** 사이드바 메뉴 관리 목록을 외부에서 불러올 때, 설정 데이터가 먼저 도착하면 체크박스 상태가 반영되지 않던 문제를 해결했습니다. 이제 모든 메뉴의 표시/숨김 상태가 정확하게 표시됩니다.

---
**Note:** v1.10.0의 대규모 업데이트 이후 안정성을 강화한 버전입니다. 최신 버전 사용을 권장합니다.
