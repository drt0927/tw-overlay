# TW-Overlay v1.18.3 릴리즈 노트 (2026.06.17)

이번 업데이트는 **계수계산기 DEX(명중) 정보 추가 및 칭호 입력란 개선**과 **게임 오버레이 경험치 HUD에 경과 시간 표시 기능**이 추가되었습니다.

## 🛠️ 주요 변경 사항 (CHANGELOG)

### Added (신규 기능 추가)
* **게임 오버레이 경험치 HUD 경과 시간 표시**: 
  - 게임 내 오버레이에 노출되는 경험치 HUD 하단에 세션의 실시간 `경과 시간`을 표시하여 1초 단위로 자동 갱신해 주도록 보강했습니다.
  - 앱 로드 및 세션 리셋 시에도 세션 시작 시점(`startTime`)을 정확히 동기화하도록 연동했습니다.

### Changed (변경 및 개선 사항)
* **계수계산기 기본 DEX(명중) 스탯 연동**:
  - 주스탯, 부스탯 열 뒤에 `기본 DEX` 열을 새로 추가하고, 장비 선택 시 데이터베이스에서 명중 스탯(`item.hit`) 정보를 실시간으로 읽어와 프리뷰 칸에 렌더링되도록 개선했습니다.
* **칭호(Title) 부스탯 및 DEX 스탯 일치**:
  - 기존 칭호 입력란에 부스탯 및 DEX 입력 필드를 더해 기존 엑셀 계수 계산기의 수식과 완벽히 일치하도록 수식을 조정했습니다.
  - 사용자가 입력한 칭호의 부스탯 및 DEX 값이 프로필 저장 및 복구(저장/불러오기) 시 정상적으로 동기화되도록 보완했습니다.

---

## 📥 설치 방법

1. 본 릴리즈 페이지 최하단의 **Assets** 섹션에서 `twOverlay-Setup-1.18.3.exe` 파일을 클릭하여 다운로드합니다.
2. 기존에 실행 중이던 `tw-overlay` 앱이 있다면 반드시 완전히 종료한 뒤 설치 파일을 실행합니다.
3. 화면의 안내에 따라 설치를 완료하면 새로운 기능과 패치가 적용됩니다.
