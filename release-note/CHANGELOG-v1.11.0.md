# [Update] v1.11.0 - 실시간 채팅 로그 엔진 및 경험치 HUD 도입

이번 v1.11.0 업데이트에서는 테일즈위버의 채팅 로그를 실시간으로 분석하여 인게임 정보와 편의 기능을 제공하는 **'채팅 로그 감시 시스템'**이 새롭게 도입되었습니다. 이제 게임 창을 가리지 않고도 획득 경험치와 외치기 기록을 스마트하게 관리할 수 있습니다.

---

## 🌟 주요 신규 기능

### 1. 실시간 경험치 HUD (XP/EPM Dashboard)
- **실시간 데이터 수집**: 테일즈위버 채팅 로그를 분석하여 현재 세션의 총 획득 경험치와 **분당 경험치(EPM)**를 실시간으로 계산합니다.
- **게임 오버레이 전용 UI**: 게임 화면 위에 투명하게 떠 있는 HUD를 통해 사냥 효율을 즉시 확인할 수 있습니다.
- **사용자 맞춤 배치**: 환경 설정에서 HUD의 위치(좌측/하단 여백)를 픽셀 단위로 직접 조정할 수 있으며, '위치 즉시 적용' 기능을 통해 설정창을 닫지 않고도 최적의 위치를 잡을 수 있습니다.

### 2. 외치기 히스토리 뷰어 (Shout History)
- **스마트 수집**: 단순 외치기뿐만 아니라, 유저가 직접 발송한 외치기(캐릭터명 접미사 포함)만 선별하여 수집합니다. (시스템 자동 외치기 필터링)
- **실시간 라이브 업데이트**: 외치기가 발생하는 즉시 히스토리 창이 갱신되어 최신 거래/대화 흐름을 놓치지 않습니다.
- **Chrome Highlight API 검색**: 최신 브라우저 기술을 이용한 정교한 검색 기능을 지원합니다. 검색어가 포함된 닉네임과 메시지가 시각적으로 강조됩니다.
- **원클릭 닉네임 복사**: 목록에서 외치기를 클릭하기만 하면 해당 유저의 닉네임이 클립보드에 복사됩니다. 인게임에서 귓속말을 보내기가 훨씬 수월해집니다.

### 3. 모험 일지(Diary) 자동 기록 연동
- **수익 자동 집계**: 채팅 로그에서 감지된 SEED 획득 기록이 모험 일지의 '수익 통계' 섹션에 자동으로 누적됩니다.
- **아이템 획득 기록**: 설정한 키워드에 맞는 아이템 획득 시, 알림과 함께 모험 일지의 타임라인에 자동으로 기록됩니다.

---

## 🎨 UI/UX 개선

- **환경 설정 탭 개편**: '채팅 로그 설정' 섹션이 신설되어 로그 경로, 키워드, HUD 위치, 알림 사운드 등을 한곳에서 관리할 수 있습니다.
- **디자인 일관성 강화**: 외치기 히스토리 창에 `Panel Glass` 효과와 시그니처 퍼플 테마를 적용하여 다른 창들과 완벽한 시각적 통일성을 유지합니다.
- **가이드 이미지 추가**: 채팅 로그 분석을 위해 필요한 인게임 설정 방법을 확인하기 쉽도록 가이드 이미지를 설정창에 배치했습니다.

---

## 🔧 성능 및 안정성 최적화

- **로그 스트리밍 엔진**: `tail` 기반의 바이너리 스트리밍 방식을 채택하여 로그 파일이 커져도 메모리 점유율을 최소화하고 CPU 부하를 방지했습니다.
- **외치기 파싱 개선**: 외치기 메시지 내에 여러 개의 대괄호(`[]`)가 포함된 경우, 가장 마지막에 위치한 대괄호를 실제 캐릭터명으로 인식하도록 로직을 정교화했습니다.
- **Z-Order 관리 최적화**: 게임이나 앱이 포커스 상태가 아닐 때도 오버레이 창이 타 프로그램(브라우저 등)을 가리던 현상을 수정했습니다. 이제 다른 앱 사용 시 오버레이 창이 정상적으로 뒤로 숨습니다.
- **자정 롤오버 대응**: 날짜가 바뀌어 로그 파일명이 변경되어도 시스템이 자동으로 새 파일을 찾아 연결을 유지합니다.
- **Z-Order 관리 안정화**: 게임 오버레이 창들이 테일즈위버 창 위에서 뒤로 숨거나 깜빡이지 않도록 윈도우 스택 관리 로직을 보강했습니다.
- **DB 최적화**: 외치기 히스토리는 최근 24시간 분량만 유지하며 자동으로 오래된 데이터를 삭제하여 데이터베이스 크기를 일정하게 유지합니다.
