# 📦 TW-Overlay Release Note (v1.12.2)

## 📅 릴리즈 날짜: 2026-05-02

---

### ✨ Added (신규 기능)

- **경험의 정수 버프 미감지 알림**: 사냥 중 자동 교환 버프가 꺼진 상태를 실시간으로 감지하여 경고합니다.
  - 마지막 교환(-100억 XP) 이후 110억 XP가 누적되어도 교환이 감지되지 않으면 게임 오버레이 중앙에 빨간 경고 알람과 사운드를 출력합니다.
  - 10억 버퍼: 교환 로그 파일 기록 지연을 흡수하기 위한 오차 허용치.
  - 알람 발생 후 100억만 차감하여 잉여분을 다음 사이클로 이월합니다. (연속 미교환 시 매 100억마다 알람)
  - XP HUD 설정 패널에서 활성화 토글 / 사운드 파일 선택 / 볼륨 조절 지원.

- **경험의 정수 세션 교환 횟수**: XP HUD 정수 카드에 이번 세션 동안 실제 교환된 정수 개수를 실시간으로 표시합니다.
  - -100억 XP 감소 로그가 감지될 때마다 카운트 +1.
  - 세션 초기화 시 0으로 리셋.

- **경험의 정수 프로그레스바 개선**: 마지막 교환 이후 누적된 XP를 기준으로 0→100% 표시하도록 변경.
  - 기존: 세션 총 XP % 100억 (세션 시작 위치와 무관)
  - 변경: 교환 감지 시점마다 0%로 리셋하여 다음 교환까지의 진행도를 정확히 표시.

### 🔧 Changed (변경 및 개선)

- **코드 정리**: `chatLogProcessor.ts` 내 미사용 필드(`_xpHistory`) 및 미사용 파라미터(`sendNotification urgency`) 제거.
- **프로그레스바 수치 캡**: 정수 프로그레스바가 100%를 초과하지 않도록 `Math.min` 처리 추가.
