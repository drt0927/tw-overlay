# TW-Overlay v1.14.0 릴리즈 노트 (2026.05.25)

이번 업데이트는 지정 단어 알림 및 전후 대화 이력 저장, 심연의 제2사도 반사 경보 오버레이, 버프 HUD 단축키 신설 및 숙제 체크리스트 오류 정정을 반영한 Minor 배포입니다.

---

## 🆕 Added (새로운 기능)

- **💬 지정 단어 알림 및 10분 대화 히스토리 저장 기능**:
  - 사냥이나 자리 비움(잠수) 상태일 때 나를 호출하는 닉네임이나 특정 아이템의 매매 단어 감지 시 알림음과 함께 즉각 경보를 띄웁니다.
  - 단어가 감지된 시점을 기준으로 **이전 5분 + 이후 5분 (총 10분)** 간 나누었던 일반/클럽 채팅 내용을 SQLite DB에 통째로 백업 저장해 주는 히스토리 뷰어를 탑재했습니다.
  - 디스크 공간 최적화를 위해 저장된 대화 데이터는 24시간이 경과하면 백그라운드 스케줄러를 통해 안전하게 자동 영구 삭제됩니다.
  - 전용 창(`word-alarm.html`)을 통해 독립적인 사용 ON/OFF 토글, 키워드 뱃지 관리, 개별 알림음 선택 및 슬라이더 볼륨 제어, 대화 맥락(Context)을 아코디언 방식으로 펼쳐보는 편의 기능을 제공합니다.
- **⚔️ 심연의 제2사도 반사 기믹 알림**:
  - 보스 "심연의 제2사도"가 반사 패턴 대사(`절제와 균형의 중심에서 빗나간 힘은 칼날이 되어 돌아오지.`)를 말할 때, 게임 화면 오버레이에 붉은색 경고창(**"공격 반사 패턴! 경고!"**)을 표출합니다.
  - **6.5초 후** 자동으로 초록색 상태창(**"공격 반사 패턴 종료!"**)으로 문구를 변경하여 즉시 딜링 복귀 타이밍을 알려주며, 2초간 안전 상태를 유지한 뒤 깔끔하게 페이드아웃됩니다. (기본값 미사용이며 설정의 채팅로그 탭에서 사용으로 활성화하셔야 작동합니다.)
- **⏱️ 버프 타이머 HUD 보이기/숨기기 단축키 추가**:
  - 언제든 버프 HUD 단추의 보임 여부를 제어할 수 있는 전용 전역 단축키(`Ctrl + Shift + B` 또는 `Cmd + Shift + B`)를 신설하여 사냥 중 화면 가림을 신속히 방어할 수 있도록 하였습니다.

---

## 🛠️ Fixed (버그 수정 및 개선)

- **📅 숙제 체크리스트(Contents Checker) 시스템 및 파서 일원화**:
  - **고대 렐릭의 성소 (신조/키시니크) 합병**: 두 던전이 횟수를 공유하고 택일하여 진행하는 방식을 반영하여 `"고대 렐릭의 성소 (신조/키시니크)"` 단일 숙제로 단순화하였으며, 번잡했던 아이템 루팅 감지 및 비교 로직을 삭제했습니다. 기존 각 캐릭터의 신조/키시니크 횟수 데이터는 손실 없이 통합 횟수로 자동 합산 마이그레이션(최대 7회 한도) 및 DB 정리가 수행됩니다.
  - **오를리 방어전 수동 체크 전환**: 실제 게임 내 완료 시점에 완료 로그가 출력되지 않아 감지가 불가능한 문제를 수정하여, 자동 감지 대상에서 완전히 제외하고 수동 체크 방식으로 통일했습니다.
  - **이클립스 (로카고스) 명칭 교정**: 게임 내 6보스 중 하나인 '로카고스'가 '셀피나'로 오기입되어 있던 숙제 템플릿 정보를 교정하였습니다. 기존 사용자가 쌓아 두었던 셀피나 완료 이력은 SQLite DB 스키마 마이그레이션을 통해 로카고스 이력으로 안전하게 100% 승계 인계됩니다.
  - **코어 마스터 및 아페티리아 파서 최적화**: 머큐리얼 계열과 어비스 심층 계열의 표기 형식 분기를 통합 분석하여 로마자(심층Ⅰ,Ⅱ,Ⅲ) 정규화 및 감지 정밀도를 개선하고 아페티리아 주간 횟수 기입형 감지 연동을 완수했습니다.
- **⏱️ 맵 이동 시 버프 타이머 초기화 방지**:
  - 맵을 이동할 때마다 인게임 버프 목록이 다시 채팅창에 재출력되어 켜져 있던 모든 버프 시간이 무한 초기화되던 버그를 해결했습니다. 타이머가 작동하고 있는 동안에는 중복 버프 감지 신호를 차단하고, 완전히 만료되어 소멸된 이후에만 신규 복용 감지를 허용하도록 매니저 로직을 안전화했습니다.
