# [Mandatory Update] TW-Overlay v2.0.1 릴리즈 노트 (2026.07.01)

이번 업데이트는 v2.0.0 메이저 업데이트 이후 발견된 일부 이스터에그 연출의 화면 가려짐 및 오작동 현상을 빠르게 바로잡은 **v2.0.1 긴급 핫픽스 릴리즈**입니다.

> [!IMPORTANT]
> 본 업데이트는 패키지 구조 및 오버레이 안정성이 개편된 **필수(강제) 업데이트**입니다. 앱 실행 시 자동으로 최신 v2.0.1로 업데이트 프로세스가 진행됩니다.

---

## 🚀 주요 변경 사항

### 1. ⚙️ Changed & Fixed (변경 및 개선된 사항)
* **이스터에그 화면 가려짐 현상 수정:**
  * 게임을 실행하지 않은 상태 또는 윈도우 데스크톱 화면에서 채팅 오버레이 창을 포커싱하여 닉네임을 클릭했을 때, 투명 오버레이 레이어가 타 프로그램 창 뒤로 깔리는 현상을 수정했습니다.
  * 폭죽 및 젤리삐 비 이스터에그가 동작하는 연출 시간(각각 5.5초, 6.5초) 동안만 투명 오버레이 레이어가 강제로 최상단(`setAlwaysOnTop`)으로 노출되도록 개선하여, 다른 화면에 가려지지 않고 온전히 터지도록 보증합니다. 연출 종료 후에는 다시 원래의 샌드위치 Z-Order로 안전하게 원복됩니다.
* **오버레이 비동기 로딩 레이스 컨디션 해결:**
  * 평소에 게임 오버레이 위젯을 띄워두지 않아 백그라운드 상에서 오버레이 창이 비활성화(Destroy)된 상태일 때, 첫 닉네임 클릭 시 창이 로딩되는 도중 이벤트가 발송되어 이스터에그가 무시되던 첫 클릭 씹힘 버그를 핫픽스했습니다.
  * 창이 새로 생성되는 타이밍에는 HTML 및 자바스크립트 로딩 완료(`did-finish-load`) 시점에 동기화하여 이벤트를 안정적으로 발송하게 변경했습니다.

---

## 📥 설치 및 적용 방법

1. 앱을 완전히 종료 후 재시작하시면 자동으로 최신 v2.0.1 사양으로 덮어쓰기 업데이트가 진행됩니다.
2. 만약 자동 업데이트가 진행되지 않는 경우, 릴리즈 페이지에서 `twOverlay-Setup-2.0.1.exe` 파일을 클릭하여 수동 업데이트하실 수 있습니다.
