# TW-Overlay v1.18.9 릴리즈 노트 (2026.06.20)

이번 업데이트는 **챗로그 실시간 인식 재연결 기능, 11개 챗로그 의존 기능 실시간 가이드 배너 탑재, 브라우저 오버레이 드래그 UX 개선**이 적용되었습니다.

## 🛠️ 주요 변경 사항 (CHANGELOG)

### Added (신규 기능 추가)
* **챗로그 경로 설정 즉시 인식 및 재연결**:
  - 설정창의 챗로그 경로 우측에 **[인식 시작]** 버튼이 추가되었습니다. 이제 경로 변경 후 앱을 재기동하지 않아도 버튼을 눌러 즉시 챗로그 탐색을 재시작합니다.
* **11개 챗로그 의존 기능 가이드 배너 및 실시간 연동**:
  - 챗로그 실시간 감지가 필요한 11가지 화면에서 경로가 올바르지 않은 경우 최상단에 주입형 경고 배너를 띄웁니다.
  - 경로 설정 후 인식이 정상적으로 시작되면 켜져 있던 모든 창들의 가이드 배너가 실시간으로 즉시 제거되어 앱 재기동 없이 편리한 사용을 지원합니다.
  - 가이드 배너의 `[설정으로 이동]`을 누르면 환경 설정의 `chatlog` 탭이 즉시 켜집니다.

### Changed / Fixed (변경 및 성능 개선)
* **브라우저 오버레이 드래그 핸들 전체 툴바 확장**:
  - 좁은 핸들을 잡고 드래그해야 했던 불편함을 개선하여, 상단 툴바 전체 영역을 드래그하여 훨씬 편리하게 창을 이동할 수 있습니다. 
  - 좌측에 지구본 아이콘과 "브라우저 오버레이" 타이틀바 레이아웃을 완성하여 UX 미학을 강화했습니다.
* **채팅 오버레이 가이드 배너 시인성 개선**:
  - 오버레이 뒤쪽으로 지나가는 채팅 글씨들과 가이드 배너 텍스트가 겹치더라도 가독성이 떨어지지 않도록, 배경 투명도를 `bg-amber-500/10` (투명도 10%)에서 `bg-amber-950/95` (투명도 95%의 짙은 어두운 톤)로 조정해 한결 불투명하게 개선했습니다.
* **경험치 세션 및 어벤던로드 리셋 로직 분리**:
  - 경험치 HUD를 초기화할 때 어벤던로드 마정석 수집 현황까지 함께 초기화되던 버그를 수정하여 완전히 분리했습니다.
  - 마정석 계산기 내부에 별도의 **[현황 초기화]** 버튼을 신설하여 어벤던로드 현황만 원할 때 수동으로 초기화할 수 있습니다.

## 📥 설치 방법

1. 본 릴리즈 페이지 최하단의 **Assets** 섹션에서 `twOverlay-Setup-1.18.9.exe` 파일을 클릭하여 다운로드합니다.
2. 기존에 실행 중이던 `tw-overlay` 앱이 있다면 반드시 완전히 종료한 뒤 설치 파일을 실행합니다.
3. 화면의 안내에 따라 설치를 완료하면 새로운 기능과 패치가 적용됩니다.
