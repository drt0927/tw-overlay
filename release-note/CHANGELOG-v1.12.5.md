# 📦 TW-Overlay Release Note (v1.12.5)

## 📅 릴리즈 일자: 2026-05-12

이번 버전에서는 어밴던로드 수익 집계의 정확성을 높이고, 오버레이 사용 중 게임으로의 포커스 전환을 더욱 매끄럽게 개선했습니다.

### 🛠 Fixed (버그 수정)
- **어밴던로드 마정석 중복 집계 방지**: 타 사용자의 마정석 획득 알림("누군가...")이 본인의 수익 통계에 합산되는 문제를 해결했습니다. 이제 본인이 획득한 마정석만 정확히 기록됩니다.

### ⚡ Improved (기능 개선)
- **포커스 자동 복구 로직 강화**: 설정 창, 갤러리 창 등 독립된 오버레이 창을 닫을 때 테일즈위버 게임 화면으로 즉시 포커스를 돌려주도록 개선되었습니다.
- **IPC 통신 안정화**: 렌더러 프로세스에서 게임 포커스 복구를 요청할 수 있는 인터페이스를 추가하여 다양한 상황에서 포커스 유실을 방지합니다.
- **창 관리 성능 최적화**: 오버레이 창의 열림/닫힘 상태 제어를 정교화하여 리소스 관리를 효율화했습니다.
