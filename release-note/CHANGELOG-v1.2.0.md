# 🚀 TW-Overlay Release Note - v1.2.0

## 📅 배포 일자: 2026-02-21

이번 업데이트에서는 테일즈위버 유저들의 필수 편의 기능인 **'필드 보스 알림 시스템'**이 새롭게 추가되었으며, 기존 버프 도우미 기능이 더욱 강력해졌습니다.

---

### 🆕 주요 추가 기능

#### 1. 필드 보스 알림 시스템 (Field Boss Notifier)
*   **실시간 보스 알림**: 주요 필드 보스(골론, 파멸의 기원, 스페르첸드, 골모답, 아칸)의 출현 시간을 감지하여 알림을 제공합니다.
*   **커스텀 알림 설정**:
    *   정각, 1분 전, 5분 전, 10분 전 중 원하는 시점에 알림을 받을 수 있습니다.
    *   7종의 개성 있는 알림 사운드를 보스별로 다르게 설정할 수 있습니다.
*   **지능형 시각화**: 설정창에서 현재 시간을 기준으로 이미 지난 보스 타임은 회색으로, 다가올 타임은 보라색으로 구분하여 가독성을 극대화했습니다.

#### 2. 버프 프리셋 관리 고도화 (Buffs Advanced)
*   **프리셋 수정 및 업데이트**: 기존 프리셋을 불러온 뒤 내용을 수정하고 바로 저장할 수 있는 기능을 추가했습니다.
*   **기본 세트(Standard) 보호**: 개발자 추천 기본 세트 선택 시 개별 수정을 잠금 처리하여 설정 혼동을 방지합니다.
*   **UX 개선**: 프리셋 선택 시 애니메이션 효과와 더욱 명확한 초기화 버튼 로직을 적용했습니다.

---

### 🛠 개선 및 수정 사항
*   **사이드바 인터페이스**: 보스 알림 설정으로 바로 진입할 수 있는 퀵 아이콘을 추가했습니다.
*   **창 관리 최적화**: 보스 설정창도 메인 오버레이와 연동되어 게임 화면에 자석처럼 붙도록 개선되었습니다.
*   **안정성 향상**: 다중 창 환경에서 설정 동기화 로직을 강화했습니다.

---

### 📦 설치 방법
1. 기존 앱을 실행 중이라면 업데이트 알림을 통해 자동으로 다운로드됩니다.
2. 또는 설정 > 앱 정보에서 '업데이트 확인'을 클릭하세요.
