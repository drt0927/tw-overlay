# TW-Overlay v1.18.7 릴리즈 노트 (2026.06.19)

이번 업데이트는 **캐시 경험치 버프 3종(전설의 군고구마, 얼리버드 경험치 부스터, 최상급 에오스의 파편)의 중복 적용 오류 수정 및 설명 최신화**가 적용되었습니다.

## 🛠️ 주요 변경 사항 (CHANGELOG)

### Fixed (오류 수정)
* **캐시 버프 3종의 중복 적용 버그 해결**:
  - `전설의 군고구마`와 `최상급 에오스의 파편`의 버프 그룹을 `none`으로 재조정하여, 게임 내에서 일반 버프(경험의 심장, 일반 에오스의 파편 등)와 함께 중복해서 사용하더라도 HUD 상에서 버프 타이머가 강제로 삭제되지 않도록 수정했습니다.
  - 이로 인해 버프가 만료되기 5초 전에 오버레이 알림(경고음 및 시각 경고)이 정상적으로 작동하지 않던 문제를 완벽히 해결했습니다.

### Changed (변경 및 개선)
* **버프 설명 최신화**:
  - `전설의 군고구마`, `얼리버드 경험치 부스터`, `최상급 에오스의 파편` 등 세 가지 캐시 상품들의 설명을 실제 획득 경로와 성격에 맞춰 '이벤트 지급'에서 **'캐시 상품'**으로 변경 및 업데이트했습니다.

---

## 📥 설치 방법

1. 본 릴리즈 페이지 최하단의 **Assets** 섹션에서 `twOverlay-Setup-1.18.7.exe` 파일을 클릭하여 다운로드합니다.
2. 기존에 실행 중이던 `tw-overlay` 앱이 있다면 반드시 완전히 종료한 뒤 설치 파일을 실행합니다.
3. 화면의 안내에 따라 설치를 완료하면 새로운 기능과 패치가 적용됩니다.
