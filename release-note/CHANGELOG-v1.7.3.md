# TW-Overlay 릴리즈 노트 - v1.7.3

## 📅 배포 일자
2026년 3월 13일

## 🚀 주요 변경 사항

### Contents Checker (숙제 체크 리스트) 고도화
숙제 관리의 편의성을 대폭 개선하기 위해 드래그 앤 드롭 및 하이브리드 섹션 뷰를 도입했습니다.

*   **Added**: 숙제 순서 자유 변경 (드래그 앤 드롭)
    *   관리 모드에서 마우스로 숙제를 직접 끌어다 놓아 순서를 변경할 수 있습니다.
    *   드래그 위치를 시각적으로 보여주는 플레이스홀더(보라색 점선 박스)가 추가되어 위치를 명확히 알 수 있습니다.
*   **Added**: 드래그 시 자동 스크롤 지원
    *   숙제가 많아 화면을 벗어날 경우, 드래그 중 마우스를 리스트 상/하단 경계로 가져가면 자동으로 리스트가 스크롤됩니다.
*   **Changed**: 하이브리드 섹션 뷰 (조회 모드)
    *   '일일 숙제(미완료)', '주간 숙제(미완료)', '완료됨(통합)'의 3단계 섹션으로 구조를 변경했습니다.
    *   아직 해야 할 숙제들만 상단에 노출되어 남은 일감을 한눈에 파악하기 좋아졌습니다.
*   **Added**: 완료 항목 식별 뱃지 및 정렬 규칙
    *   완료된 숙제들에 'DAILY', 'WEEKLY' 뱃지를 추가하여 원래 주기를 쉽게 알 수 있습니다.
    *   완료 섹션 내에서도 '일일 -> 주간' 순서 및 사용자 설정 순서가 유지됩니다.
*   **Improved**: 관리 모드 UI 최적화
    *   관리 모드에서는 완료 여부와 상관없이 주기별(일일/주간)로 항목을 모아서 보여주어 순서 조정이 더 편리해졌습니다.

## 🛠 버그 수정 및 개선
*   드래그 앤 드롭 시 레이아웃이 떨리거나 위치 계산이 어긋나던 문제를 수정하여 부드러운 사용자 경험을 제공합니다.
*   데이터 저장 시점 및 정렬 로직을 동기화하여 앱 재시작 후에도 순서가 정확히 유지됩니다.
