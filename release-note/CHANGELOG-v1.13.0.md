# Release Note - v1.13.0 (2026.05.16)

이번 업데이트에서는 테일즈위버의 핵심 강화 시스템 중 하나인 **'시에나의 기운'을 위한 전용 시뮬레이터가 신규 추가**되었습니다.

## 🚀 주요 신규 기능: 시에나의 기운 시뮬레이터 (New)

### 1. 전용 시뮬레이션 환경 구축
- **증폭 시뮬레이션:** 1단계부터 10단계까지의 증폭 과정과 재료(가루, 혼) 및 재화 소모를 미리 체험할 수 있습니다.
- **능력치 재설정:** 에이라의 망치를 사용한 능력치 재설정 및 잠금 기능을 지원합니다.
- **추가 옵션 재설정:** 환류의 서와 정환의 서를 사용한 추가 옵션 시스템을 완벽하게 시뮬레이션합니다.

### 2. 자동화 및 편의 기능 (Auto Reset)
- **자동 능력치 설정:** 원하는 능력치 종류와 등급이 나올 때까지 자동으로 재설정을 반복하는 기능입니다.
- **자동 추가 옵션 설정:** 특정 슬롯 또는 전체 슬롯에 목표 옵션이 등장할 때까지 자동화가 가능합니다.
- **스마트 안전장치:** 정환의 서 모드에서 이미 다른 슬롯에 존재하는 옵션을 목표로 잡는 등의 불가능한 설정을 사전에 차단합니다.

### 3. 정밀한 데이터 분석 및 리포트
- **공식 확률 기반:** 한국/일본 서버의 최신 공식 데이터를 반영하여 실제 게임과 동일한 확률을 제공합니다.
- **기댓값 리포트:** 목표 단계까지의 예상 비용 및 **평균 시도 횟수**를 실시간으로 계산하여 합리적인 강화 전략을 돕습니다.
- **실시간 히스토리:** 강화 과정을 상세히 기록하며, 가독성을 위해 자동 줄바꿈을 지원합니다.

## 🛠 기타 개선 사항
- **레이아웃 최적화:** 시뮬레이터의 정보 집약도를 높이기 위해 전용 창의 크기를 확장하고 직관적인 3단 패널 구조를 적용했습니다.
- **버그 수정:** 장비 부위 변경 시 발생할 수 있는 데이터 정합성 이슈를 해결했습니다.
