# TW-Overlay v1.18.12 릴리즈 노트 (2026.06.22)

이번 업데이트는 **최초 실행 사용자를 위한 인터랙티브 무테 웰컴 가이드 창 신설, 가이드 내비게이션 추가 및 이미지 찌그러짐 개선, MsgerLog 설명 제거, Dock 모드용 오버레이 알림 토스트 연동 및 백엔드 IPC 최적화**가 적용되었습니다.

## 🛠️ 주요 변경 사항 (CHANGELOG)

### Added (새 기능)
* **독립형 무테 사용법 가이드 창 (`welcome-guide.html`) 신설**:
  - 최초 앱 구동 시 스플래시 화면 종료와 함께 자동으로 팝업되어 사용법을 핵심 요약하여 알려주는 인터랙티브(탭 전환 방식) 웰컴 가이드가 추가되었습니다.
  - 가이드 탭은 '앱 소개', '필수 설정', '전체 창 모드 팁', '단축키 요약' 4가지로 구성하여 첫 기동 시 필요한 정보를 한눈에 파악할 수 있도록 도왔습니다.
* **가이드 내 이전/다음 내비게이션 및 확인 버튼 제어**:
  - 탭 직접 클릭 외에도 양방향으로 순차 이동이 가능한 `[이전]`, `[다음]` 버튼을 푸터에 새롭게 디자인하여 배치했습니다.
  - 첫 번째 페이지에서는 `[이전]` 버튼을 숨기며, 마지막 페이지(단축키 요약)에 도달했을 때만 `[확인]` 버튼이 나타나 가이드를 완료할 수 있도록 제어했습니다.
* **사이드바 환경설정 내 가이드 다시보기 버튼 추가**:
  - '환경설정 ➡️ 일반 설정' 패널 최하단에 `가이드 다시 보기` 버튼을 추가하여 사용자가 언제든 웰컴 가이드를 재호출할 수 있도록 편의성을 넓혔습니다.
* **Dock 모드 전용 읽기 전용 토스트 알림**:
  - 사이드바 위치를 'Dock'으로 지정해 사이드바가 보이지 않는 전체 창모드 사용 환경에서, 화면 우측 하단에 보스 스폰 알림 및 사기 감지, 버프 타이머 5초전 경고 카드를 실시간 팝업해주는 기능이 이식되었습니다.

### Changed (개선 및 변경점)
* **백엔드(메인 프로세스) 알림 전송 조건 최적화**:
  - `game-overlay.html`이 항상 켜져 있으므로, 불필요하게 알림 IPC(`play-sound`, `scam-alert`)를 상시 전송하던 구조에서 백엔드에서 사용자의 설정(`sidebarPosition === 'dock'`) 여부를 사전에 체크하여 필요한 경우에만 전송하도록 전송 분기를 최적화하였습니다.
* **가이드 이미지 비율 왜곡 및 겹침 버그 개선**:
  - 가이드 내에 들어가는 설정 및 Dock 가이드 이미지(`guide_setup.png`, `guide_dock.png`)의 가로세로 비율(990x715)이 깨지지 않고 정상 비율로 크게 노출되도록 `height: auto` 스타일로 수정하였습니다.
  - `guide_dock.png`를 성공적으로 불러왔을 때 그 밑에 있는 폴백 텍스트("전체 창 모드용 Dock 설정...")가 겹쳐 보이던 UI 중복 렌더링 마크업 버그를 해결하였습니다.
* **가이드 내 설명 정제**:
  - 필수 설정 탭에서 귓속말 로그 `MsgerLog`에 관한 설명을 삭제하고, 사냥 효율 및 득템 기록에 가장 중요한 `ChatLog` 폴더 설정 안내에만 집중하도록 텍스트를 간소화했습니다.

### Fixed (버그 수정)
* **버프 타이머 및 사기 감지 알림 동시 다발 시 토스트 고착 버그 수정**:
  - 사냥 시작 등으로 인해 여러 버프 경고가 찰나의 시간(0ms에 준함) 내에 동시에 발생할 때, `Date.now()`로만 생성된 토스트 엘리먼트 ID가 겹쳐서 지워지지 않고 화면에 영원히 남아있던 문제를 고유 난수(Random String)를 ID 뒤에 결합하여 완벽히 정상적으로 순차 소멸하도록 조치하였습니다.
* **모험일지 자동 득템 기록 이미지 매핑 복구**:
  - 모험일지에서 득템 자동 수집 시 획득 아이템의 이미지가 깨지던 문제를 데이터 파일 매핑 보완 및 특수문자 제거 폴백 로직을 통해 복구하였습니다.

## 📥 설치 방법

1. 본 릴리즈 페이지 최하단의 **Assets** 섹션에서 `twOverlay-Setup-1.18.12.exe` 파일을 클릭하여 다운로드합니다.
2. 기존에 실행 중이던 `tw-overlay` 앱이 있다면 반드시 완전히 종료한 뒤 설치 파일을 실행합니다.
3. 화면의 안내에 따라 설치를 완료하면 새로운 기능과 패치가 적용됩니다.
