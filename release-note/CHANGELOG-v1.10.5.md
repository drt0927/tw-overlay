# 📦 Release Note - v1.10.5 (2026-04-02)

이번 버전에서는 게임 화면을 보고 있지 않을 때도 중요한 소식을 놓치지 않도록 **하이브리드 알림 시스템**을 도입하고, 모험 일지의 **데이터 무결성 및 편의성**을 대폭 강화했습니다.

## 🚀 주요 변경 사항

### 1. 🔔 하이브리드 알림 시스템 (Windows Toast)
- 게임 창이 **최소화**되어 있거나 **종료**된 상태일 때, 필드보스 및 커스텀 알림을 Windows 네이티브 알림(Toast)으로 발송합니다.

### 2. 📖 모험 일지 지능화 및 중복 방지
- **자동 날짜 선택:** 모험 일지 창을 열면 번거로운 클릭 없이 자동으로 오늘 날짜가 선택되어 즉시 기록을 확인하거나 작성할 수 있습니다.
- **중복 기록 차단:** 동일한 보스와 스폰 시간에 대해 여러 번 기록 버튼을 눌러도 일지에 중복 생성되지 않도록 DB 레벨에서 완벽하게 차단합니다.
- **UI 피드백:** 이미 기록된 경우 버튼이 '이미 기록됨!' 상태로 표시되어 직관적인 확인이 가능합니다.

### 3. ✨ UI/UX 및 기타 개선
- **사이드바 팝업 보정:** 알림 발생 시 사이드바 너비를 즉시 확장하여 팝업 내용이 짤려 보이는 현상을 수정했습니다.
- **창 높이 확장:** 필드보스 설정 창의 높이를 확장(600px -> 750px)하여 더 많은 정보를 한눈에 볼 수 있게 했습니다.
- **커서 상속 수정:** 윈도우 컨트롤 버튼 내 아이콘 위에서 마우스 커서가 기본형으로 돌아가는 버그를 수정했습니다.

## 🛠 수정 및 개선 내역
- **Added:** `pollingLoop`에 게임 상태 추적 기능 추가 및 외부 노출.
- **Added:** `diaryDb`에 기록 존재 여부 확인 함수(`isActivityLogged`) 추가.
- **Changed:** `bossNotifier`, `customNotifier`에 Windows Noti 발송 로직 통합.
- **Fixed:** CSS 상속 문제로 인한 버튼 내 커스텀 커서 미적용 현상 수정.
- **Fixed:** 비동기 로드 시점 차이로 인한 모험 일지 초기 렌더링 최적화.
