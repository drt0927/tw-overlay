# 📦 릴리즈 노트 v1.13.1 (2026.05.17)

이번 업데이트는 **'시에나의 기운 시뮬레이터' 기능 고도화 및 사용자 편의성 개선**을 중심으로 진행되었습니다.

## 🚀 주요 변경 사항

### ⚡ 시에나의 기운 시뮬레이터 개선
- **즉시 단계 설정 기능 추가**: 비용 소모 없이 시에나를 3단계, 7단계, 10단계로 즉시 강화할 수 있는 기능을 추가했습니다. (빠른 옵션 테스트 지원)
- **탭별 맞춤형 대시보드 및 기댓값**: 증폭, 능력치, 추가 옵션 각 탭에 진입할 때 해당 기능에 특화된 실시간 소모 내역과 기댓값 리포트를 보여줍니다.
- **종합 '통계' 탭 신설**: 모든 강화 공정에서 소모된 전체 재화(SEED, ELSO, 가루, 혼, 망치, 서 등)를 한눈에 확인할 수 있는 페이지를 추가했습니다.
- **추가 옵션 슬롯별 비용 추적**: 각 슬롯(1, 2, 3)에서 소모된 재화 내역을 독립적으로 추적하고 개별적으로 초기화할 수 있도록 개선했습니다.

### ⚙️ 시스템 및 UI/UX 최적화
- **논리적 초기화 시스템**: 환류의 서(전체 슬롯 리셋)와 정환의 서(단일 슬롯 리셋)의 실제 동작 방식에 맞춰 초기화 로직을 분리 및 정립했습니다.
- **UI 시인성 강화**: 대시보드 영역의 폰트 크기를 상향 조정하고, 불필요한 패딩을 제거하여 가독성을 높였습니다.
- **자동 강화 UI 깜박임 해결**: 추가 옵션 탭의 엘소(ELSO) 비용을 고정 너비 만 단위 형식(예: 15만 0000)으로 표시하여 자릿수 변화에 따른 레이아웃 흔들림을 방지했습니다.
- **동적 능력치 잠금 제한**: 현재 개방된 슬롯 수에 맞춰 최대 잠금 가능 개수가 자동으로 조절되도록 수정했습니다. (`현재 랭크 - 1`)

## 🛠 수정 및 보완
- **버그 수정**: 추가 옵션 재설정 시 선택하지 않은 재화가 계산되어 표시되던 렌더링 오류를 수정했습니다.
- **문구 정리**: 주요 강화 재료들의 명칭을 약어에서 풀네임(에이라의 망치, 힌덴의 가루, 장인의 혼)으로 일괄 통일했습니다.
- **SVG 아이콘 렌더링 수정**: 탭 전환 시 SVG 요소의 속성 접근 에러로 인해 대시보드가 갱신되지 않던 문제를 해결했습니다.
