# 🎮 TW-Overlay (TalesWeaver Companion Browser)

테일즈위버(TalesWeaver) 플레이어를 위한 **지능형 다중 창 오버레이 브라우저**입니다. 게임 화면과 연동되는 자석형 위젯과 독립된 브라우저 창을 통해 최상의 플레이 환경을 제공합니다.

## ✨ 주요 기능

*   **✅ 일일/주간 숙제 체크 리스트:** 테일즈위버의 30여종 기본 숙제와 커스텀 항목을 체계적으로 관리하세요. 드래그 앤 드롭 순서 변경, 하이브리드 섹션 뷰, 지능형 자동 초기화와 실시간 검색 기능을 제공합니다.
*   **🧮 진화 재료 비용 계산기:** 장비 및 무기 등급 진화에 필요한 소모 재료 단가를 입력하여 총 소모 시드와 엘소를 직관적으로 파악할 수 있는 시뮬레이터를 제공합니다. 누락되었던 **손목(방패/보조)** 부위 데이터가 추가되었습니다.
*   **🧮 계수 및 명중 계산기:** 테일즈위버 캐릭터의 스탯, 장비, 버프를 종합하여 총 계수와 명중률을 실시간으로 시뮬레이션할 수 있는 전용 계산기입니다.
*   **🧩 지능형 창 관리:** Registry 아키텍처를 통해 모든 오버레이 창이 빠르고 정확하게 게임 화면과 동기화됩니다.
*   **🚀 게임 네트워크 최적화 (Fast Ping):** 네이글 알고리즘(Nagle's Algorithm)을 버튼 하나로 제어하여 게임 응답 속도(핑)를 개선합니다.
*   **⏰ 커스텀 알림 (Custom Alerts):** 매일 특정 시각 또는 매시 특정 분에 맞춰 울리는 반복 알림을 설정하세요. 다중 시점 알림과 전용 사운드 지정 기능을 제공합니다.
*   **⏰ 필드 보스 알림:** 필드 보스 출현 시간을 감지하여 소리 알림을 제공합니다. v1.9.0에서 설정 UI가 전용 창으로 통합되어 더욱 깔끔하게 관리할 수 있습니다.

## 🚀 설치 및 사용 방법

### 1. 설치하기
[Releases](https://github.com/drt0927/tw-overlay/releases) 페이지에서 최신 버전의 `twOverlay-Setup-1.9.1.exe` 파일을 다운로드하여 설치하세요. 기존 사용자는 앱 내 업데이트 기능을 사용할 수 있습니다.

### 2. 실행하기
1. 테일즈위버를 실행합니다.
2. `twOverlay`를 실행하면 자동으로 게임 창 우측에 사이드바가 나타납니다.
3. 사이드바의 각 아이콘을 클릭하여 기능을 조작할 수 있습니다.

## 📝 최신 업데이트 내역 (v1.9.1)
- **데이터 오류 및 오타 수정 (Data Maintenance):**
    - `evolution_data.json` 내 진화 재료 명칭 오타 수정 (예: 가짜 달여왕 군단의 건틀릿 파편 등).
    - 기타 내부 데이터 파일의 텍스트 오타 제거 및 일관성 개선.

## 📝 이전 업데이트 내역 (v1.9.0)
- **신규 기능: 커스텀 알림 (Custom Alerts):**
    - 매일 특정 시각/매시 특정 분 반복 알림 기능 및 전용 관리 창(`custom-alert.html`) 추가.
    - 등록된 알림을 즉시 수정할 수 있는 **편집(Edit)** 기능 포함.
- **UI/UX 최적화:**
    - 필드 보스 알림 설정을 전용 보스 설정창으로 일원화하여 메인 설정창의 복잡도 감소.
    - 개발(DEV) 모드 실행 시 스플래시 화면이 계속 남아있던 버그 수정.

## 📝 이전 업데이트 내역 (v1.8.1)
- **장비 진화 계산기 업데이트:**
    - 누락되었던 **손목(방패)** 및 **손목(보조/브레이슬릿)** 진화 재료 데이터 추가.
    - 진화 데이터 내 오타('군단의의') 수정.
- **분석 시스템 강화:**
    - 앱 사용성 측정을 위한 기초 마련 (v1.9.0에서 표준 방식으로 고도화됨).

## 📝 이전 업데이트 내역 (v1.8.0)
- **신규 기능 (진화 재료 계산기):**
    - 단계별 장비/무기 진화 재료 단가 기록 및 총 비용(시드, 엘소 분할) 계산 툴 추가 도입.
    - 태청금액신단 전용 엘소 구매 모드 전환 등 다양한 계산 옵션 지원.
- **앱 안정기능 추가:**
    - 환경 설정에서 `진화 재료 계산기` 메뉴 표기 설정 및 스타일 일관성 수정.
