# TW-Overlay 릴리즈 노트 (v1.7.2)

본 버전은 컨텐츠 체크 리스트의 데이터 중복 오류 수정 및 안정성 향상을 위한 패치 업데이트입니다.

## 🛠 수정 사항 (Fixed)
- **Contents Checker:** '이클립스 6보스', '보급품 탈환', '별동대 토벌' 항목이 동일한 ID를 공유하여 개별 체크가 되지 않던 버그를 수정했습니다. 이제 각 항목의 완료 상태가 독립적으로 저장됩니다.

## 📦 설치 방법
1. [Releases](https://github.com/drt0927/tw-overlay/releases)에서 `twOverlay-Setup-1.7.2.exe`를 다운로드합니다.
2. 파일을 실행하여 설치를 완료합니다. (기존 버전이 설치되어 있어도 덮어쓰기됩니다.)
3. 앱 내 '환경설정 > 앱 정보'에서 '업데이트 확인'을 통해 자동 업데이트도 가능합니다.
