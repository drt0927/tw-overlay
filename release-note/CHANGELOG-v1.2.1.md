# 🚀 TW-Overlay Release Note - v1.2.1

## 📅 배포 일자: 2026-02-21

v1.2.0 배포 직후, 사용자 편의성을 한층 더 강화한 패치 업데이트를 진행합니다. 환경설정의 구조적 개편과 새로운 사운드 팩이 추가되었습니다.

---

### 🆕 주요 추가 및 개선 사항

#### 1. 환경설정 UI 전면 개편 (Tabs System)
*   사용자 편의를 위해 환경설정창을 **6개의 전문 탭**으로 분리했습니다.
    *   **오버레이 설정**: 기본 URL 및 창 크기.
    *   **갤러리 알림**: 키워드 필터링.
    *   **필드보스 알림**: 알림 시점 및 전용 음량 조절.
    *   **장판 감지 설정**: 민감도 및 전용 음량 조절.
    *   **퀵슬롯 관리**: 사이드바 링크 편집 및 순서 변경.
    *   **앱 정보**: 버전 및 업데이트 확인.

#### 2. 사운드 시스템 고도화
*   **신규 사운드 추가**: 7종의 개성 있는 알림 사운드가 추가되었습니다 (총 15종).
*   **장판 감지 사운드 선택**: 이제 장판 감지 시에도 원하는 사운드를 직접 선택하고 미리 들을 수 있습니다.
*   **음량 개별 조절**: 장판 감지와 필드보스 알림의 음량을 각각 독립적으로 조절할 수 있습니다.
*   **기본 사운드 교체**: 표준 기본 사운드가 `Orb`로 변경되었습니다. (기존 유저 호환성 유지)

#### 3. 내부 최적화 및 공통화
*   **UI 유틸리티 통합**: 여러 창에서 중복 사용되던 로직을 `ui-utils.js`로 공통화하여 앱 실행 속도와 안정성을 개선했습니다.
*   **데이터 외부화**: 사운드 목록을 독립된 데이터 파일(`sounds.json`)로 분리하여 관리 효율성을 높였습니다.

---

### 🛠 수정 사항
*   `boss-settings.html`의 중복된 설정 항목을 제거하여 UI를 간소화했습니다.
*   이미 지난 보스 시간의 시각화 로직을 최적화했습니다.

---

### 📦 설치 방법
1. 기존 앱을 실행 중이라면 업데이트 알림을 통해 자동으로 다운로드됩니다.
2. 또는 설정 > 앱 정보에서 '업데이트 확인'을 클릭하세요.
