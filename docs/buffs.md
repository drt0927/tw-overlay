# 버프 백과 및 계산기 (Buff Encyclopedia)

## 1. 기능 개요 및 목적
테일즈위버에서 사용되는 수많은 버프(도핑) 아이템의 정보를 한눈에 확인하고, 여러 버프를 중첩했을 때의 최종 합계 효과를 계산해주는 도구입니다. 복잡한 버프 중첩 규칙(그룹 제한 등)을 자동으로 적용하여 최적의 도핑 세팅을 지원합니다.

## 2. 주요 UI 구성 요소 설명
- **도핑 프리셋 (Presets):** '기본 도핑 세트' 등 자주 사용하는 버프 조합을 원클릭으로 적용하거나 사용자 정의 프리셋을 저장할 수 있습니다.
- **검색 및 카테고리 필터:** 버프 이름이나 효과로 검색하고, 경험치/레어/능력치 등 속성별로 필터링할 수 있습니다.
- **버프 카드 (Buff Card):** 각 버프의 지속 시간, 세부 효과, 사망/재접속 시 유지 여부를 상세히 표시합니다.
- **합계 요약 패널 (Summary):** 하단에 현재 선택된 모든 버프의 효과 합계(대미지%, 능력치, 경험치% 등)를 실시간으로 합산하여 보여줍니다.

## 3. 세부 기능 및 작동 방식
- **중첩 그룹 관리:** 동일 그룹의 버프(예: 이자벨 류)는 동시에 선택할 수 없도록 자동 비활성화 로직이 작동합니다.
- **프리셋 관리:** 사용자가 직접 구성한 버프 조합을 이름과 함께 저장하고 관리할 수 있습니다.
- **위험 요소 알림:** 선택된 버프 중 '사망 시 삭제' 또는 '재접속 시 삭제' 속성이 포함된 경우 경고 메시지를 표시합니다.
- **효과 자동 합계:** 텍스트 형태의 버프 효과를 정규표현식으로 분석하여 수치화된 합산 결과를 도출합니다.
