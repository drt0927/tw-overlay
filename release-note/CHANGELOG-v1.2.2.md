# 🚀 TW-Overlay Release Note - v1.2.2

## 📅 배포 일자: 2026-02-21

이번 업데이트에서는 설정창에서의 알림음 미리보기 기능을 대폭 개선하여, 사용자가 저장 전에도 정확한 음량과 사운드를 확인할 수 있도록 했습니다.

---

### 🆕 주요 개선 사항

#### 1. 지능형 사운드 미리보기 시스템
*   **실시간 음량 반영**: 이제 환경설정에서 음량 슬라이더를 조절한 직후 '미리보기'를 누르면, 저장하지 않아도 **현재 슬라이더 위치의 음량**으로 소리가 들립니다.
*   **필드보스 미리보기 강화**: 필드보스 설정 탭에 사운드 선택 상자를 추가하여, 원하는 알림음을 골라 실제 음량을 테스트해볼 수 있습니다.

#### 2. UI/UX 정밀 튜닝
*   장판 감지 및 필드보스 음량 조절부의 디자인을 통일하여 시각적 일관성을 높였습니다.
*   미리보기 버튼의 위치와 안내 문구를 보강하여 사용자가 기능을 쉽게 인지할 수 있도록 개선했습니다.

---

### 🛠 수정 사항
*   IPC 통신 레이어에 `volume` 파라미터를 추가하여 메인 프로세스와 렌더러 간의 음량 동기화 문제를 해결했습니다.
*   사이드바(`index.html`)의 사운드 재생 엔진을 최적화했습니다.

---

### 📦 설치 방법
1. 기존 앱을 실행 중이라면 업데이트 알림을 통해 자동으로 다운로드됩니다.
2. 또는 설정 > 앱 정보에서 '업데이트 확인'을 클릭하세요.
