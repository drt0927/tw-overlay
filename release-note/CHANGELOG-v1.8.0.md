# TW-Overlay v1.8.0 릴리즈 노트

**배포일자:** 2026-03-24

## ✨ Added (신규 추가 분)
- **진화 재료 비용 계산기:** 
  - 엔키라부터 이클립스 투구/아뮬렛까지, 각 장비 단계별 진화 재료의 필요 수량과 시세/단가를 실시간으로 계산하는 전용 창 도구 제공.
  - *특화 기능:* '태청금액신단' 재료 전용으로 엘소 구매 활성화 토글을 제공하여 시드(Seed)와 엘소(Elso)로 분할된 총 비용 합산 결과 출력 (23,000 엘소 변환).
- **사용자 이용 빈도 분석(GA4):**
  - 향후 주요 기능 개선 및 유지보수 우선순위를 판단하기 위해 Google Analytics 4 (GA4) 트래킹 시스템 도입. 

## 🔄 Changed (변경 및 개선)
- **UI 및 사용자 경험 통일:** 
  - 진화 재료 계산기 메뉴의 체크박스를 프로젝트 고유 테마(보라색)로 변경.
  - 환경 설정(`settings.html`)의 앱 정보 및 탭 네비게이션 전반과 앱 메인 버전 표기를 1.8.0으로 일괄 갱신.
  - 계산기 내 보유 시드/금액 단위가 커져서 텍스트가 잘리는 현상 방지를 위해 레이아웃 비율 및 그리드 간격 추가 확장.
