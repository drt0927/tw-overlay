# TW-Overlay v1.18.1 릴리즈 노트 (2026.06.16)

이번 업데이트는 **대장간, 골고다, 공허 3종 도전과제**를 실시간으로 추적하는 통합 트래커를 탑재하고, 설정창의 채팅 로그 설정 영역을 **4개의 가로 서브 탭(상단 Sticky 스크롤 고정) 레이아웃으로 전면 개편**했습니다. 또한 **닉네임 색상 커스터마이징 기능**을 복구하고 편의성을 높였습니다.

---

## 🌟 Added (신규 기능)

- **3종 도전과제(대장간, 골고다, 공허) 통합 트래커 및 동적 HUD**:
  - 기존 대장간 전용 도전과제 트래커를 확장하여 **대장간(1,500마리)**, **골고다(1,000마리)**, **공허(800마리)** 3종의 도전과제를 모두 지원하는 통합 트래커를 신설했습니다.
  - 게임 로그 감지를 통해 새로운 도전과제 시작 문구(`[twOverlay] 대장간/골고다/공허 도전과제 시작`)가 감지되면 기존 타이머와 진행 상황을 즉각 초기화하고 새로운 퀘스트 타입으로 20분 제한 시간과 카운트를 덮어씁니다(Overwrite).
  - 도전과제 완료 시 노출되는 중앙 팝업 카드가 퀘스트 타입에 맞춰 Lucide 아이콘(대장간: `hammer`, 골고다: `shield`, 공허: `orbit`), 카드 타이틀, 완료 문구 배지(`몬스터 X마리 처치 완료`)로 실시간 스위칭되며 Lucide 아이콘 라이브러리의 `refreshIcons()`가 연동 작동합니다.

- **도전과제 가이드 클립보드 복사 버튼 구현**:
  - 설정창의 기믹 & 도전과제 서브 탭 내 가이드의 각 시작 문구 옆에 Lucide `copy` 단추를 부착했습니다.
  - 복사 버튼 클릭 시 클립보드로 문구를 복사하고, 1.5초 동안 아이콘이 `check` 모양으로 변경되는 마이크로 피드백 애니메이션을 구현하여 편의성을 극대화했습니다.

- **도전과제 HUD 위치 즉시 확인 (미리보기) 기능 추가**:
  - HUD 위치 설정 인풋 필드 바로 하단에 `위치 즉시 확인 (대장간 테스트 시작)` 버튼을 추가했습니다.
  - 버튼 클릭 시 설정창의 Left 및 Bottom 좌표 값을 인게임 오버레이 창으로 즉각 전송해 위치 설정을 적용함과 동시에, 가상으로 대장간 도전과제 HUD가 노출되도록 테스트 이벤트를 발생시켜 사용자가 수정한 위치를 눈으로 바로 확인할 수 있게 하였습니다. 성공적으로 반영되면 버튼이 1.5초간 녹색 체크 아이콘과 함께 "반영 완료" 상태로 변화합니다.

## 🔧 Improved & Fixed (개선 및 수정)

- **설정 화면 '채팅 로그 설정' 4개 가로 서브 탭 개편**:
  - 채팅 로그 관련 설정을 4개 서브 탭(채팅 로그, 채팅 오버레이, 득템 & 외치기, 기믹 & 도전과제)으로 명확히 분할하고, 스크롤 시 탭바가 상단에 Sticky로 고정되도록 사용성을 보강했습니다 (`position: sticky; top: -40px; z-index: 50;`).
  - 탭바 뒤로 밑에서 올라오는 설정 아이템들이 깨끗하게 가려지며 넘어가도록 설정창 본문 영역과 일치하는 불투명한 배경색(`background: rgba(15, 18, 30, 0.98);`)과 위아래 여백 조정을 결합하여 고급스러운 내비게이션 사용성을 완성했습니다.
  - 탭바 아이콘은 Windows 기본 이모지 대신 일관성 있고 아름다운 **Lucide 아이콘**(`<i data-lucide="..."></i>`)을 세련되게 바인딩했습니다.

- **닉네임 색상 커스터마이징 기능 추가 복구**:
  - `chatOverlayNicknameColorMode`에 맞추어 `same`(채팅 글씨색과 동일) 혹은 `custom`(직접 지정)을 선택하는 드롭다운 셀렉터를 생성했습니다.
  - 직접 지정 모드 시 각 채널별(일반, 귓속말, 팀, 클럽, 외치기) 닉네임 색상을 Pickr 선택기로 조절할 수 있도록 `nicknamePickers`를 새로 구현했습니다.
  - `onConfigData`, `saveSettings`, 그리고 `applyChatOverlaySettingsOnly` 함수에서 닉네임 색상 설정값들이 전송 및 즉시 반영되도록 구현했습니다.

- **채팅 오버레이 서브 탭 최적화**:
  - 사용자 편의를 위해 **"주 사용 캐릭터 서버 설정"** 카드를 채팅 오버레이 탭 내에서 **최상단**으로 이동시켰습니다.

## 📦 설치 및 업데이트 방법

1. **설치 파일 다운로드**:
   - 본 릴리즈 페이지 최하단의 **Assets** 섹션에서 `twOverlay-Setup-1.18.1.exe` 파일을 클릭하여 다운로드합니다.
2. **프로그램 실행 및 설치**:
   - 다운로드한 `.exe` 파일을 실행하면 자동으로 설치 및 업데이트가 진행됩니다.
   - *참고: Windows SmartScreen 경고가 뜰 경우, `추가 정보` 클릭 후 `실행`을 누르면 됩니다.*
