# TW-Overlay v1.18.11 릴리즈 노트 (2026.06.21)

이번 업데이트는 **모험일지 뷰(1400x920) 대폭 확장 및 일자별 득템 스택 뷰 개편, 득템 설정 2컬럼 전송 UI 리뉴얼, 채팅 자동 수집 득템 파싱 및 중복 합산 개선**이 적용되었습니다.

## 🛠️ 주요 변경 사항 (CHANGELOG)

### Added (새 기능)
* **모험일지 화면 확장 및 레이아웃 2단 분리 개편**:
  - 창 해상도를 `1000x850`에서 `1400x920`으로 확장하여 가로 폭과 달력 가독성을 극대화했습니다.
  - 화면을 **좌측 (달력 메인 - `flex-1`)**과 **우측 (선택일 상세 - `w-[400px]`)**의 2단 레이아웃으로 개편하여 정보가 잘림 없이 넓게 보이도록 디자인했습니다.
  - 통계 탭 상단을 3열로 쪼개고 **"나의 모험 골든 타임" 시간대별 활동 분석 카드**를 신설하여 다각도 분석 기능을 제공합니다.
* **달력 내 득템 스택 뷰 및 이미지 아이콘 연동**:
  - 날짜 셀 하단에 그날 얻은 아이템들이 미니 이미지 아이콘과 수량이 적힌 배지 스택 형태로 예쁘게 누적 표시됩니다.
  - 아이템 등급/종류에 따라 적절한 투명 배지 컬러(경험의 정수: 황색, 심장/기운: 적색, 결정체: 자색, 일반: 회색 등)가 자동 매핑됩니다.
  - 개발자 도구(DevTools)에서 수동으로 경험의 정수 버프 경고를 띄울 수 있는 `testEssenceAlert()` 테스트 함수를 추가했습니다.
* **득템 키워드 설정 2컬럼 전송(Transfer List) UI 리뉴얼**:
  - 득템 키워드 설정을 편리하게 할 수 있도록 미등록 / 등록 리스트를 좌우로 분리했습니다.
  - 실시간 검색 필터링과 기본값 복원, 대량 등록 및 해제 기능을 추가해 조작성을 비약적으로 향상시켰습니다.

### Changed (개선 및 변경점)
* **통계 탭 누적 리스트 고정 및 스크롤 개선**:
  - 이번 달 누적 득템 리스트와 누적 수익 리스트 카드의 높이를 `320px`로 완벽히 고정했습니다.
  - 데이터가 늘어나더라도 카드가 상하로 늘어나 통계 구조를 깨뜨리지 않으며, 카드 내부 전용 스크롤바(`custom-scroll flex-1 overflow-y-auto`)를 통해 깔끔하게 확인이 가능합니다.
* **주간 셀 선택(Selected) 시각 효과 보강**:
  - 기존의 구형 인라인 스타일 코드를 제거하고 스타일시트 제어 클래스로 이관했습니다.
  - 주간 셀이 선택된 상태일 때 **두꺼운 보라색 윤곽선**과 **보랏빛 광채 효과(shadow)**, 입체 부유 효과를 적용해 가시성을 강화했습니다.
* **텐키(Tenkey) 단축키 설정 수정 및 개선**:
  - 텐키 관련 단축키 기능의 세부 스펙을 보완했습니다.

### Fixed (버그 수정)
* **채팅 자동 득템 로그 파싱 및 중복 합산 버그 수정**:
  - 채팅 로그를 통해 수집되는 자동 득템 메시지 포맷(`[득템] [아이템명]을(를) [수량]개 획득하였습니다.`)을 정밀 파싱하는 정규표현식 매칭을 추가했습니다.
  - 이를 통해 모든 자동 수집 득템 아이템의 이름이 올바르게 정규화되어 달력 셀과 누적 목록에 중복 수량이 완벽히 합산되며, 이미지 엑박 깨짐 오류도 완전히 해결되었습니다.
* **구버전 득템 키워드 마이그레이션(V2) 적용**:
  - 기존 사용자가 수동 설정했던 키워드 데이터를 74종 기본값으로 강제 덮어쓰고, 이후 수정 사항은 영구 보존되도록 1회성 마이그레이션 로직을 백엔드에 안전하게 추가했습니다.
* **활동 타임라인 깨짐 및 이미지 로드 실패 대응**:
  - 타임라인 내 `onerror` HTML 파싱 깨짐 문제를 전역 예외 처리기(`handleLootImageError`)로 안전하게 보호하고 시스템 아이콘 폴백을 연동했습니다.
* **활동 히트맵 및 요일 리듬 그래프 다크 툴팁 복구**:
  - Tailwind CSS의 가상 요소 초기화 간섭을 우회하는 스타일 재정의를 통해, 마우스 호버 시 다크 테마 커스텀 툴팁이 가려지지 않고 최상단에 안정적으로 출력되도록 수정했습니다.
* **타임라인 삭제 버튼 수직/수평 중앙 정렬**:
  - 수동 입력 및 득템 타임라인 내 `x` 삭제 버튼에 Flex 정렬을 적용하여 아이콘이 한쪽으로 찌그러지지 않고 중앙에 이쁘게 정렬되도록 버그를 잡았습니다.

## 📥 설치 방법

1. 본 릴리즈 페이지 최하단의 **Assets** 섹션에서 `twOverlay-Setup-1.18.11.exe` 파일을 클릭하여 다운로드합니다.
2. 기존에 실행 중이던 `tw-overlay` 앱이 있다면 반드시 완전히 종료한 뒤 설치 파일을 실행합니다.
3. 화면의 안내에 따라 설치를 완료하면 새로운 기능과 패치가 적용됩니다.
