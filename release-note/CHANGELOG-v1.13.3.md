# 📦 TW-Overlay Release Note - v1.13.3 (2026.05.21)

이번 v1.13.3 업데이트에서는 **숙제 체크리스트 주간화 및 횟수제 UI/UX 전면 개편**과 **에토스 기믹 알림 UI 간소화(홀로그램 디자인 적용)**를 진행하였습니다.

---

## 🚀 주요 변경 사항

### 📅 숙제 체크리스트 주간화 및 횟수제 UI/UX 개편
테일즈위버의 주간 숙제 완화 패치에 부합하도록 체크리스트 시스템을 전면 재설계했습니다.
* **주간 숙제 변환**: 기존 15종의 일일 숙제가 주간 숙제(매주 월요일 오전 0시 리셋)로 변경되었습니다.
* **횟수 카운터 UI 도입**: 단순 체크박스에서 횟수를 조작할 수 있는 카운터형 컴포넌트(0/7)로 변경되었습니다.
* **직관적인 마우스 조작 방식**:
  * **마우스 좌클릭**: 완료 횟수 `+1` 증가 (최대 도달 시 초록색 완료 처리)
  * **마우스 우클릭**: 완료 횟수 `-1` 감소
  * **즉시 완료 체크박스**: 카운터 오른쪽에 위치한 체크박스를 클릭하여 즉시 100% 완료 상태로 만들거나 완료 해제 처리(토글)
* **안전한 데이터 마이그레이션**: 이전 버전을 쓰던 사용자의 기존 설정(숙제 정렬 순서, 미참여 상태, 가시성 등)을 유실하지 않고 새 형식에 맞추어 자동으로 주간 숙제 ID 및 설정 값으로 마이그레이션합니다.

### ✨ 에토스 기믹 알림 UI 간소화 (홀로그램 디자인)
에토스 기믹 알림을 게임 화면에 더욱 자연스럽고 몰입감 있게 표시하도록 디자인을 조율했습니다.
* **화면 텍스트 제거**: 화면 머리 위에 가독성을 해치던 텍스트(예: "에토스: NE")를 전면 제거하였습니다.
* **홀로그램 화살표 지시자**: 방향 화살표의 크기를 80px로 최적화하고, 불투명한 검은색 원형 배경(Scrim)을 제거하여 화면 가림을 최소화했습니다.
* **세련된 SF 스타일 스타일**: 맑은 반투명 화이트 색상 채우기, 세련된 스카이블루 테두리, 그리고 은은한 블루 네온 빛의 Glow 펄스 애니메이션을 결합하여 가시성을 확보하면서도 촌스럽지 않은 고급스러운 느낌을 제공합니다.

## 🛠 상세 내역

### Added (추가)
* 숙제 카운트 조작용 마우스 이벤트 리스너(좌클릭, 우클릭) 및 즉시 완료 토글용 체크박스 이벤트 바인딩
* 에토스 알림용 홀로그램 화살표 펄스 애니메이션 및 스타일 정의
* 새로운 주간 숙제 횟수 제어용 IPC 채널 (`contents-checker:update-count`) 추가

### Changed (변경)
* 일일 숙제 15종의 ID를 `daily-*`에서 `weekly-*`로 마이그레이션
* 리셋 주기(매주 월요일 0시) 및 최대 횟수(`maxCount: 7`) 적용
* "룬 경험의 던전" 표시 명칭 축소 수정
* 에토스 기믹 표시 화살표의 디자인 및 크기(80px) 조정, 텍스트 가이드 완전 제거

### Fixed (수정)
* 누락되어 있었던 숙제 3종 추가 반영
* 숙제 리스트 카테고리 일괄 정리 및 안전한 기존 설정 마이그레이션 예외 처리
