# TW-Overlay v2.0.5 릴리즈 노트 (2026.07.04)

이번 업데이트는 팔색조 언덕 일일 숙제 카운팅 방식 개선 및 디스코드 알람 스팸 방지 예외 필터 등을 반영한 **패치 v2.0.5 업데이트**입니다.

---

## 🚀 주요 변경 사항

### ⚙️ Changed (기능 개선 사항)

* **🦜 팔색조 언덕 일일 숙제 카운팅 방식 개선**:
  * 기존 보상 획득 시 누적으로 카운트가 올라가던 방식을 제거하고, 던전 입장 시점에 출력되는 남은 에너지 (`K`) 메시지를 기반으로 전체 숙제 완료 횟수를 `(20 - K) + 1`로 계산하여 즉시 동기화하도록 변경했습니다.
  * 보상을 받지 못하거나 메시지가 유실되어 카운트가 누락되던 부작용을 방지하고 정확한 숙제 현황을 보장합니다.
* **🦜 K값 인식 불가 시 예외 안전장치 추가**:
  * 만약 에너지 수치(`K`) 파싱에 실패하거나 비정상적인 데이터가 유입될 경우 에너지를 `0`으로 처리하며, 계산된 횟수가 유효 범위(1~5회)를 벗어날 경우 숙제 정보가 잘못 오염되지 않도록 갱신을 건너뛰는 예외 처리를 반영했습니다.
* **💬 디스코드 알림 클럽 공지 제외 필터 추가**:
  * 지정 단어 알림과 감지 규칙을 동일하게 정렬하여, '클럽 공지' 발신자의 공지 대사들이 디스코드 알림 웹훅을 불필요하게 트리거하지 않도록 예외 필터를 추가했습니다.

## 📥 설치 및 적용 방법

**방법 1: 앱 내 설정 메뉴를 통한 업데이트 (권장)**
1. 앱 내부에서 **[설정]** 메뉴를 엽니다.
2. **[앱 정보]** 탭으로 이동한 뒤 **[업데이트 확인]** 버튼을 클릭하여 수동 업데이트를 진행합니다.

**방법 2: 직접 다운로드하여 업데이트**
1. [Releases](https://github.com/drt0927/tw-overlay/releases) 페이지에서 최신 버전의 `twOverlay-Setup-2.0.5.exe` 파일을 다운로드합니다.
2. 기존 앱을 완전히 종료하신 후, 다운로드한 `.exe` 설치 파일을 실행하여 설치를 진행합니다.
