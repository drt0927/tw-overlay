# 📦 TW-Overlay Release Note [v1.11.5]

v1.11.5 패치는 이전 버전(v1.11.4)에서 발견된 숙제 체크리스트의 치명적인 데이터베이스 오류를 긴급 수정하였습니다.

## 🐞 Fixed (버그 수정)

- **숙제 체크리스트 SQL 오류 수정**: 
  - 숙제를 완료 상태로 변경할 때 발생하던 `SqliteError: 5 values for 6 columns` 문제를 해결했습니다.
  - 데이터베이스 삽입 쿼리의 매개변수 개수가 맞지 않던 로직을 정상화하여 이제 숙제 체크 및 점수 합산이 정상적으로 동작합니다.
