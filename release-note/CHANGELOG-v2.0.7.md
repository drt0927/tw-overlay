# TW-Overlay v2.0.7 릴리즈 노트 (2026.07.06)

이번 업데이트는 채팅 오버레이가 화면 영역 바깥으로 이탈해 사라지는 현상을 복구하는 로직을 담은 **패치 v2.0.7 업데이트**입니다.

---

## 🚀 주요 변경 사항

### 🐛 Fixed (버그 수정 및 최적화)

* **🐛 채팅 오버레이 화면 이탈 복구 로직 추가**:
  * 채팅 오버레이 창(`chatOverlay`, `chatOverlaySub`, `chatOverlaySub2`)이 화면 영역 바깥으로 밀려나 완전히 사라진 경우를 감지하여, 창이 다시 켜질 때 현재 활성화된 모니터 화면 중앙으로 자동 복구하고 해당 위치를 즉시 저장하도록 오버레이 창 배치 기능을 개선했습니다.

---

## 📥 설치 및 적용 방법

**방법 1: 앱 내 설정 메뉴를 통한 업데이트 (권장)**
1. 앱 내부에서 **[설정]** 메뉴를 엽니다.
2. **[앱 정보]** 탭으로 이동한 뒤 **[업데이트 확인]** 버튼을 클릭하여 수동 업데이트를 진행합니다.

**방법 2: 직접 다운로드하여 업데이트**
1. [Releases](https://github.com/drt0927/tw-overlay/releases) 페이지에서 최신 버전의 `twOverlay-Setup-2.0.7.exe` 파일을 다운로드합니다.
2. 기존 앱을 완전히 종료하신 후, 다운로드한 `.exe` 설치 파일을 실행하여 설치를 진행합니다.
