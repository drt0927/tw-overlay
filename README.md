# 🎮 TW-Overlay (TalesWeaver Companion Browser)

테일즈위버(TalesWeaver) 플레이어를 위한 **지능형 다중 창 오버레이 브라우저**입니다. 게임 화면과 연동되는 자석형 위젯과 독립된 브라우저 창을 통해 최상의 플레이 환경을 제공합니다.

## ✨ 주요 기능

*   **📊 월간 모험 리포트 (Infographic):** 이번 달 출석 히트맵, 최애 보스 Top 3, 수익 합계 등 나의 모험을 시각적인 인포그래픽으로 한눈에 파악할 수 있습니다.
*   **📖 모험 일지 (Adventure Log):** 보스 처치, 숙제 완료, 득템 등을 자동으로 기록합니다. 활동량에 따른 **성장 아이콘 시스템**을 즐겨보세요.
*   **⚙️ 독립 유틸리티 모드:** 게임이 꺼져 있어도 시스템 트레이 메뉴를 통해 계산기나 일지 기능을 자유롭게 사용할 수 있습니다.
*   **💾 데이터 백업 및 복구:** 모든 설정과 일지 데이터를 ZIP 파일로 안전하게 백업하고 원클릭으로 복구할 수 있습니다.
*   **🎨 통합 테마 시스템:** 사이드바 아이콘 컬러에 맞춰 각 기능 창의 디자인이 자동으로 매칭되는 세련된 테마를 제공합니다.
*   **✅ 일일/주간 숙제 체크 리스트:** 30여종의 숙제 관리 및 자동 초기화 지원.
*   **🚀 게임 네트워크 최적화 (Fast Ping):** 네이글 알고리즘 제어를 통한 응답 속도 개선.

## 🚀 설치 및 사용 방법

### 1. 설치하기
[Releases](https://github.com/drt_0/tw-overlay/releases) 페이지에서 최신 버전의 `twOverlay-Setup-1.10.3.exe` 파일을 다운로드하여 설치하세요. 기존 사용자는 앱 내 업데이트 기능을 사용할 수 있습니다.

### 2. 실행하기
1. `twOverlay`를 실행하면 테일즈위버 감지 시 자동으로 사이드바가 나타납니다.
2. 게임이 없어도 **시스템 트레이 아이콘** 우클릭 메뉴로 모든 도구를 실행할 수 있습니다.

## 📝 최신 업데이트 내역 (v1.10.3)
- **신규 기능: 모험 일지 통계:** 월간 출석 히트맵 및 보스 랭킹을 보여주는 인포그래픽 탭 추가.
- **기능 개선: 독립 실행 지원:** 게임 미실행/최소화 시에도 트레이 메뉴를 통해 유틸리티 도구 단독 사용 가능.
- **UI/UX 개선:** 다이어리 창 크기 확장(1000px) 및 가시성 개선, 커스텀 툴팁 적용.

## 📝 이전 업데이트 내역 (v1.10.1)
- **버그 수정:** 환경 설정 내 '사이드바 메뉴 관리' 체크박스 데이터 바인딩 오류 수정.
- **기능 개선:** 필드보스 알림 설정 화면 UI 가독성 향상.
- **신규 기능: 모험 일지 (Adventure Log):** SQLite 기반 로컬 활동 기록 시스템 및 성장 아이콘 시스템 도입.
- **신규 기능: 마정석 계산기 (Magic Stone Calculator):** 수익 계산 및 일지 자동 연동 기능.
- **신규 기능: 데이터 백업 및 복구:** 모든 설정과 DB를 ZIP으로 묶어 저장/복구 기능 추가.
- **UI/UX 대규모 개편:** 사이드바 메뉴 중앙 관리 및 통합 테마 시스템 구축.
