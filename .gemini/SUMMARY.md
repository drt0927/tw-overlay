# TalesWeaver Companion Browser (TW-Overlay) 프로젝트 가이드 (v1.10.5)

이 문서는 v1.10.5 버전을 기준으로 작성되었습니다.

## 1. 개요 및 최종 상태
- **목적:** 테일즈위버 전용 지능형 오버레이 브라우저 및 게임 동기화 위젯.
- **최종 빌드:** **TypeScript**, Electron, **Native Win32 API(Koffi)**, **SQLite(better-sqlite3)** 기반.
- **주요 개선:** Windows 네이티브 알림 통합 및 모험 일지 중복 기록 방지.

## 2. 주요 기능 명세
- **Hybrid Notifier:** 게임 최소화/종료 시 Windows 네이티브 알림(Toast) 발송 (필드보스, 커스텀 알람).
- **Smart Adventure Log:** 일지 창 로드 시 오늘 날짜 자동 선택 및 DB 레벨의 중복 기록 차단.
- **Custom Mouse Cursors:** 앱 전체에 애니메이션 및 전용 마우스 커서 적용.
- **Infographic Statistics:** 월간 활동 히트맵, 최애 보스 Top 3 등 비주얼 리포트 제공.
- **Field Boss Notifier:** 실시간 필드 보스 알림 및 처치 기록 연동 UI.

- **업데이트 히스토리 (v1.10.5 핵심)**
- **신규 기능:** Windows 네이티브 알림(Toast) 지원 (게임 화면을 보고 있지 않을 때 필드보스/커스텀 알림 제공).
- **데이터 무결성:** 동일 보스/스폰 시간 중복 기록 방지 로직 및 UI 피드백('이미 기록됨!') 추가.
- **UX 개선:** 모험 일지 열기 시 오늘 날짜 자동 선택, 알림 발생 시 사이드바 너비 자동 확장.
- **UI 보정:** 필드보스 설정 창 높이 확장(750px), 커스텀 커서 상속 버그(버튼 내 SVG) 수정.

- **업데이트 히스토리 (v1.10.4 핵심)**
- **신규 기능:** 앱 전역 커스텀 마우스 커서 시스템 (Windows Aero & 테마 기반).
- **사운드 강화:** 일일 숙제 완료("와우!"), 주간 완료("Max Affection"), 보스 기록 사운드 추가.
- **UI/UX 개선:** 필드보스 설정 화면 내 가이드 배너, 상태 아이콘(칼/체크/시계) 및 툴팁 가이드 추가.
